惠更斯的原著是以**欧几里得几何学的公理化体例**写成的。他没有使用现代代数方程，而是完全依靠**几何比例、相对性思想实验（船）以及伽利略落体定律**，完成了一场纯逻辑的无瑕疵推演。

以下是惠更斯推导完全弹性碰撞定律的**完整严密论述与证明步骤**（为了现代读者易读，原著中的几何线段比例被我如实地替换为了代数符号 $m, v$，但**逻辑链条与原著完全一致**）。

### 第一部分：公理（Hypotheses）
惠更斯的推导建立在以下三个不可动摇的公理之上：

*   **公理 I（惯性）：** 运动的物体如果不受阻碍，将以恒定速度沿直线永远运动。
*   **公理 II（绝对对称性）：** 两个**质量相等**的物体，以**相等的速度**相向运动发生正碰，碰撞后它们必然以**各自原有的速度反向弹回**。（证明理由：因为两边条件完全对等，没有任何充分理由使其中一个碰后速度大于或小于另一个。）
*   **公理 III（相对性原理）：** 无论观察者是静止在岸上，还是在一艘匀速直线行驶的船上，物体碰撞相互作用的结果（它们彼此分离的相对速度）对观察者来说必然遵循相同的自然法则。

---

### 第二部分：等质量物体的碰撞推演

惠更斯的第一阶段任务是：在不借助任何“力”或“能量”概念的情况下，仅用**公理 II（对称）**和**公理 III（相对性/船）**，推导出等质量碰撞的所有情况。

#### 命题 I 的证明：等质量，一动一静
**命题：** 设质量相等的物体 A 和 B。A 以速度 $v$ 运动，撞击静止的 B。证明：碰后 A 静止，B 以速度 $v$ 运动。

**惠更斯的证明过程（经典的“船的论证”）：**
1. 设岸上的真实情况是：A 的速度为 $v$（向右），B 的速度为 $0$。
2. 现在，想象这一切发生在一艘河边的船上。假设这艘船正以 $v/2$ 的速度**向右**匀速行驶。
3. 安排一名站在船上的观察者。根据相对速度的几何关系，在船上观察者看来：
   * A 的速度 = 岸上 A 的速度 - 船速 = $v - v/2 = \mathbf{v/2}$（向右）。
   * B 的速度 = 岸上 B 的速度 - 船速 = $0 - v/2 = \mathbf{-v/2}$（向左）。
4. **引入公理 II：** 在船上的观察者看来，此时 A 和 B 是两个等质量物体，以绝对相等的速度（$v/2$）相向运动。根据对称性公理，碰后它们必然以原速反弹。
   * 碰后在船上，A 的速度变为 $\mathbf{-v/2}$（向左）。
   * 碰后在船上，B 的速度变为 $\mathbf{+v/2}$（向右）。
5. **引入公理 III：** 回到岸上观察者的视角。必须把船的速度（向右的 $v/2$）加回到碰后的物体上：
   * 碰后岸上 A 的速度 = 船上 A 的碰后速度 + 船速 = $-v/2 + v/2 = \mathbf{0}$。
   * 碰后岸上 B 的速度 = 船上 B 的碰后速度 + 船速 = $+v/2 + v/2 = \mathbf{v}$。
6. **结论（Q.E.D.）：** A 彻底停下，B 获得了 A 原来的全部速度。两者完美交换了速度。

#### 命题 II 与 命题 III 的证明：等质量，任意速度
**惠更斯的证明思路：**
只要是等质量物体（无论同向追及还是反向相撞，速度设为 $v_A$ 和 $v_B$），惠更斯只需设定船的速度为 $\frac{v_A + v_B}{2}$（即两者的算术平均值）。
在这艘特定的船上，A 和 B 永远会以相等的速度相向运动。应用一次对称性反弹，再变换回岸上，数学结果**必然是 A 和 B 交换速度**。

### 第三部分：不等质量物体的碰撞推演（核心难点）

当 $m_A \neq m_B$ 时，**公理 II（对称性）失效了**。没有任何直观公理告诉我们一重一轻相撞会怎样。

为了突破这一点，惠更斯在著作的中间部分隐式地引入了**第四个公理（托里拆利原理 / 重心不能自动升高原理）**，这是整个物理学史上极其光辉的一笔。

#### 新公理（重心原理）：
**“一个物体系统的重心，绝不能通过系统内部的相互作用，上升到比它初始状态更高的高度。”**（否则就造出了永动机）。

惠更斯将这一静力学原理与伽利略落体定律（落体高度 $h \propto v^2$）结合，将其转化为动力学约束。

#### 命题 IV 的证明：反比于质量的碰撞（寻找新的对称基准点）
**命题：** 若 $m_A \neq m_B$，但它们的速度大小与质量成反比（即 $m_A v_A = m_B v_B$），则碰撞后，两者各自以**原来的速度大小反弹**（即 $v'_A = -v_A$, $v'_B = -v_B$）。

**惠更斯的证明过程（反证法与重心原理）：**
1. 假设 A 和 B 分别从高度 $h_A$ 和 $h_B$ 沿光滑斜面滑下，恰好在底部获得了速度 $v_A$ 和 $v_B$。根据伽利略定律，$h_A \propto v_A^2$， $h_B \propto v_B^2$。
2. 此时，释放前系统整体重心的初始高度为：$H_{初} = \frac{m_A h_A + m_B h_B}{m_A + m_B}$。
3. 碰撞后，假设它们以未知速度 $v'_A$ 和 $v'_B$ 反弹，并冲上斜面到达新的高度 $h'_A$ 和 $h'_B$。
4. 碰后系统的重心高度为：$H_{末} = \frac{m_A h'_A + m_B h'_B}{m_A + m_B}$。
5. **反证第一步：** 假设它们碰后的反弹速度**大于**碰前。那么 $h'_A > h_A$ 且 $h'_B > h_B$，导致 $H_{末} > H_{初}$。这就违背了“重心不能自动升高”的公理，造出了永动机。所以反弹速度不可能变大。
6. **反证第二步：** 假设它们碰后的反弹速度**小于**碰前。惠更斯再次召唤出**“船的论证”**：如果在一艘静止的船上反弹速度变小（重心下降了），那么我们总能找到一艘以特定速度行驶的船，对那艘船上的观察者而言，经过相对速度变换后，系统的重心不仅没有下降，反而**升高了**！（惠更斯在这里给出了极其冗长且严密的几何不等式证明，证明只要碰后相对速度减小，必然存在一个参考系使得重心上升）。
7. 但是物理法则在任何船上都必须一致，既然在某艘船上重心上升了，同样违背了永动机不可能原理。
8. **结论（Q.E.D.）：** 碰后的速度既不能变大，也不能变小，**必定完全等于**碰前的速度。于是，惠更斯为不等质量的碰撞找到了一个**绝对的对称基准点**（即现代物理中的总动量为零的参考系）。

#### 命题 IX 的证明：任意质量与任意速度的普适碰撞定律
有了命题 IV 作为新的基石，推导普适定律水到渠成。

**惠更斯的推导过程：**
1. 设任意质量 $m_A, m_B$，以任意速度 $v_A, v_B$ 运动（均为相对于岸的地面坐标系）。
2. 惠更斯设定一艘船，使其以速度 $u = \frac{m_A v_A + m_B v_B}{m_A + m_B}$ （即质心速度）行驶。
3. 站在船上观察：
   * 船上 A 的速度 = $v_A - u$
   * 船上 B 的速度 = $v_B - u$
4. 惠更斯通过代数（原著为几何比例）证明：在船上，$m_A (v_A - u) = - m_B (v_B - u)$。即在船上，**两者的速度必定与质量成反比**。
5. **套用命题 IV：** 因为在船上满足了反比关系，所以碰后它们在船上的速度必然是**原速反弹**：
   * 碰后船上 A 的速度 = $-(v_A - u)$
   * 碰后船上 B 的速度 = $-(v_B - u)$
6. **套用相对性原理回到岸上：** 加上船的速度 $u$。
   * 碰后岸上 A 的速度 $v'_A = -(v_A - u) + u = 2u - v_A$
   * 碰后岸上 B 的速度 $v'_B = -(v_B - u) + u = 2u - v_B$
7. 将 $u = \frac{m_A v_A + m_B v_B}{m_A + m_B}$ 代入上述两式，立刻得出完全弹性碰撞的终极公式：
   $$v'_A = \frac{(m_A - m_B)v_A + 2m_B v_B}{m_A + m_B}$$
   $$v'_B = \frac{(m_B - m_A)v_B + 2m_A v_A}{m_A + m_B}$$
   （至此，整个一维弹性碰撞问题被彻底解决）。

### 第四部分：推导三大守恒定理作为推论

在完成了具体的碰撞推导后，惠更斯反过头来检视他的数学结果，从中提取出了物理学最基础的守恒律。他证明了这些守恒律不是前提，而是**必然的数学推论**。

#### 证明相对速度守恒（恢复系数恒等于1）
由上述推导中的第6步：
$v'_A - v'_B = (2u - v_A) - (2u - v_B) = -v_A + v_B = -(v_A - v_B)$
**惠更斯的论述：** 两个物体碰撞前相互接近的相对速度，必然在数值上精确等于它们碰撞后相互分离的相对速度。

#### 证明质心运动守恒（等价于动量守恒）
**惠更斯的论述：** 碰后的质心速度 $u' = \frac{m_A v'_A + m_B v'_B}{m_A + m_B}$。
将上述得到的 $v'_A$ 和 $v'_B$ 代入此式展开化简，惠更斯在数学上证明了 $u' \equiv u$。
也就是说：无论碰撞发生得多么猛烈，整个系统共同的重心始终以**完全不变的速度，沿着原有的直线继续运动**。这一证明彻底埋葬了笛卡尔只考虑大小不考虑方向的“动量守恒”。

#### 命题 XI：证明“质量乘以速度平方”之和的守恒（动能守恒）
**惠更斯的完整证明：**
在命题 IV 的推导中，为了保证重心不升高（系统不创造动力），惠更斯已经确定碰后物体能达到的高度的加权平均值，必须等于碰前高度的加权平均值：
$H_{初} = H_{末}$
即： $\frac{m_A h_A + m_B h_B}{m_A + m_B} = \frac{m_A h'_A + m_B h'_B}{m_A + m_B}$
去掉分母得到：
$m_A h_A + m_B h_B = m_A h'_A + m_B h'_B$

因为根据伽利略落体定律，物体下落高度与其获得速度的平方成正比（在真空中 $h \propto v^2$）。
因此，将等式两边的高度 $h$ 替换为对应的速度平方 $v^2$，惠更斯直接得出了那个震撼物理学史的结论：
**$m_A v_A^2 + m_B v_B^2 = m_A (v'_A)^2 + m_B (v'_B)^2$**

**惠更斯的原话论述：** “如果有两个物体相互碰撞，将其各自的质量乘以其速度的平方，把得到的两个乘积相加；那么在碰撞之后，如果我们做同样的操作，得到的总和必定与碰撞前完全相等。”

### 全文论述总结

这就是惠更斯《论碰撞引起的物体运动》的完整逻辑证明链。他**没有**先验地假设动量守恒或动能守恒。

他是这样做的：
**第一步：** 相信匀速直线运动不可区分（相对性/船）。
**第二步：** 相信绝对相等的条件产生绝对相等的结果（对称性）。
**第三步：** 相信宇宙中不可能凭空造出动力让物体重心上升（反永动机）。

由这三个最质朴的直觉，惠更斯通过纯粹的几何与代数推演，强行推导出了弹性碰撞的全部公式，并在推导的终点，发现了隐藏在方程中的动量守恒和动能守恒。这是17世纪理性力学最完美的演绎之一。
