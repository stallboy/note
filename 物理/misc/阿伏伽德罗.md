

# 阿伏伽德罗假说

> **在相同温度和压强下，相同体积的任何气体都含有相同数目的分子。**
> ——阿梅代奥·阿伏伽德罗，1811年

---

## 第一部分：历史背景——假说诞生前的困境

### 一、拉瓦锡和道尔顿已经建立的基础

到19世纪初，化学已经取得了重要进展：

- **拉瓦锡**（1780年代）建立了现代化学元素的概念，推翻了燃素说。
- **道尔顿**（1803-1808年）提出了**原子论**：每种元素由不可分割的原子组成，不同元素的原子质量不同，化合物由原子以简单整数比结合而成。

### 二、盖-吕萨克的气体体积定律（1808年）

盖-吕萨克通过精确实验发现了一个惊人的简洁规律：

> **在相同温度和压强下，参与反应的气体的体积之间呈简单整数比。**

实验证据：

| 反应 | 体积比 |
|---|---|
| 氢气 + 氯气 → 氯化氢 | 1 : 1 : 2 |
| 氢气 + 氧气 → 水蒸气 | 2 : 1 : 2 |
| 氮气 + 氢气 → 氨气 | 1 : 3 : 2 |

这些简洁的整数比强烈暗示：**气体的体积与其中所含的微粒数之间存在某种简单关系**。

### 三、道尔顿的困惑与拒绝

道尔顿本人看到了盖-吕萨克的数据，却**拒绝接受**。原因在于：

如果体积正比于微粒数目，那么：

$$1 \text{体积氢} + 1 \text{体积氯} → 2 \text{体积氯化氢}$$

按道尔顿的理解：
- 1个氢原子 + 1个氯原子 → 1个氯化氢"原子"（道尔顿不区分原子和分子）
- 那么微粒总数：1 + 1 = 1 ???
- 但产物体积是2，暗示产生了2个微粒？？？

**一个氢原子加一个氯原子怎么可能产生两个微粒？这在道尔顿的框架下完全不可理解。**

更令人困惑的是水的生成：

$$2 \text{体积氢} + 1 \text{体积氧} → 2 \text{体积水蒸气}$$

如果氢的最小单位是一个氢原子，氧的最小单位是一个氧原子，那么：
- 2个氢原子 + 1个氧原子 → 2个水微粒？
- 但1个氧原子怎么分成两半，分别进入两个水微粒？

**原子不是不可分割的吗？** 这里出现了根本性的矛盾。

道尔顿因此怀疑盖-吕萨克的实验数据有误，两人产生了激烈争论。

## 第二部分：阿伏伽德罗的突破（1811年）

### 解决方案的核心

阿伏伽德罗在1811年发表的论文《关于决定物体基本分子的相对质量及其化合比例的一种方法》（*Essai d'une manière de déterminer les masses relatives des molécules élémentaires des corps*）中，做出了**两个关键区分**：

#### 关键区分一：原子 ≠ 分子

> 单质气体的最小独立存在单位不是单个原子，而是由**若干个相同原子结合而成的分子**。

例如：
- 氢气不是由单个 H 原子组成的，而是由 **H₂ 分子**组成的。
- 氧气不是由单个 O 原子组成的，而是由 **O₂ 分子**组成的。
- 氯气是 **Cl₂ 分子**。
- 氮气是 **N₂ 分子**。

**这是一个革命性的概念：同种原子可以彼此结合。** 道尔顿认为同种原子之间只有排斥力，不可能结合——阿伏伽德罗打破了这个教条。

#### 关键区分二：等体积等数目假说

> **在相同温度和压强下，相同体积的任何气体都含有相同数目的分子。**

即：
$$V \propto N \quad (P, T \text{ 恒定})$$

### 矛盾的化解

有了这两个假设，所有困难迎刃而解。以水的生成为例：

$$2 \text{体积 } H_2 + 1 \text{体积 } O_2 \to 2 \text{体积 } H_2O$$

用"等体积等数目"翻译为分子数：

$$2N \text{个} H_2 + N \text{个} O_2 \to 2N \text{个} H_2O$$

即每个反应单元：

$$2H_2 + O_2 \to 2H_2O$$

展开为原子层面：

$$4 \text{个} H + 2 \text{个} O \to 2 \text{个} H_2O$$

每个水分子含 2个H 和 1个O → $H_2O$ ✓

**没有任何原子被"劈开"，一切自洽。**

再看氯化氢：

$$1 \text{体积 } H_2 + 1 \text{体积 } Cl_2 \to 2 \text{体积 } HCl$$

$$N \text{个} H_2 + N \text{个} Cl_2 \to 2N \text{个} HCl$$

$$H_2 + Cl_2 \to 2HCl$$

每个 $H_2$ 提供2个H，每个 $Cl_2$ 提供2个Cl，恰好组成2个HCl ✓

再看氨的生成：

$$1 \text{体积 } N_2 + 3 \text{体积 } H_2 \to 2 \text{体积 } NH_3$$

$$N_2 + 3H_2 \to 2NH_3$$

1个 $N_2$ 提供2个N，3个 $H_2$ 提供6个H → 2个 $NH_3$（每个含1个N和3个H）✓

## 第三部分：假说被忽视的半个世纪（1811-1860）

### 为什么这么好的理论被埋没了近50年？

这是科学史上最令人扼腕的案例之一。原因是多方面的：

#### 1. 阿伏伽德罗本人的局限

- 他是意大利都灵的一位物理学教授，远离当时化学研究的中心（巴黎、伦敦、斯德哥尔摩）。
- 他的论文发表在一本不太知名的意大利期刊上。
- 他的文风晦涩，论文可读性差。

#### 2. 权威的反对

- **道尔顿**拒绝承认同种原子可以结合成分子。
- **贝采利乌斯（Berzelius）**——当时化学界的最高权威——提出了**电化学二元论**：他认为原子之间的结合是由电性相反的原子相互吸引引起的。两个相同的氧原子带同种电荷，怎么可能互相吸引形成 $O_2$？因此他认为"双原子分子"的概念是荒谬的。
- **安培和杜马**虽然各自独立提出过类似想法，但也未能说服主流化学界。

#### 3. 概念上的混乱

当时"原子""分子""当量""原子量"等概念混为一谈。同一种物质，不同化学家使用不同的化学式：

| 化学家 | 水的化学式 | 水的"原子量" |
|---|---|---|
| 道尔顿 | HO | 8 (以H=1) |
| 贝采利乌斯 | H₂O (有时) | 不确定 |
| 日拉尔 | H₂O₂ | — |

**整个化学界对基本化学式都没有共识。** 在这样的混乱中，阿伏伽德罗假说的重要性被淹没了。

## 第四部分：康尼扎罗的复兴（1860年）

### 卡尔斯鲁厄大会

1860年，在德国卡尔斯鲁厄（Karlsruhe）召开了**历史上第一次国际化学会议**。召开的原因正是化学界在原子量、分子式等基本问题上的混乱已经到了不可忍受的地步。

意大利化学家**斯坦尼斯劳·康尼扎罗（Stanislao Cannizzaro）**在会上发表了激情澎湃的演讲。他系统地展示了如何利用阿伏伽德罗假说，结合气体密度的测量，**一致地、无矛盾地确定所有元素的原子量和化合物的分子式**。

### 康尼扎罗的方法

其核心逻辑非常优美：

**第一步：** 由阿伏伽德罗假说，相同条件下等体积气体含等数分子。因此：

$$\frac{\text{气体A的密度}}{\text{气体B的密度}} = \frac{\text{A的分子质量}}{\text{B的分子质量}}$$

通过测量各种气体的密度（这在实验上很容易做到），就可以确定它们的**相对分子质量**。

**第二步：** 分析含某一元素的**大量不同化合物**的气态密度，找出该元素在各化合物中所占的质量份数。

**第三步：** 这些质量份数的**最大公约数**，就是该元素的**原子量**。

例如，分析多种含氧化合物：

| 化合物 | 分子质量 | 含氧质量 | 含氧质量/16 |
|---|---|---|---|
| 水 H₂O | 18 | 16 | 1 |
| 二氧化碳 CO₂ | 44 | 32 | 2 |
| 一氧化碳 CO | 28 | 16 | 1 |
| 二氧化硫 SO₂ | 64 | 32 | 2 |

含氧质量的最小值为16 → **氧的原子量 = 16** ✓

### 大会的影响

康尼扎罗在会后分发了他的小册子。与会者阅读后深受震撼：

- **洛塔尔·迈耶尔**（Lothar Meyer）后来回忆说："读了康尼扎罗的小册子，就像鳞片从眼上落下。"
- **门捷列夫**也出席了这次大会。正是康尼扎罗理清的正确原子量，使门捷列夫在1869年能够发现**元素周期律**。

**可以说，没有阿伏伽德罗假说的复兴，就没有元素周期表。**

## 第五部分：现代物理学如何解释阿伏伽德罗假说

### 从 $PV = Nk_BT$ 直接推出

现代理想气体状态方程：

$$PV = Nk_BT$$

其中 $N$ 是分子总数。

如果两种气体 A 和 B 处于相同的 $P$、$V$、$T$：

$$PV = N_A k_B T = N_B k_B T$$

两边约掉 $Pk_BT$：

$$\boxed{N_A = N_B}$$

**证毕。** 阿伏伽德罗假说是理想气体状态方程的直接推论。

### 从微观角度理解

根据气体分子运动论：

$$P = \frac{N}{V} \cdot \frac{1}{3}m\overline{v^2} = \frac{N}{V} \cdot \frac{2}{3}\overline{E_k}$$

在相同温度下，**无论气体种类如何**，每个分子的平均平动动能都相同：

$$\overline{E_k} = \frac{3}{2}k_BT$$

因此：

$$P = \frac{N}{V} \cdot \frac{2}{3} \cdot \frac{3}{2}k_BT = \frac{N}{V}k_BT$$

这意味着：**压强只取决于单位体积内的分子数 $N/V$ 和温度 $T$，而与分子质量 $m$（即气体种类）完全无关。**

物理图像：
- 重的分子（如 $O_2$）运动慢，但每次撞壁的冲量大。
- 轻的分子（如 $H_2$）运动快，但每次撞壁的冲量小。
- 在相同温度下，**这两个效应恰好精确抵消**，使得每个分子对压强的贡献完全相同。
- 因此，相同压强、温度、体积的条件下，分子数必然相同。

### 数学上看这个精确抵消

单个分子对压强的贡献正比于：

$$m \cdot \overline{v^2}$$

而 $\frac{1}{2}m\overline{v^2} = \frac{3}{2}k_BT$ → $m\overline{v^2} = 3k_BT$

**$m$ 和 $\overline{v^2}$ 的乘积与分子质量无关！**

- 分子越重 → $m$ 大 → 但 $\overline{v^2} = 3k_BT/m$ 同比例缩小
- 分子越轻 → $m$ 小 → 但 $\overline{v^2} = 3k_BT/m$ 同比例增大

结果：$m\overline{v^2}$ 对所有气体分子完全相同。

## 第六部分：阿伏伽德罗常数

### 定义

一摩尔任何物质所含的微粒数：

$$N_A = 6.022 \, 140 \, 76 \times 10^{23} \; \text{mol}^{-1}$$

2019年国际单位制重新定义后，$N_A$ 被赋予了**精确值**。

### 历史上的测定方法

| 方法 | 年代 | 科学家 |
|---|---|---|
| 气体分子运动论估算 | 1865 | 洛施密特（Loschmidt） |
| 布朗运动 | 1908 | 让·佩兰（Jean Perrin） |
| 油滴实验（测 $e$，结合法拉第常数） | 1909 | 密立根（Millikan） |
| X射线晶体衍射 | 20世纪 | 多位 |
| 硅球法（最精确） | 2010s | 国际阿伏伽德罗计划 |

特别值得一提的是**让·佩兰**的工作——他通过显微镜观察悬浮在液体中花粉粒子的布朗运动，利用爱因斯坦1905年的布朗运动理论，首次精确测定了 $N_A$，从而**直接证实了分子的存在**。他因此获得1926年诺贝尔物理学奖。

## 总结

| 层面 | 内容 |
|---|---|
| **假说内容** | 等温等压下，等体积的任何气体含有等数目的分子 |
| **解决的问题** | 调和了道尔顿原子论与盖-吕萨克体积定律的矛盾 |
| **关键创新** | 区分"原子"与"分子"；承认同种原子可以结合（$H_2, O_2, N_2, Cl_2$） |
| **被忽视的原因** | 作者边缘化、权威反对、概念混乱 |
| **复兴** | 1860年康尼扎罗在卡尔斯鲁厄大会上的演讲 |
| **深远影响** | 正确的原子量 → 元素周期表 → 整个现代化学 |
| **现代解释** | $PV = Nk_BT$ 的直接推论；本质是温度只决定单分子平均动能，与分子质量无关，所以每个分子对压强的贡献相同 |

**阿伏伽德罗假说的故事告诉我们：一个正确的理论可能被忽视半个世纪，但真理终将被历史证明。它也展现了物理学与化学在最深处的统一——宏观的化学反应计量比，最终根植于微观的分子运动力学。**
