# 沃利斯如何论证动量守恒方程

这是一个非常好的问题。沃利斯对动量守恒的论证并非像现代物理教科书那样从牛顿第三定律出发（牛顿的《原理》要到1687年才问世），而是基于一套独特的、混合了经验直觉、静力学类比和数学推理的方法。

---

## 一、沃利斯论证的思想来源

### 1. 静力学中的杠杆/天平类比

沃利斯论证的**核心思想基础**来自**静力学**，特别是**天平（balance）和杠杆（lever）原理**。这一点至关重要——他并不是凭空提出动量守恒，而是将一个已知的、被普遍接受的静力学原理**推广**到动力学领域。

在静力学中，天平平衡的条件早已被阿基米德确立：

$$m_1 \times d_1 = m_2 \times d_2$$

其中 $d_1, d_2$ 是两侧的力臂。沃利斯将这一思想进行了关键性的类比转换：

> **将"力臂距离"替换为"速度"**，将静力学的平衡条件转化为碰撞中的运动条件。

### 2. 伽利略的惯性思想

沃利斯也继承了伽利略的**惯性原理**：物体在不受外部作用时会保持其运动状态。这意味着两个物体各自携带的"运动量"（$m \times v$）在碰撞前是确定的，碰撞过程中这些运动量必须以某种方式"重新分配"。

### 3. 笛卡尔的运动量概念（及其修正）

笛卡尔提出了"运动量"（*quantitas motus*）= 体积 × 速率的概念，并认为宇宙中的运动总量守恒。但笛卡尔的致命错误在于他将运动量视为**标量**（不考虑方向）。沃利斯接受了运动量的基本概念，但对其进行了关键修正——**赋予方向性**。

## 二、沃利斯的具体论证过程

### 第一步：建立"运动力"（*vis motrix* / *impetus*）的概念

沃利斯首先定义了每个运动物体所携带的**"运动力"或"冲力"**（他有时使用 *impetus*，有时使用 *vis motrix*）：

$$\text{运动力（impetus）} = \text{重量（pondus）} \times \text{速度（celeritas）}$$

即：

$$I = p \times v$$

这里的"重量"在当时的语境中接近于我们今天的"质量"概念（虽然沃利斯尚未明确区分质量与重量）。

**关键点**：沃利斯明确规定速度是**有方向的量**——向一个方向运动的速度为正，向相反方向运动的速度为负。

### 第二步：天平类比论证

沃利斯的核心论证可以重构如下：

#### 天平模型

想象一个**天平**（fulcrum balance）：

```
        物体A                    物体B
    质量 m₁, 速度 v₁          质量 m₂, 速度 v₂
    ←——————————————|——————————————→
               支点（碰撞点）
```

在天平问题中，我们知道系统的**平衡点**（重心）取决于两侧的质量和距离：

$$\text{重心位置} = \frac{m_1 x_1 + m_2 x_2}{m_1 + m_2}$$

沃利斯的类比推理是：

> 正如天平的平衡点由质量和距离决定，碰撞后两个物体合为一体时的共同速度，应当由质量和速度以相同的数学形式决定。

即**碰撞后的共同速度就是运动的"重心"**：

$$v_f = \frac{m_1 v_1 + m_2 v_2}{m_1 + m_2}$$

#### 为什么这个类比是合理的？

沃利斯的理由是：在一定时间 $t$ 内，物体A走过的距离为 $v_1 \cdot t$，物体B走过的距离为 $v_2 \cdot t$。这两个位移可以类比为天平两臂的"臂长"。那么整个系统的"运动重心"移动的速度就应该是：

$$v_f = \frac{m_1 (v_1 t) + m_2 (v_2 t)}{(m_1 + m_2) t} = \frac{m_1 v_1 + m_2 v_2}{m_1 + m_2}$$

这就自然导出了：

$$m_1 v_1 + m_2 v_2 = (m_1 + m_2) v_f$$

即**碰撞前后动量总和不变**。

### 第三步：重心运动不变性原理

沃利斯论证中一个更深层的物理直觉是**重心运动不变性原理**：

> **一个由多个部分组成的系统，其整体的重心运动状态，不会因为各部分之间的内部相互作用（如碰撞）而改变。**

这一原理的推理逻辑是：

1. 碰撞前，系统（由物体A和物体B组成）的重心以某个速度 $v_{cm}$ 运动
2. 碰撞是系统的**内部事件**——两个物体之间的相互作用不涉及外部力
3. 因此，碰撞不应改变系统重心的运动状态
4. 碰撞后两物体合为一体，其速度就等于 $v_{cm}$

而重心速度为：

$$v_{cm} = \frac{m_1 v_1 + m_2 v_2}{m_1 + m_2}$$

这正好给出了完全非弹性碰撞的结果。

沃利斯在他的力学著作《Mechanica》（1670-1671）中，正式系统地阐述了**重心理论**，并将其作为力学的基本原理之一。他在碰撞理论中对重心概念的运用，实际上就是这一更广阔理论框架的应用。

### 第四步：经验验证与具体案例

沃利斯并非纯粹进行抽象推理。他还通过**具体的数值案例**来展示其理论的合理性，并将结果与经验观察（包括单摆碰撞实验等）进行比较。

#### 案例一：对称碰撞

两个**等质量**物体以**等速**从**相反方向**碰撞：

$$v_f = \frac{m \cdot v + m \cdot (-v)}{m + m} = \frac{0}{2m} = 0$$

碰撞后静止。这与日常经验完全吻合，而且这恰恰是**笛卡尔的标量运动量守恒无法解释的**——因为按笛卡尔的理论，碰撞前的运动总量为 $mv + mv = 2mv$，碰撞后应该仍然为 $2mv$，不可能静止。

沃利斯由此强调了**考虑方向**的必要性。

#### 案例二：不等质量碰撞

质量为3的物体以速度2向右运动，质量为1的物体以速度4向左运动：

$$v_f = \frac{3 \times 2 + 1 \times (-4)}{3 + 1} = \frac{6 - 4}{4} = \frac{1}{2}$$

合体以速度 $\frac{1}{2}$ 向右运动。沃利斯指出，较重的物体"占主导"，但被较轻物体的反向运动部分抵消。

#### 案例三：一方静止

质量为2的物体以速度3向右运动，撞击质量为1的静止物体：

$$v_f = \frac{2 \times 3 + 1 \times 0}{2 + 1} = \frac{6}{3} = 2$$

合体以速度2向右运动。运动物体变慢了，因为它现在要"带动"额外的质量。

## 三、论证的逻辑结构总结

将沃利斯的整个论证重构为清晰的逻辑链：

```
前提1：每个物体携带的"运动力"= 质量 × 速度（有方向）
    ↓
前提2：天平/重心原理——复合系统的整体运动状态
       由各部分的质量和速度按重心法则决定
    ↓
前提3：碰撞是内部相互作用，不应改变系统
       整体（重心）的运动状态
    ↓
前提4：完全非弹性碰撞后两物体合一，
       以共同速度运动
    ↓
结论：v_f = (m₁v₁ + m₂v₂)/(m₁ + m₂)
      即 m₁v₁ + m₂v₂ = (m₁+m₂)v_f
      动量守恒
```

## 四、与现代论证的比较

### 现代论证（从牛顿第三定律出发）

现代物理学中，动量守恒的标准证明是：

1. 牛顿第三定律：$\vec{F}_{12} = -\vec{F}_{21}$
2. 牛顿第二定律：$\vec{F} = \frac{d\vec{p}}{dt}$
3. 因此：$\frac{d\vec{p}_1}{dt} = -\frac{d\vec{p}_2}{dt}$
4. 积分得：$\Delta \vec{p}_1 = -\Delta \vec{p}_2$
5. 即：$\vec{p}_1 + \vec{p}_2 = \text{const}$

### 沃利斯论证的特点

| 方面 | 沃利斯的方法 | 现代方法 |
|------|------------|---------|
| 出发点 | 静力学（天平/重心）类比 | 牛顿运动定律 |
| 力的概念 | 未涉及碰撞过程中的瞬时力 | 通过第二、三定律处理力 |
| 论证性质 | 类比推理 + 经验验证 | 严格的数学演绎 |
| 适用范围 | 主要针对完全非弹性碰撞 | 普遍适用于所有碰撞和相互作用 |
| 守恒量的地位 | 从重心不变性"推出"守恒 | 从力的相互性"证明"守恒 |

## 五、沃利斯论证的深刻之处

### 1. 重心运动不变性——一个惊人的洞见

沃利斯使用的"重心运动不变性"原理，在现代看来实际上是**非常深刻的**。在现代力学中，这正是**内力不改变系统质心运动状态**这一定理的内容，而这本身就等价于动量守恒（或牛顿第三定律）。

也就是说，沃利斯虽然没有牛顿定律的形式体系，但他通过重心概念**抓住了问题的本质**。

### 2. 方向性的引入

沃利斯将速度视为有方向的量，从而使运动力（动量）成为有符号量（或矢量量），这是对笛卡尔标量处理的**根本性修正**。这一修正的物理意义在于：

- 相反方向的运动可以**相互抵消**
- 运动的"净效果"才是守恒量
- 这为后来牛顿将动量定义为矢量奠定了基础

### 3. 从静力学到动力学的桥梁

沃利斯的方法体现了17世纪力学发展的一个重要特征：**通过静力学（已经成熟的学科）的概念和方法来理解动力学（尚在发展的学科）**。天平原理是阿基米德时代就已确立的知识，沃利斯天才地将其推广到运动问题中。

---

## 六、后续影响

沃利斯的论证对后来的发展产生了直接影响：

1. **牛顿**在《原理》中明确承认了沃利斯（以及雷恩和惠更斯）的碰撞理论工作，并在推论部分讨论了碰撞实验对运动定律的验证。

2. **牛顿对"质心"概念的使用**很可能受到了沃利斯重心方法的影响。《原理》中的推论IV明确指出："系统的共同重心不会因为物体之间的相互作用而改变其运动状态。"这与沃利斯的核心论证思路完全一致。

3. 沃利斯将静力学中的重心概念推广到动力学中的方法，成为了力学发展的一个标准范式。

---

## 七、结语

沃利斯对动量守恒的论证，虽然缺乏现代标准下的数学严格性，但展现了卓越的物理直觉。他通过**天平类比**和**重心运动不变性**这两个核心思想，在牛顿定律体系建立之前近20年，就为碰撞中的动量守恒提供了一个概念上清晰、数学上可操作的论证框架。这一工作不仅解决了完全非弹性碰撞的具体问题，更为整个经典力学的动量概念和守恒原理的形成做出了不可磨灭的贡献。
