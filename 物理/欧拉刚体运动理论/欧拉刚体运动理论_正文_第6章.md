# 正文第6章：物体的转动惯量

*De momentis inertiae corporum*

---

## 本章概述

这一章中，欧拉建立了转动惯量的完整理论。他不再局限于绕固定轴的情况，而是研究刚体关于**通过固定点的任意方向轴**的转动惯量。这个研究导致了一个极其重要的发现：转动惯量随方向变化的规律，可以用一个二次型来描述——这就是**惯量张量**（inertia tensor）的概念，尽管欧拉使用的不是现代术语。

> **批注**：这一章是全书数学上最深刻的章节之一。欧拉在这里实际上在研究一个实对称二次型 $I(\hat{\mathbf{n}}) = \sum m_i(\mathbf{r}_i \times \hat{\mathbf{n}})^2$ 的性质——尽管他使用的是18世纪的分析几何语言，而不是矩阵语言。这个二次型在现代物理中被称为惯量张量，它是刚体动力学最重要的数学对象。

---

## 1. 转动惯量的方向依赖性

在第3章中，转动惯量定义为 $I = \sum m_i r_i^2$，其中 $r_i$ 是质点到旋转轴的垂直距离。现在欧拉问一个更深入的问题：**如果旋转轴通过固定点 $O$，但方向可以任意变化，转动惯量如何依赖于轴的方向？**

设旋转轴的方向由单位矢量 $\hat{\mathbf{n}}$ 给出，则第 $i$ 个质点（位置 $\mathbf{r}_i$）到该轴的垂直距离为：
$$r_i = |\mathbf{r}_i \times \hat{\mathbf{n}}|$$

因此转动惯量为：
$$I(\hat{\mathbf{n}}) = \sum_i m_i |\mathbf{r}_i \times \hat{\mathbf{n}}|^2$$

> **批注**：转动惯量不再是常数——它依赖于轴的方向 $\hat{\mathbf{n}}$。这意味着同一个刚体，绕不同方向的轴旋转时，"抵抗角加速度"的能力不同。这个事实在直觉上是合理的：一根长杆绕其长度方向的轴旋转比绕垂直于长度方向的轴旋转要容易得多。

## 2. 转动惯量的展开

欧拉将 $|\mathbf{r}_i \times \hat{\mathbf{n}}|^2$ 展开。利用恒等式 $|\mathbf{a} \times \mathbf{b}|^2 = |\mathbf{a}|^2|\mathbf{b}|^2 - (\mathbf{a}\cdot\mathbf{b})^2$：

$$|\mathbf{r}_i \times \hat{\mathbf{n}}|^2 = r_i^2 - (\mathbf{r}_i \cdot \hat{\mathbf{n}})^2$$

设 $\hat{\mathbf{n}} = (\alpha, \beta, \gamma)$（方向余弦），则：
$$(\mathbf{r}_i \cdot \hat{\mathbf{n}})^2 = (x_i\alpha + y_i\beta + z_i\gamma)^2$$
$$= x_i^2\alpha^2 + y_i^2\beta^2 + z_i^2\gamma^2 + 2x_iy_i\alpha\beta + 2x_iz_i\alpha\gamma + 2y_iz_i\beta\gamma$$

因此：
$$I(\hat{\mathbf{n}}) = A\alpha^2 + B\beta^2 + C\gamma^2 - 2D\alpha\beta - 2E\alpha\gamma - 2F\beta\gamma$$

其中：
$$A = \sum_i m_i(y_i^2 + z_i^2), \quad B = \sum_i m_i(x_i^2 + z_i^2), \quad C = \sum_i m_i(x_i^2 + y_i^2)$$

$$D = \sum_i m_i x_i y_i, \quad E = \sum_i m_i x_i z_i, \quad F = \sum_i m_i y_i z_i$$

> **批注**：这就是惯量张量的完整表达式！用现代矩阵记号：
> $$I(\hat{\mathbf{n}}) = \hat{\mathbf{n}}^T \cdot \mathbf{I} \cdot \hat{\mathbf{n}}$$
>
> 其中惯量张量矩阵为：
> $$\mathbf{I} = \begin{pmatrix} A & -D & -E \\ -D & B & -F \\ -E & -F & C \end{pmatrix}$$
>
> 对角元素 $A, B, C$ 称为**转动惯量**（moments of inertia），非对角元素 $D, E, F$ 称为**惯量积**（products of inertia）。欧拉虽然不使用矩阵符号，但他完全理解这个二次型的数学结构。

## 3. 转动惯量的定义

欧拉对六个量给出了明确的定义：

- **$A = \sum m_i(y_i^2 + z_i^2)$**：绕 $x$ 轴的转动惯量
- **$B = \sum m_i(x_i^2 + z_i^2)$**：绕 $y$ 轴的转动惯量
- **$C = \sum m_i(x_i^2 + y_i^2)$**：绕 $z$ 轴的转动惯量
- **$D = \sum m_i x_i y_i$**：关于 $x$-$y$ 平面的惯量积
- **$E = \sum m_i x_i z_i$**：关于 $x$-$z$ 平面的惯量积
- **$F = \sum m_i y_i z_i$**：关于 $y$-$z$ 平面的惯量积

> **批注**：这六个量完整地刻画了刚体关于某一点 $O$ 的惯性特性。给定这六个量，可以计算绕通过 $O$ 点的任意方向轴的转动惯量。转动惯量（对角项）总是正的，惯量积（非对角项）可以是正的、负的或零。

## 4. 惯量椭球

欧拉引入了一个优美的几何表示——**惯量椭球**（又称"柯西惯量椭球"）。

对于通过 $O$ 点的每个方向 $\hat{\mathbf{n}}$，在该方向上取一个点，使得该点到 $O$ 的距离与 $\frac{1}{\sqrt{I(\hat{\mathbf{n}})}}$ 成正比。所有这样的点构成一个椭球面。

等价地，惯量椭球的方程为：
$$Ax^2 + By^2 + Cz^2 - 2Dxy - 2Exz - 2Fyz = 1$$

> **批注**：惯量椭球是理解刚体惯性特性的最直观的几何工具。椭球的长轴对应于转动惯量最小的方向（最容易旋转的轴），短轴对应于转动惯量最大的方向（最难旋转的轴）。椭球的形状完全由质量分布决定——它"编码"了刚体的惯性信息。

## 5. 连续体的转动惯量

对于连续分布的刚体，所有的求和都变为积分：

$$A = \int (y^2 + z^2)\,dm = \int (y^2 + z^2)\rho(\mathbf{r})\,dV$$
$$B = \int (x^2 + z^2)\,dm$$
$$C = \int (x^2 + y^2)\,dm$$
$$D = \int xy\,dm, \quad E = \int xz\,dm, \quad F = \int yz\,dm$$

欧拉计算了几个具有对称性的物体的惯量分量：

### 均匀长方体（边长 $a, b, c$，关于质心）
$$A = \frac{m}{12}(b^2 + c^2), \quad B = \frac{m}{12}(a^2 + c^2), \quad C = \frac{m}{12}(a^2 + b^2)$$
$$D = E = F = 0$$（由于对称性）

### 均匀球体（半径 $R$，关于球心）
$$A = B = C = \frac{2}{5}mR^2, \quad D = E = F = 0$$

### 一般物体
对于没有特殊对称性的物体，惯量积 $D, E, F$ 通常不为零。

> **批注**：对称性导致惯量积为零——这是一个非常重要的观察。如果物体关于某个坐标平面有镜像对称性，那么相应的惯量积为零。例如，如果物体关于 $x$-$y$ 平面对称，则 $E = F = 0$。欧拉对这个观察的系统化处理，为他后面发现主轴定理做了直接准备。

## 6. 坐标变换下转动惯量的变换

欧拉研究了当坐标系旋转时，六个惯量分量如何变换。

他发现：
- 转动惯量（对角元素）和惯量积（非对角元素）在坐标变换下按照特定的规则变换
- 存在某些特殊的坐标系，在其中惯量积全部为零
- 在这些特殊坐标系中，转动惯量的表达式最为简洁

> **批注**：这正是"对角化实对称矩阵"的问题！用现代语言说，欧拉在研究一个实对称矩阵 $\mathbf{I}$ 在正交变换下的标准形式。他知道（至少在计算层面）可以通过选择适当的坐标系来消除所有的惯量积。下一章将给出这个结果的严格证明。

## 本章小结

正文第6章建立了转动惯量的完整数学理论：

1. **方向依赖性**：$I(\hat{\mathbf{n}}) = \sum m_i |\mathbf{r}_i \times \hat{\mathbf{n}}|^2$
2. **二次型表示**：$I(\hat{\mathbf{n}}) = A\alpha^2 + B\beta^2 + C\gamma^2 - 2D\alpha\beta - 2E\alpha\gamma - 2F\beta\gamma$
3. **六个惯量分量**：三个转动惯量 $A, B, C$ 和三个惯量积 $D, E, F$
4. **惯量椭球**：转动惯量的几何表示
5. **对称性与惯量积的关系**

> **批注**：这一章的核心发现是：刚体的惯性特性不能用一个简单的数（如总质量）来描述，而需要一个更复杂的数学对象——惯量张量（六个分量）。这个张量包含了关于质量分布的全部信息。下一章将证明，通过选择适当的坐标系（主轴坐标系），这个张量可以简化为最简洁的形式。
