# 正文第19章：圆柱体在水平面上的运动

*De motu corporum cylindricorum in plano horizontali*

---

## 本章概述

欧拉在正文最后一章讨论了圆柱体在水平面上的运动——滚动、滑动及其组合。这个问题展示了刚体动力学中平动与转动的耦合，以及摩擦力的关键作用。

> **批注**：圆柱体的滚动是刚体动力学中最具教育意义的例子之一。它既足够简单（旋转对称体，二维运动），又足够丰富（平动-转动耦合，摩擦的必要作用），是理解刚体运动"发生机制"的理想模型。

---

## 1. 圆柱体的惯量性质

均匀圆柱体（半径 $R$，质量 $M$）的主转动惯量：
- 绕对称轴（长轴方向）：$I_3 = \frac{1}{2}MR^2$
- 绕过质心垂直于对称轴的轴：$I_1 = I_2 = \frac{1}{4}MR^2 + \frac{1}{12}ML^2$（$L$ 为柱体长度）

对于圆盘（$L \to 0$）：
- $I_3 = \frac{1}{2}MR^2$
- $I_1 = I_2 = \frac{1}{4}MR^2$

## 2. 纯滚动条件

圆柱体在水平面上纯滚动的条件是接触点无滑动：
$$V_{\text{cm}} = R\omega$$

其中 $V_{\text{cm}}$ 是质心速度，$\omega$ 是角速度，$R$ 是半径。

## 3. 纯滚动下的运动方程

在纯滚动条件下，平动和转动通过约束条件耦合：

$$Ma = f$$
$$I_3\dot{\omega} = fR$$

利用 $a = R\dot{\omega}$：
$$M \cdot R\dot{\omega} = \frac{I_3\dot{\omega}}{R}$$

$$\dot{\omega} = \frac{fR}{I_3 + MR^2}$$

对于均匀圆柱体（$I_3 = \frac{1}{2}MR^2$）：
$$\dot{\omega} = \frac{fR}{\frac{3}{2}MR^2} = \frac{2f}{3MR}$$

> **批注**：注意，摩擦力 $f$ 是启动和维持滚动的关键。没有摩擦力（$f = 0$），圆柱体只滑动不滚动。摩擦力同时产生质心的加速度和角加速度，两者通过纯滚动条件关联。加速度是同样力作用于无摩擦面上时的 $2/3$——因为一部分力"用于"旋转而非平动。

## 4. 滚动与滑动的转换

欧拉分析了滚动和滑动之间的转换：

- 当需要的摩擦力超过最大静摩擦力时（$f > \mu_s Mg$），纯滚动条件被破坏，圆柱体开始滑动。
- 滑动时，动摩擦力 $f = \mu_k Mg$ 使圆柱体逐渐从滑动过渡到纯滚动。

**过渡条件**：从滑动到纯滚动发生在 $V_{\text{cm}} = R\omega$ 时。

> **批注**：滚动-滑动的转换是轮胎力学的基础。汽车加速时，如果驱动力矩太大，轮胎会打滑（空转）；刹车时，如果制动力矩太大，轮胎会抱死（滑动而非滚动）。防抱死制动系统（ABS）正是为了防止这种情况——它通过快速调节制动力，使轮胎保持在滚动（而非滑动）状态，从而保持方向控制能力。

## 5. 圆柱体的动能

纯滚动时的总动能：
$$T = \frac{1}{2}MV_{\text{cm}}^2 + \frac{1}{2}I_3\omega^2 = \frac{1}{2}MV^2 + \frac{1}{4}MV^2 = \frac{3}{4}MV^2$$

其中 $V = V_{\text{cm}} = R\omega$。

> **批注**：滚动圆柱体的动能中有 $2/3$ 是平动的，$1/3$ 是转动的。这意味着一个滚动的圆柱体比一个以相同速度滑动的无旋转物体多出 $50\%$ 的动能。

## 6. 斜面上的滚动

欧拉讨论了圆柱体在斜面上的滚动。

沿斜面方向的运动方程：
$$Ma = Mg\sin\alpha - f$$
$$I_3\dot{\omega} = fR$$

纯滚动条件 $a = R\dot{\omega}$：
$$a = \frac{g\sin\alpha}{1 + I_3/(MR^2)} = \frac{2}{3}g\sin\alpha$$

摩擦力的值：
$$f = \frac{I_3 Mg\sin\alpha}{I_3 + MR^2} = \frac{1}{3}Mg\sin\alpha$$

纯滚动的条件：$f \leq \mu_s Mg\cos\alpha$，即：
$$\tan\alpha \leq 3\mu_s$$

> **批注**：这个结果意味着，只有当斜面倾角不太大（$\tan\alpha \leq 3\mu_s$）时，圆柱体才能纯滚动。如果倾角太大，圆柱体会又滚又滑。这是一个可以简单验证的实验——在冰面（$\mu_s$ 很小）上，圆柱体几乎不可能纯滚动。

---

## 本章小结

正文第19章讨论了圆柱体在平面上的运动：

1. **纯滚动条件**：$V = R\omega$
2. **运动方程**：平动与转动通过摩擦力耦合
3. **滚动-滑动转换**：摩擦力不足时发生
4. **斜面上的滚动**：加速度 $a = \frac{2}{3}g\sin\alpha$
5. **纯滚动条件**：$\tan\alpha \leq 3\mu_s$

> **批注**：正文部分至此全部完成。从第1章的刚体定义到第19章的具体应用，欧拉用了19章建立了一个完整的刚体动力学体系。附录将讨论摩擦对刚体运动的影响——这是欧拉对理论完整性的追求的又一体现。
