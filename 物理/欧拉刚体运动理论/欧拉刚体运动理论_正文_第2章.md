# 正文第2章：绕固定轴的无扰动旋转

*De motu gyratorio circa axem fixum a nullis viribus turbato*

---

## 本章概述

欧拉从最简单的旋转运动开始：刚体绕固定轴的匀角速度旋转，不受任何外力矩的扰动。这个看似简单的情形蕴含了旋转运动最基本的特征——角速度、角动量和旋转惯性的概念。

> **批注**：欧拉的推理策略是"从最简单的情况开始，逐步增加复杂性"。这与他在引言中的策略一致：先处理无力的情况（惯性运动），再引入力的作用。在这里，"惯性旋转"对应于直线运动中的"惯性运动"——没有外力矩时，角动量守恒。

---

## 1. 固定轴旋转的描述

假设刚体绕一条固定轴（取为 $z$ 轴）旋转。

刚体上的每个质点都在做圆周运动，圆心在旋转轴上。第 $i$ 个质点的运动可以描述为：

$$x_i = r_i\cos(\theta_i), \quad y_i = r_i\sin(\theta_i), \quad z_i = \text{常数}$$

其中 $r_i$ 是到旋转轴的距离，$\theta_i$ 是角度。

由于刚体是刚性的，所有质点以**相同的角速度**旋转：
$$\omega = \frac{d\theta}{dt}$$

每个质点的线速度为：
$$v_i = r_i\omega$$

方向沿圆周的切线方向。

> **批注**：角速度 $\omega$ 是描述刚体旋转的基本量。它的特点是整个刚体共享同一个角速度——无论质点距离旋转轴多远，角速度都相同。这与线速度不同——线速度 $v = r\omega$ 随距轴的距离 $r$ 而增大。

## 2. 角动量的引入

欧拉引入了角动量（momentum gyratorium）的概念。

对于绕 $z$ 轴旋转的刚体，第 $i$ 个质点对 $z$ 轴的角动量为：
$$L_i = m_i r_i^2 \omega$$

整个刚体的角动量为：
$$L = \sum_i m_i r_i^2 \omega = \omega \sum_i m_i r_i^2$$

> **批注**：角动量的引入是旋转力学中最关键的一步。欧拉发现，在旋转运动中，与牛顿第二定律 $\mathbf{F} = m\mathbf{a}$ 对应的不是"力等于质量乘以角加速度"，而是"力矩等于角动量的变化率"。角动量 $L = I\omega$ 在旋转中的地位，相当于动量 $p = mv$ 在平移中的地位。

## 3. 无力矩时的角速度守恒

**命题**：如果不受任何外力矩作用，绕固定轴旋转的刚体保持恒定的角速度。

**证明**：没有外力矩意味着没有外部原因改变刚体的旋转状态。由于刚体是刚性的，$\sum_i m_i r_i^2$ 是常数（各质点间的距离不变），所以：

$$L = \omega \sum_i m_i r_i^2 = \text{常数}$$

因此 $\omega = \text{常数}$。

$\blacksquare$

> **批注**：这就是旋转运动的惯性定律——没有外力矩时，角速度保持恒定。它与直线运动的惯性定律（没有外力时，速度保持恒定）完全平行。在实际情况中，这就是为什么地球在不受明显外力矩的情况下能够保持几乎恒定的自转角速度。

## 4. 角速度的方向

欧拉将角速度推广为矢量 $\boldsymbol{\omega}$，方向沿旋转轴，指向由右手定则确定。

对于绕 $z$ 轴的旋转：
$$\boldsymbol{\omega} = (0, 0, \omega)$$

每个质点的线速度可以用叉积表示：
$$\mathbf{v}_i = \boldsymbol{\omega} \times \mathbf{r}_i$$

> **批注**：角速度矢量的引入看似自然，但实际上隐藏着一个微妙的问题：有限角度的旋转不满足交换律（先绕 $x$ 轴转90°再绕 $y$ 轴转90°，与反过来做的结果不同）。但无穷小旋转（以及角速度）确实满足交换律，可以表示为矢量。欧拉在第5章和第9章中将深入讨论这个问题。

## 5. 旋转动能

绕固定轴旋转的刚体的动能为：
$$T = \sum_i \frac{1}{2}m_i v_i^2 = \sum_i \frac{1}{2}m_i r_i^2\omega^2 = \frac{1}{2}\omega^2 \sum_i m_i r_i^2$$

令 $I = \sum_i m_i r_i^2$（这就是**转动惯量**，将在下一章正式定义），则：

$$\boxed{T = \frac{1}{2}I\omega^2}$$

> **批注**：这个公式与平移动能 $T = \frac{1}{2}mv^2$ 完全平行：
> - 质量 $m$ → 转动惯量 $I$
> - 速度 $v$ → 角速度 $\omega$
>
> 这种平移与旋转的平行关系贯穿整个刚体力学。欧拉是第一个系统性地建立这种平行关系的人。

## 6. 旋转运动与直线运动的类比

欧拉在本章中建立了一个重要的类比表：

| 直线运动 | 旋转运动 |
|---|---|
| 质量 $m$ | 转动惯量 $I = \sum m_i r_i^2$ |
| 速度 $v$ | 角速度 $\omega$ |
| 动量 $p = mv$ | 角动量 $L = I\omega$ |
| 力 $F$ | 力矩 $M$ |
| 动能 $\frac{1}{2}mv^2$ | 旋转动能 $\frac{1}{2}I\omega^2$ |
| $F = ma = m\dot{v}$ | $M = I\dot{\omega}$（绕固定轴） |
| $F = \dot{p}$ | $M = \dot{L}$ |
| 惯性定律：$F=0 \Rightarrow v=\text{const}$ | 惯性定律：$M=0 \Rightarrow \omega=\text{const}$ |

> **批注**：这个类比表是刚体力学教育中最有力的工具之一。它让学习者在已经掌握的直线运动概念基础上，直接理解对应的旋转概念。但欧拉也清楚地意识到，这个类比只有在绕固定轴旋转时才成立——当旋转轴可以自由变化时，情况要复杂得多。这正是后面章节要处理的问题。

## 本章小结

正文第2章建立了绕固定轴旋转的基本框架：

1. **角速度**：整个刚体共享同一个角速度
2. **角动量**：$L = I\omega$，其中 $I = \sum m_i r_i^2$
3. **惯性旋转**：无力矩时角速度恒定
4. **旋转动能**：$T = \frac{1}{2}I\omega^2$
5. **平移-旋转类比表**

> **批注**：这一章虽然只处理了最简单的旋转（绕固定轴、匀角速度），但它引入的概念——角速度、角动量、转动惯量——是整个刚体动力学的基石。在后面的章节中，欧拉将逐步放松限制：允许外力矩（第3-4章），允许旋转轴变化（第5章），引入完整的惯量张量（第6章），最终得到最一般的旋转运动方程（第10章）。
