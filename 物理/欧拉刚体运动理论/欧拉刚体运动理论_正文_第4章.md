# 正文第4章：受任意力作用下的旋转

*De motu gyratorio corporis a viribus quibuscunque sollicitati*

---

## 本章概述

这一章中，欧拉将第2-3章的结果推广到更一般的受力情况。他处理了各种类型的力（重力、弹性力、碰撞力等）对绕固定轴旋转的刚体的影响，并研究了振荡运动（物理摆）等具体问题。

> **批注**：这一章看似只是第3章的简单推广，但实际上它引入了一个关键的新概念——旋转运动的"平衡"和"稳定"。这些概念在后面的陀螺理论和天体力学中极为重要。

---

## 1. 一般力作用下的旋转方程

对于绕固定轴旋转的刚体，受任意力系作用，运动方程仍为：
$$I\dot{\omega} = M_{\text{外}}$$

但现在 $M_{\text{外}}$ 可以是角度 $\theta$ 的函数（如重力力矩）、角速度 $\omega$ 的函数（如阻力力矩）或时间的函数。

### 重力力矩
如果刚体受重力作用，旋转轴水平放置，则重力力矩为：
$$M = -Mgd\sin\theta$$

其中 $d$ 是质心到旋转轴的距离，$\theta$ 是从平衡位置量起的角度。负号表示力矩倾向于使 $\theta$ 减小（恢复力矩）。

运动方程：
$$I\ddot{\theta} = -Mgd\sin\theta$$

> **批注**：这个方程是物理摆（复摆）的运动方程。当角度 $\theta$ 很小时，$\sin\theta \approx \theta$，方程变为简谐运动方程：
> $$I\ddot{\theta} = -Mgd\theta$$
> 解为 $\theta(t) = \theta_0\cos(\Omega t + \phi)$，其中 $\Omega = \sqrt{Mgd/I}$。

## 2. 物理摆（复摆）

欧拉详细分析了物理摆的行为。

### 小角度近似
当 $\theta$ 很小时，运动方程线性化：
$$\ddot{\theta} + \frac{Mgd}{I}\theta = 0$$

这是简谐运动，周期为：
$$T = 2\pi\sqrt{\frac{I}{Mgd}}$$

### 等效单摆长度
物理摆的振动周期与一个长度为 $\ell$ 的单摆相同，其中：
$$\ell = \frac{I}{Md}$$

这个 $\ell$ 称为**等效单摆长度**（或简化长度）。

利用平行轴定理 $I = I_{\text{cm}} + Md^2$：
$$\ell = \frac{I_{\text{cm}} + Md^2}{Md} = \frac{I_{\text{cm}}}{Md} + d$$

> **批注**：等效单摆长度的概念使得物理摆的运动可以与单摆直接比较。注意 $\ell > d$——等效单摆的悬点比物理摆的旋转轴更低。在 $\ell$ 处有一个特殊的点，称为**振动中心**（center of oscillation）。

### 可逆性（悬点与振动中心的互换性）

欧拉证明了一个优美的性质：如果将旋转轴从原来的位置移到振动中心，新的振动周期与原来相同。

> **批注**：这个"可逆性"是物理摆的一个迷人性质。它意味着旋转轴和振动中心可以互换而不改变周期。这个性质被用来精确测量重力加速度——通过找到同一个摆的两个等周期悬点位置，可以确定等效长度 $\ell$，再通过周期公式 $T = 2\pi\sqrt{\ell/g}$ 求出 $g$。

## 3. 大角度摆动

对于大角度摆动，$\sin\theta$ 不能近似为 $\theta$，运动方程：
$$\ddot{\theta} + \frac{Mgd}{I}\sin\theta = 0$$

是非线性的。

欧拉用首次积分（能量守恒）来分析：
$$\frac{1}{2}I\dot{\theta}^2 + Mgd(1 - \cos\theta) = E = \text{常数}$$

由此得到：
$$\dot{\theta} = \pm\sqrt{\frac{2}{I}(E - Mgd(1 - \cos\theta))}$$

周期可以通过积分计算：
$$T = 4\sqrt{\frac{I}{2Mgd}}\int_0^{\theta_0}\frac{d\theta}{\sqrt{\cos\theta - \cos\theta_0}}$$

其中 $\theta_0$ 是最大摆角。

这个积分是**第一类完全椭圆积分**。

> **批注**：椭圆积分是欧拉在数学上做出过重大贡献的领域。他发现了椭圆积分的加法定理，这是18世纪分析数学最深刻的结果之一。在这里，物理摆的大角度运动自然地引出了椭圆积分——理论与数学的完美结合。

## 4. 有阻力时的旋转运动

当旋转运动受到与角速度成正比的阻力力矩时：
$$I\dot{\omega} = M_{\text{外}} - b\omega$$

对于物理摆加阻力：
$$I\ddot{\theta} + b\dot{\theta} + Mgd\sin\theta = 0$$

小角度近似下：
$$I\ddot{\theta} + b\dot{\theta} + Mgd\theta = 0$$

这是阻尼谐振子的旋转版本，具有与引言中讨论的线形阻尼谐振子完全相同的解。

## 5. 旋转平衡的稳定性

欧拉讨论了旋转平衡的概念。如果刚体绕固定轴旋转并受重力作用：

- **稳定平衡**：当质心在旋转轴正下方时（$\theta = 0$）。微小的扰动导致小幅振荡。
- **不稳定平衡**：当质心在旋转轴正上方时（$\theta = \pi$）。微小的扰动导致大幅偏离。

> **批注**：旋转平衡的稳定性概念在工程中极为重要。桥梁、建筑物、机械的稳定性分析都涉及类似的概念。欧拉对稳定性问题的关注预示了后来整个稳定性理论的发展方向。

---

## 本章小结

正文第4章将绕固定轴的旋转理论应用于具体问题：

1. **物理摆**：$\ddot{\theta} + (Mgd/I)\sin\theta = 0$
2. **等效单摆长度**：$\ell = I/(Md)$
3. **可逆性**：悬点与振动中心可以互换
4. **大角度摆动**：引出椭圆积分
5. **阻尼旋转**：旋转版本的阻尼谐振子
6. **稳定性**：稳定与不稳定平衡

> **批注**：到这一章为止，欧拉已经穷尽了绕固定轴旋转的所有主要问题。从下一章开始，他将面对一个根本性的挑战——当旋转轴不是固定的时候，会发生什么？这需要全新的数学工具（瞬时旋转轴、惯量张量、主轴、欧拉角），而正是这些工具的发明构成了欧拉最伟大的贡献。
