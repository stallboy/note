# 正文第12章：有固定点的刚体运动

*De motu corporis rigidi, cuius punctum quodcunque fixum manet*

---

## 本章概述

这一章中，欧拉深入研究了绕固定点旋转的刚体——这是刚体动力学中最经典、应用最广泛的问题之一。陀螺、儿童玩具、地球的自转、天文望远镜的指向——都是有固定点旋转的实例。

> **批注**：有固定点的刚体只有3个自由度（三个旋转角），比自由刚体的6个自由度简单。但它已经包含了旋转运动最丰富的现象——进动、章动、稳定性变化等。欧拉在这里的分析为后来的陀螺理论和天体力学奠定了基础。

---

## 1. 自由旋转的守恒量

当没有外力矩（$\mathbf{M} = 0$）时，欧拉旋转方程给出两个守恒量：

### 角动量守恒
$$\mathbf{L} = \text{常数矢量（在空间坐标系中）}$$

分量形式：
$$L_1^2 + L_2^2 + L_3^2 = I_1^2\omega_1^2 + I_2^2\omega_2^2 + I_3^2\omega_3^2 = L^2 = \text{常数}$$

### 动能守恒
$$T = \frac{1}{2}(I_1\omega_1^2 + I_2\omega_2^2 + I_3\omega_3^2) = \text{常数}$$

> **批注**：两个守恒量意味着角速度矢量 $\boldsymbol{\omega}$ 的端点被限制在两个曲面的交线上。这两个曲面分别是：
> - **能量椭球**（惯量椭球）：$I_1\omega_1^2 + I_2\omega_2^2 + I_3\omega_3^2 = 2T$
> - **角动量球**：$I_1^2\omega_1^2 + I_2^2\omega_2^2 + I_3^2\omega_3^2 = L^2$
>
> 交线是空间中的一条曲线，角速度的端点沿着这条曲线运动——这就是自由旋转的几何描述。

## 2. 潘索几何描述

欧拉的自由旋转理论后来被潘索（Poinsot, 1834）给出了优美的几何解释：

- 惯量椭球在角速度矢量 $\boldsymbol{\omega}$ 的方向上的切平面是固定的（垂直于角动量 $\mathbf{L}$）。
- 惯量椭球在这个固定切平面上无滑动地滚动。
- 滚动的角速度就是刚体的角速度 $\boldsymbol{\omega}$。

> **批注**：潘索的几何描述是欧拉分析结果的直观化。它提供了一个优美的物理图像：惯量椭球（固连于刚体）在一个固定平面上滚动，滚动过程中椭球与平面的接触点描绘出一条曲线。这条曲线在椭球面上形成本体极迹（polhode），在固定平面上形成空间极迹（herpolhode）。

## 3. 绕主轴旋转的稳定性

欧拉分析了绕三个主轴旋转的稳定性：

### 绕最大惯量轴（$I_3$）旋转
如果初始角速度几乎沿 $I_3$ 轴方向（$\omega_3 \gg \omega_1, \omega_2$），则：

由欧拉方程（$\mathbf{M} = 0$）：
$$I_1\dot{\omega}_1 = (I_2 - I_3)\omega_2\omega_3 \approx -(I_3 - I_2)\omega_2\omega_3$$

设 $\omega_1, \omega_2$ 很小，$\omega_3 \approx \Omega$（常数），线性化：
$$I_1\ddot{\omega}_1 = (I_2 - I_3)\dot{\omega}_2\Omega = \frac{(I_2 - I_3)(I_3 - I_1)}{I_2}\Omega^2\omega_1$$

由于 $I_3 > I_1$ 且 $I_3 > I_2$，系数为负，$\omega_1$ 做简谐振荡：
$$\omega_1 \propto \cos(\lambda t)$$

**结论：绕最大惯量轴的旋转是稳定的。** 微小扰动只引起小幅振荡。

### 绕最小惯量轴（$I_1$）旋转
类似分析表明：**绕最小惯量轴的旋转也是稳定的。**

### 绕中间惯量轴（$I_2$）旋转
线性化后，系数为正，解为指数增长：
$$\omega_1 \propto e^{\lambda t}$$

**结论：绕中间惯量轴的旋转是不稳定的。** 微小扰动将指数增长，旋转轴大幅偏离。

> **批注**：这就是著名的**网球拍效应**（又称德安纳姆效应或中间轴定理）。你可以自己验证：把一个网球拍（或一本书）抛向空中，让它绕三个不同的轴旋转。绕长轴和短轴的旋转是稳定的，但绕中间轴的旋转会翻来覆去——这就是不稳定性的直观体现。欧拉是第一个从理论上分析这个现象的人。

## 4. 对称体的自由旋转

对于旋转对称体（$I_1 = I_2$），自由旋转有解析解。

欧拉方程（$\mathbf{M} = 0$）简化为：
$$I_1\dot{\omega}_1 = (I_1 - I_3)\omega_2\omega_3$$
$$I_1\dot{\omega}_2 = (I_3 - I_1)\omega_1\omega_3$$
$$I_3\dot{\omega}_3 = 0$$

由第三个方程：$\omega_3 = \Omega = \text{常数}$。

代入前两个方程，引入 $\omega_+ = \omega_1 + i\omega_2$：
$$I_1\dot{\omega}_+ = (I_1 - I_3)\Omega(-\omega_2 + i\omega_1) = i(I_3 - I_1)\Omega\omega_+$$

解为：
$$\omega_+ = Ae^{i\lambda t}$$

其中 $\lambda = \frac{(I_3 - I_1)}{I_1}\Omega$。

**结论**：在对称体的自由旋转中，角速度在对称轴上的分量 $\omega_3$ 为常数，而在垂直于对称轴的平面内的分量 $(\omega_1, \omega_2)$ 以角频率 $\lambda$ 做匀速旋转。

> **批注**：这意味着角速度矢量在刚体内部画出一个圆锥面——角速度绕对称轴匀速旋转，频率为 $\lambda = (I_3 - I_1)\Omega/I_1$。在空间中看，角速度沿固定方向（角动量方向）和绕对称轴旋转的叠加，描绘出更复杂的轨迹。

---

## 本章小结

正文第12章深入分析了有固定点的刚体运动：

1. **自由旋转的守恒量**：角动量和动能
2. **几何描述**：惯量椭球在固定平面上滚动
3. **稳定性分析**：绕最大和最小惯量轴稳定，中间轴不稳定
4. **对称体的解析解**：角速度绕对称轴旋转

> **批注**：稳定性分析是这一章最深刻的贡献。它解释了为什么一些旋转运动是稳定的（如陀螺绕其对称轴旋转），而另一些则不是（如网球拍绕中间轴旋转）。这个结果在天体力学中极为重要——小行星的自转稳定性、航天器的姿态控制等，都依赖于欧拉稳定性分析的基本思想。
