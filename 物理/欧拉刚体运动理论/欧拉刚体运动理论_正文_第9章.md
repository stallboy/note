# 正文第9章：多个旋转的合成

*De motu corporum rigidorum, quotuscunque motus gyratorii compositorum*

---

## 本章概述

这一章中，欧拉研究了多个旋转运动的叠加问题。他证明了角速度的矢量合成定律，并展示了如何将复杂的旋转运动分解为绕三个坐标轴的旋转分量。这为下一章推导欧拉旋转方程做了最后的准备。

> **批注**：旋转的合成看似简单，实际上涉及一个微妙的问题——有限角度的旋转不满足交换律（先绕 $x$ 轴转再绕 $y$ 轴转，与反过来做的结果不同）。但欧拉证明了，角速度（无穷小旋转的时间变化率）确实满足矢量加法。这个区分是刚体运动学的关键。

---

## 1. 两个旋转的合成

假设刚体同时参与两个旋转运动：
- 旋转1：角速度 $\boldsymbol{\omega}_1$，绕通过点 $O$ 的某条轴
- 旋转2：角速度 $\boldsymbol{\omega}_2$，绕通过点 $O$ 的另一条轴

**命题**：这两个旋转的合效果等价于一个角速度为 $\boldsymbol{\omega} = \boldsymbol{\omega}_1 + \boldsymbol{\omega}_2$ 的旋转。

### 证明

由速度的叠加原理，刚体上位置为 $\mathbf{r}$ 的点的速度为：
$$\mathbf{v} = \mathbf{v}_1 + \mathbf{v}_2 = \boldsymbol{\omega}_1 \times \mathbf{r} + \boldsymbol{\omega}_2 \times \mathbf{r} = (\boldsymbol{\omega}_1 + \boldsymbol{\omega}_2) \times \mathbf{r} = \boldsymbol{\omega} \times \mathbf{r}$$

叉积的线性性保证了角速度的加法。$\blacksquare$

> **批注**：这个证明的关键是叉积的分配律。由于 $\mathbf{v} = \boldsymbol{\omega} \times \mathbf{r}$ 是一个线性关系，多个角速度的效果可以直接叠加。但要注意，这只对通过同一点的旋转轴才成立——如果两个旋转轴不交于同一点，情况要复杂得多。

## 2. 角速度的分量分解

在任何时刻，刚体的角速度可以沿三个主轴方向分解：

$$\boldsymbol{\omega} = \omega_1\hat{\mathbf{e}}_1 + \omega_2\hat{\mathbf{e}}_2 + \omega_3\hat{\mathbf{e}}_3$$

其中 $\hat{\mathbf{e}}_1, \hat{\mathbf{e}}_2, \hat{\mathbf{e}}_3$ 是三个主轴方向的单位矢量，$\omega_1, \omega_2, \omega_3$ 是角速度的三个分量。

> **批注**：这个分解的意义在于，每个分量 $\omega_i$ 代表绕第 $i$ 个主轴的旋转的快慢。三个分量 $(\omega_1, \omega_2, \omega_3)$ 完全确定了刚体在任一瞬时的旋转状态。

## 3. 角动量的分量表示

在主轴坐标系中，角动量的分量表示极为简洁：

$$\mathbf{L} = I_1\omega_1\hat{\mathbf{e}}_1 + I_2\omega_2\hat{\mathbf{e}}_2 + I_3\omega_3\hat{\mathbf{e}}_3$$

分量形式：
$$L_1 = I_1\omega_1, \quad L_2 = I_2\omega_2, \quad L_3 = I_3\omega_3$$

> **批注**：这就是主轴坐标系的优势！在一般坐标系中，角动量与角速度的关系是 $\mathbf{L} = \mathbf{I}\boldsymbol{\omega}$（一个矩阵乘法），有交叉项。但在主轴坐标系中，惯量张量是对角的，关系简化为分量直接相乘。每个角动量分量只依赖于对应的角速度分量——没有"耦合"。

## 4. 旋转动能的分量表示

旋转动能同样被简化：

$$T_{\text{旋转}} = \frac{1}{2}\boldsymbol{\omega} \cdot \mathbf{L} = \frac{1}{2}(I_1\omega_1^2 + I_2\omega_2^2 + I_3\omega_3^2)$$

> **批注**：这是最简洁的旋转动能表达式。三个平方项，每项对应一个主轴方向的旋转，没有交叉项。如果 $I_1 = I_2 = I_3$（球体），则 $T = \frac{1}{2}I\omega^2$，与绕固定轴的情况完全一样。

## 5. 连续旋转的参数化

欧拉讨论了如何用有限个参数来描述连续的旋转运动。

一个三维旋转有3个自由度，因此可以用3个独立参数来描述。欧拉考虑了多种参数化方式：

### 方向余弦（9个量，6个约束）
用固连坐标系的三个轴在空间坐标系中的方向余弦。9个元素，6个正交约束，3个独立。

### 轴-角表示（3+1=4个量，1个约束）
用旋转轴的方向（2个角度）和旋转角（1个量）。共3个独立量。

### 欧拉角（3个量）
用三个特定的角度来描述取向——这是最常用的参数化方式，将在下一章详细介绍。

> **批注**：参数化的选择影响了运动方程的形式和求解的难度。欧拉角的引入（下一章）是欧拉的一个天才选择——它将一个一般的旋转分解为三个绕坐标轴的简单旋转的序列，使得运动方程的建立变得系统化。

## 6. 角加速度

欧拉定义了角加速度——角速度的时间变化率：

$$\boldsymbol{\alpha} = \frac{d\boldsymbol{\omega}}{dt}\bigg|_{\text{空间}}$$

在固连坐标系中，利用时间导数变换公式：
$$\boldsymbol{\alpha} = \frac{d\boldsymbol{\omega}}{dt}\bigg|_{\text{固连}} + \boldsymbol{\omega} \times \boldsymbol{\omega} = \dot{\boldsymbol{\omega}}$$

（由于 $\boldsymbol{\omega} \times \boldsymbol{\omega} = 0$，角加速度在两个坐标系中是一样的。）

分量形式：
$$\alpha_1 = \dot{\omega}_1, \quad \alpha_2 = \dot{\omega}_2, \quad \alpha_3 = \dot{\omega}_3$$

> **批注**：角加速度在固连坐标系和空间坐标系中相同，这是一个特殊的简化。它不适用于一般的矢量——对于一般的矢量 $\mathbf{A}$，空间导数和固连导数之间差一个 $\boldsymbol{\omega} \times \mathbf{A}$ 项。但对角速度自身，这个差项恰好为零。

---

## 本章小结

正文第9章完成了推导旋转运动方程前的所有准备：

1. **角速度的矢量加法**：$\boldsymbol{\omega} = \boldsymbol{\omega}_1 + \boldsymbol{\omega}_2$
2. **分量分解**：$\boldsymbol{\omega} = \omega_1\hat{\mathbf{e}}_1 + \omega_2\hat{\mathbf{e}}_2 + \omega_3\hat{\mathbf{e}}_3$
3. **角动量分量**：$L_i = I_i\omega_i$（主轴坐标系中的简化）
4. **旋转动能**：$T = \frac{1}{2}(I_1\omega_1^2 + I_2\omega_2^2 + I_3\omega_3^2)$
5. **旋转的参数化**：用3个独立参数描述取向

> **批注**：所有必要的数学工具已经就位：角速度矢量、角动量分量、旋转动能、主轴坐标系、时间导数变换公式。下一章，欧拉将把这些工具组合起来，推导出经典力学最优美的方程之一——欧拉旋转方程。
