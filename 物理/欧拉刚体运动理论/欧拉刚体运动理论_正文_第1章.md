# 正文第1章：刚体的平移运动

*De motu progressivo corporum rigidorum*

---

## 本章概述

从这一章开始，欧拉进入全书的核心——刚体运动理论。第1章奠定了刚体力学的基石：**刚体的严格定义**和**质心运动定理**。

> **批注**：这是整个刚体力学大厦的第一块砖。欧拉在这里做了两件开创性的事情：第一，给出了刚体的严格数学定义（定义1，约第260节）；第二，证明了刚体的平移运动完全由质心的运动决定。这两个结果看似简单，但它们的严格建立需要极其精细的推理。

---

## 定义1：刚体（Corpus rigidum）

> **"Corpus rigidum vocatur, cuius partes inter se perpetuo eandem positionem servant."**
> "刚体是指其各部分之间永远保持相同相对位置的物体。"

欧拉的刚体定义意味着：在刚体的运动过程中，任意两个质点之间的距离保持不变。如果第 $i$ 个质点的位置为 $\mathbf{r}_i$，第 $j$ 个质点的位置为 $\mathbf{r}_j$，则：
$$|\mathbf{r}_i - \mathbf{r}_j| = \text{常数}$$

> **批注**：这个定义的深刻之处在于，它用纯数学语言（距离不变）定义了一个物理概念（刚性）。这种"用约束条件定义物理对象"的方法是分析力学的基本范式。刚体是最简单的非质点力学系统——它有无穷多个质点，但距离约束将自由度从无穷多个减少到了6个（3个平动 + 3个转动）。

## 定义2：质心（Centrum gravitatis sive centrum inertiae）

欧拉定义了质心——质量分布的"平均位置"：

$$\mathbf{R}_{\text{cm}} = \frac{\sum_i m_i \mathbf{r}_i}{\sum_i m_i} = \frac{1}{M}\sum_i m_i \mathbf{r}_i$$

对于连续分布的刚体：
$$\mathbf{R}_{\text{cm}} = \frac{1}{M}\int \mathbf{r}\,dm = \frac{1}{M}\int \mathbf{r}\rho(\mathbf{r})\,dV$$

其中 $M = \sum_i m_i$ 是总质量，$\rho$ 是密度。

> **批注**：欧拉是第一个系统使用质心概念的人之一。他认识到，对于刚体的整体运动，质心起着"代表"的作用——好像刚体的全部质量都集中在质心上一样。这个认识在下面的质心运动定理中得到了严格的表述。

## 定义3：平移运动（Motus progressivus）

欧拉定义了平移运动：刚体上所有点具有相同的速度。

$$\mathbf{v}_1 = \mathbf{v}_2 = \cdots = \mathbf{v}_i = \cdots = \mathbf{V}_{\text{cm}}$$

在平移运动中，刚体不旋转，只做整体移动。

> **批注**：欧拉对平移运动的定义是精确的——不是"看起来像是平移"，而是"每个点都有相同的速度"。这个精确的定义使得平移运动可以与旋转运动严格区分开来。

## 命题1：质心运动定理

**刚体质心的运动，如同将刚体的全部质量集中于质心，并将所有外力集中于该点一样。**

数学表述：设外力的总和为 $\mathbf{F}_{\text{外}} = \sum_i \mathbf{F}_i^{\text{外}}$，则：
$$M\frac{d^2\mathbf{R}_{\text{cm}}}{dt^2} = \mathbf{F}_{\text{外}}$$

### 证明

欧拉的证明如下：

对刚体中的第 $i$ 个质点，运动方程为：
$$m_i\frac{d^2\mathbf{r}_i}{dt^2} = \mathbf{F}_i^{\text{外}} + \sum_{j \neq i} \mathbf{F}_{ij}$$

其中 $\mathbf{F}_i^{\text{外}}$ 是外力，$\mathbf{F}_{ij}$ 是第 $j$ 个质点对第 $i$ 个质点的内力。

对所有质点求和：
$$\sum_i m_i\frac{d^2\mathbf{r}_i}{dt^2} = \sum_i \mathbf{F}_i^{\text{外}} + \sum_i\sum_{j \neq i} \mathbf{F}_{ij}$$

由牛顿第三定律，$\mathbf{F}_{ij} = -\mathbf{F}_{ji}$，所以所有内力成对抵消：
$$\sum_i\sum_{j \neq i} \mathbf{F}_{ij} = 0$$

因此：
$$\sum_i m_i\frac{d^2\mathbf{r}_i}{dt^2} = \sum_i \mathbf{F}_i^{\text{外}}$$

左边可以写成：
$$\sum_i m_i\frac{d^2\mathbf{r}_i}{dt^2} = \frac{d^2}{dt^2}\left(\sum_i m_i\mathbf{r}_i\right) = M\frac{d^2\mathbf{R}_{\text{cm}}}{dt^2}$$

证毕。$\blacksquare$

> **批注**：这个证明的精髓在于内力的成对抵消。由于内力满足牛顿第三定律（作用等于反作用），它们对整个系统的总效果为零。因此，刚体的整体运动（质心运动）只取决于外力。这个定理的一个直接推论是：**内力不能改变刚体的质心运动状态**。无论刚体内部发生什么（只要保持刚性），质心只对外力有响应。

## 推论1：孤立刚体的运动

如果刚体不受任何外力（$\mathbf{F}_{\text{外}} = 0$），则：
$$M\frac{d^2\mathbf{R}_{\text{cm}}}{dt^2} = 0$$
$$\frac{d\mathbf{R}_{\text{cm}}}{dt} = \mathbf{V}_0 = \text{常数}$$

**质心以恒定速度做匀速直线运动**——这是惯性原理对刚体的推广。

> **批注**：这个推论意味着，如果我们把太阳系看作一个不受外力的"刚体"（近似），则太阳系的质心在做匀速直线运动。同样，如果我们踢一个足球，足球的质心沿着一条抛物线运动（受重力作用），无论足球同时如何旋转。

## 推论2：力的简化

质心运动定理意味着，对于平移运动的讨论，我们不需要知道每个力的具体作用点——只需要知道所有外力的**矢量和**。

$$\mathbf{F}_{\text{合}} = \sum_i \mathbf{F}_i^{\text{外}}$$

这个合力可以"平移"到质心上，用来计算质心的加速度。

> **批注**：但欧拉立即警告读者——力的简化只对平移运动有效。对于旋转运动，力的作用点至关重要。同一个力作用在不同的点上，对平移运动的效果相同，但对旋转运动的效果可能完全不同。这正是"力矩"概念的来源——它度量了力产生旋转效果的能力。

## 命题2：动量定理

刚体的总动量等于质心动量：
$$\mathbf{P} = \sum_i m_i\mathbf{v}_i = M\mathbf{V}_{\text{cm}}$$

因此，动量定理可以表述为：
$$\frac{d\mathbf{P}}{dt} = \mathbf{F}_{\text{外}}$$

> **批注**：这个定理将质点系的动量定理应用于刚体，是质心运动定理的等价表述。

## 命题3：动能与质心动能的关系（柯尼希定理的雏形）

欧拉指出，刚体中每个质点的速度可以分解为质心速度和相对于质心的速度：
$$\mathbf{v}_i = \mathbf{V}_{\text{cm}} + \mathbf{v}_i'$$

其中 $\mathbf{v}_i' = \mathbf{v}_i - \mathbf{V}_{\text{cm}}$ 是相对于质心的速度。

总动能为：
$$T = \sum_i \frac{1}{2}m_i v_i^2 = \sum_i \frac{1}{2}m_i(\mathbf{V}_{\text{cm}} + \mathbf{v}_i')^2$$

展开：
$$T = \frac{1}{2}MV_{\text{cm}}^2 + \mathbf{V}_{\text{cm}} \cdot \sum_i m_i\mathbf{v}_i' + \sum_i \frac{1}{2}m_i v_i'^2$$

由于 $\sum_i m_i\mathbf{v}_i' = 0$（在质心参考系中，总动量为零），中间项为零。因此：

$$\boxed{T = \frac{1}{2}MV_{\text{cm}}^2 + \sum_i \frac{1}{2}m_i v_i'^2}$$

**总动能 = 质心动能 + 相对于质心的动能**

> **批注**：这就是著名的柯尼希定理（König's theorem）。它表明刚体的动能可以自然地分解为两部分：质心平动的动能和绕质心运动的动能。这个分解是质心-转动分解定理在能量层面的体现。右边第二项（相对于质心的动能）在纯平移时为零——这对应于所有质点速度相同的情况。在后面引入转动惯量和角速度后，这一项将变为 $\frac{1}{2}I\omega^2$ 或更一般的 $\frac{1}{2}\boldsymbol{\omega} \cdot \mathbf{I} \cdot \boldsymbol{\omega}$。

## 注释（Scholium）：为什么刚体是六自由度系统

欧拉在本章的注释中讨论了一个重要的观点：

一个由 $N$ 个质点组成的系统，有 $3N$ 个自由度。但当这些质点之间的距离都保持不变（构成刚体）时，约束条件大大减少了自由度的数量。

- 在三维空间中，确定一个刚体的位置只需要：
  - 质心的三个坐标 $(X, Y, Z)$ — 3个自由度
  - 刚体的取向（方位）— 3个自由度

总共 **6个自由度**。

> **批注**：这个"6个自由度"的结论是刚体力学的出发点。它意味着描述刚体的运动只需要6个独立变量（如质心坐标和欧拉角），而不是无穷多个。这使得刚体力学成为一个有限的、可处理的问题。欧拉在后面的章节中将选择这6个变量的最佳组合，并推导出它们满足的6个微分方程（3个平动方程 + 3个转动方程）。

## 本章小结

正文第1章建立了刚体力学的两大基石：

1. **刚体的严格定义**：各部分间距离不变的物体
2. **质心运动定理**：$M\ddot{\mathbf{R}}_{\text{cm}} = \mathbf{F}_{\text{外}}$，质心运动等同于质点运动
3. **动能分解**：$T = \frac{1}{2}MV_{\text{cm}}^2 + T_{\text{相对}}$
4. **刚体有6个自由度**：3个平动 + 3个转动

> **批注**：第1章解决的是刚体6个自由度中的3个——平动自由度。质心运动定理告诉我们，刚体的平动完全由外力的矢量和决定，与刚体的形状和取向无关。剩下的问题是：刚体的转动（另外3个自由度）由什么决定？这就是后面18章要回答的问题。
