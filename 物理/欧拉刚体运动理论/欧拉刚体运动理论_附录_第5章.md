# 附录第5章：进一步的摩擦问题

---

## 本章概述

在附录的最后一章，欧拉讨论了各种与摩擦相关的补充问题，包括滚动摩擦、摩擦在机械中的应用等。

> **批注**：欧拉对摩擦问题的全面讨论，从定义到具体应用，展示了他对物理理论完整性的追求。他不愿留下"空白"——即使摩擦是一个使问题复杂化的因素，也必须被纳入理论框架。

---

## 1. 滚动摩擦

除了滑动摩擦外，欧拉讨论了**滚动摩擦**（rolling friction）。

当圆柱体或球体在表面上纯滚动时，虽然没有宏观的滑动，但由于接触面的微小变形，仍然存在阻力。

滚动摩擦力矩：
$$M_{\text{滚动}} = \mu_r N$$

其中 $\mu_r$ 是滚动摩擦系数（有长度量纲），通常远小于滑动摩擦系数对应的等效量。

> **批注**：滚动摩擦比滑动摩擦小得多——这就是为什么轮子的发明是人类文明史上的重大进步。滑动摩擦系数 $\mu_k \sim 0.1-0.5$，而等效的滚动摩擦系数 $\mu_r/R \sim 0.001-0.01$。滚动将摩擦力减小了1-2个数量级。

## 2. 摩擦在简单机械中的作用

欧拉讨论了摩擦对几种简单机械效率的影响：

### 斜面
效率 $\eta = \frac{\sin\alpha}{\sin\alpha + \mu_k\cos\alpha} = \frac{\tan\alpha}{\tan\alpha + \mu_k}$

### 螺旋（等效于卷起的斜面）
效率类似，但等效倾角更大，摩擦的影响更显著。

### 滑轮
轴承摩擦降低了滑轮的效率。

> **批注**：欧拉对简单机械效率的分析具有实际工程意义。在设计任何机械系统时，摩擦导致的能量损失都必须被考虑。欧拉在这里给出了定量分析的方法。

## 3. 摩擦力的双重性

欧拉以一个深刻的观察结束了附录的讨论：

**摩擦力既是运动的敌人，又是运动的朋友。**

- **敌人**：摩擦消耗能量，使运动减速，降低机械效率。
- **朋友**：没有摩擦力，我们无法行走（脚底打滑），车辆无法行驶（车轮空转），螺丝无法固定（松动），绳子无法打结（滑脱）。

> **批注**：这个"双重性"的观察体现了欧拉对物理世界的深刻理解。物理定律通常有两面性——同一条规律，在不同的情境下可以产生截然不同的效果。摩擦是最典型的例子：它既是我们需要克服的障碍，又是我们赖以生存的条件。

## 4. 全书结论

通过30章的系统论述（引言6章 + 正文19章 + 附录5章），欧拉完成了经典力学史上最伟大的理论构建之一。

### 引言部分（第1-6章）——基础

欧拉从质点力学出发，建立了全书所需的分析框架：

- **运动学**：位置→速度→加速度的三层结构，微积分符号的系统运用
- **动力学**：牛顿三定律的分析形式表述，力与加速度、动量与力的关系
- **直线运动**：恒力、位置相关力（谐振子）、速度相关力（阻力）的完整求解
- **曲线运动**：运动的分解、切向与法向加速度、角动量守恒（面积定律）
- **阻尼介质**：终端速度、抛体在有阻力介质中的轨迹

### 正文部分（第1-19章）——核心

这是全书的主体，构成了完整的刚体动力学理论体系。概念展开遵循严密的逻辑顺序：

**第一阶段——平移运动（第1章）**：
- 刚体的严格定义（定义1：距离不变的质点系）
- 质心运动定理（$\mathbf{F}_{\text{外}} = M\ddot{\mathbf{R}}_{\text{cm}}$）
- 刚体有6个自由度

**第二阶段——绕固定轴的旋转（第2-4章）**：
- 角速度、角动量的概念
- 转动惯量 $I = \sum m_i r_i^2$ 的定义
- 旋转运动方程 $I\dot{\omega} = M$
- 物理摆理论和等效单摆长度
- 平行轴定理 $I = I_{\text{cm}} + Md^2$

**第三阶段——一般旋转的几何基础（第5章）**：
- **欧拉旋转定理**：有限旋转等价于绕某轴的一次旋转
- 瞬时旋转轴和角速度矢量的概念
- 速度公式 $\mathbf{v} = \boldsymbol{\omega} \times \mathbf{r}$

**第四阶段——惯量张量与主轴（第6-7章）**：
- 转动惯量的方向依赖性：$I(\hat{\mathbf{n}}) = A\alpha^2 + B\beta^2 + C\gamma^2 - 2D\alpha\beta - 2E\alpha\gamma - 2F\beta\gamma$
- 惯量椭球
- **主轴定理**：三个互相垂直的轴使惯量积为零（实对称矩阵对角化）
- 主转动惯量 $I_1, I_2, I_3$

**第五阶段——运动方程的推导（第8-10章）**：
- 双坐标系（空间固定 + 固连于刚体）
- 时间导数变换公式：$\frac{d\mathbf{A}}{dt}\big|_{\text{空间}} = \frac{d\mathbf{A}}{dt}\big|_{\text{固连}} + \boldsymbol{\omega} \times \mathbf{A}$
- 角速度的矢量加法
- **欧拉旋转方程**（全书核心）：
$$I_1\dot{\omega}_1 - (I_2 - I_3)\omega_2\omega_3 = M_1$$
$$I_2\dot{\omega}_2 - (I_3 - I_1)\omega_3\omega_1 = M_2$$
$$I_3\dot{\omega}_3 - (I_1 - I_2)\omega_1\omega_2 = M_3$$
- 欧拉角参数化和完整的6方程体系

**第六阶段——应用与推广（第11-19章）**：
- 球对称体、旋转对称体、一般三轴体的分类讨论
- 自由旋转的稳定性分析（绕中间轴不稳定——网球拍效应）
- 自由刚体运动（6自由度系统）
- 重陀螺理论（进动、章动、休止陀螺）
- 陀螺运动的详尽分析
- 柔性轴上的刚体运动（陀螺耦合振动、临界转速）
- 球体和圆柱体的滚动

### 附录部分（第1-5章）——完善

欧拉将摩擦纳入理论力学，使理论更接近实际：

- 摩擦的基本定律（静/动摩擦，库仑摩擦）
- 摩擦对平移和旋转运动的减速效果
- 摩擦对陀螺运动的影响（自转减速→进动加速→稳定性丧失→倒下）
- 滚动摩擦和机械效率

### 核心贡献总结

| 贡献 | 现代名称 | 意义 |
|---|---|---|
| 刚体的严格定义 | — | 建立了刚体力学的研究对象 |
| 质心-转动分解 | 柯尼希定理 | 将6自由度问题分解为两个3自由度问题 |
| 转动惯量 | 二次型/惯量张量 | 刚体抵抗角加速度的度量 |
| 主轴定理 | 谱定理的特例 | 惯量张量的对角化 |
| 欧拉旋转方程 | — | 刚体旋转运动的基本方程 |
| 欧拉角 | — | 旋转的参数化描述 |
| 旋转稳定性 | 中间轴定理 | 理解自由旋转的关键 |

> **批注**：欧拉的《刚体运动理论》是物理学史上最重要的著作之一。它不仅解决了"刚体如何运动"这个物理问题，还在数学上开创了新的领域（惯量张量的理论、实对称矩阵的对角化、非线性微分方程的定性分析）。欧拉以一己之力，将刚体动力学从零发展为一门成熟的科学——这无疑是他众多伟大贡献中最辉煌的之一。

---

## 全书结语

欧拉在1765年完成了这部著作，当时他58岁。他在此前多年中视力逐渐衰退（约1738年一场重病后右眼几乎完全失明），而1766年左眼也将完全失明。也就是说，这部包含大量复杂推导和计算的著作，是欧拉在视力严重受损的情况下完成的。

> **批注**：更令人惊叹的是，欧拉在完全失明后仍然继续工作，通过口述让助手记录，又产出了大量的数学和物理论文。他一生发表了超过800篇论文和著作，全集至今仍未完全出版完毕（已超过80卷）。欧拉的创造力堪称人类智慧的巅峰。

这部著作中的每一个方程、每一个定理，都仍在今天的物理学、工程学和数学中发挥着作用。从卫星的姿态控制到分子光谱的计算，从陀螺仪导航到机器人运动学——250多年过去了，欧拉的方程依然在描述着我们这个世界。
