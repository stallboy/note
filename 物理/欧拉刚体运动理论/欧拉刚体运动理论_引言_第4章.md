# 引言第4章：直线运动的进一步讨论

*De motu progressivo corporum*

---

## 本章概述

这一章是第3章的延续，欧拉深入讨论了直线运动中的更复杂情形。他特别关注了力的各种依赖形式，以及相应的求解技巧。

> **批注**：欧拉在第3章和第4章中建立的工具箱——能量积分、变量分离、首次积分——在后面的刚体问题中都有对应的推广。特别是，刚体的旋转动能和角动量守恒，就是这里"能量积分"和"首次积分"思想的直接延伸。

---

## 1. 力作为时间函数的情形

欧拉讨论了力是时间的已知函数的情况：$F = F(t)$。

运动方程：
$$m\frac{d^2x}{dt^2} = F(t)$$

可以直接对时间积分两次：
$$v(t) = v_0 + \frac{1}{m}\int_0^t F(t')\,dt'$$
$$x(t) = x_0 + v_0 t + \frac{1}{m}\int_0^t\int_0^{t'} F(t'')\,dt''\,dt'$$

或者用分部积分简化为单重积分：
$$x(t) = x_0 + v_0 t + \frac{1}{m}\int_0^t (t-t')F(t')\,dt'$$

> **批注**：最后这个表达式实际上就是"杜哈梅积分"（Duhamel's integral）的一个特例——它将一般力的效果表示为脉冲响应的叠加。这个思想在后来的振动理论、信号处理和控制论中极为重要。

### 脉冲力

欧拉特别讨论了脉冲力的情形：力在极短的时间内作用，但产生有限的动量变化。如果力 $F(t)$ 在时间 $\Delta t$ 内作用，产生的动量变化为：
$$\Delta p = \int_0^{\Delta t} F(t)\,dt$$

当 $\Delta t \to 0$ 时，力趋于无穷大，但积分 $\Delta p$ 保持有限。这就是**冲量**的概念。

> **批注**：冲量的概念在碰撞理论中至关重要。欧拉将在引言第6章和正文第18章中深入讨论碰撞问题。

## 2. 多个力的合成

当多个力同时作用时，由力的叠加原理：
$$m\frac{d^2x}{dt^2} = \sum_i F_i$$

欧拉讨论了几种典型的合力情况：

### 常力与恢复力
$$m\frac{d^2x}{dt^2} = F_0 - kx$$

这是一个偏移的谐振子，解为：
$$x = \frac{F_0}{k} + A\cos(\omega t + \phi)$$

平衡位置从 $x = 0$ 移到了 $x = F_0/k$。

### 恢复力与阻力
$$m\frac{d^2x}{dt^2} = -kx - b\frac{dx}{dt}$$

这是**阻尼谐振子**，是物理学中最重要的模型之一。

**求解过程**：引入参量 $\gamma = b/(2m)$ 和 $\omega_0 = \sqrt{k/m}$，方程重写为：
$$\ddot{x} + 2\gamma\dot{x} + \omega_0^2 x = 0$$

尝试解 $x = e^{\lambda t}$，代入得特征方程：
$$\lambda^2 + 2\gamma\lambda + \omega_0^2 = 0$$

判别式 $\Delta = 4\gamma^2 - 4\omega_0^2 = 4(\gamma^2 - \omega_0^2)$。

根据判别式的符号，分三种情况讨论：

**欠阻尼**（$\gamma < \omega_0$，即 $b^2 < 4mk$）：

两个共轭复根 $\lambda = -\gamma \pm i\omega_d$，其中 $\omega_d = \sqrt{\omega_0^2 - \gamma^2}$。

通解：
$$x(t) = Ae^{-\gamma t}\cos(\omega_d t) + Be^{-\gamma t}\sin(\omega_d t) = Ce^{-\gamma t}\cos(\omega_d t + \phi)$$

其中 $C = \sqrt{A^2 + B^2}$，$\phi = -\arctan(B/A)$。解是一个振幅指数衰减的振荡。衰减的特征时间为 $\tau = 1/\gamma = 2m/b$，阻尼振荡的频率 $\omega_d$ 低于无阻尼频率 $\omega_0$。

**临界阻尼**（$\gamma = \omega_0$，即 $b^2 = 4mk$）：

重根 $\lambda = -\gamma$。通解：
$$x(t) = (A + Bt)e^{-\gamma t}$$

临界阻尼是回到平衡位置最快的非振荡运动。当 $t \to \infty$ 时，$x \to 0$。

**过阻尼**（$\gamma > \omega_0$，即 $b^2 > 4mk$）：

两个不同的实根 $\lambda_{1,2} = -\gamma \pm \sqrt{\gamma^2 - \omega_0^2}$（均为负）。通解：
$$x(t) = Ae^{-\lambda_1 t} + Be^{-\lambda_2 t}$$

运动是两个指数衰减的叠加，没有振荡，但回到平衡位置比临界阻尼更慢。

> **批注**：阻尼谐振子是工程中最常见的模型之一——汽车的减震器、建筑物的抗震设计、电子电路的RLC振荡器，都是阻尼谐振子的实例。欧拉对三种阻尼情形的分类，至今仍是教科书中的标准内容。特别值得注意的是临界阻尼的特殊地位——它是振荡与非振荡的边界，也是回到平衡位置最快的方式。这就是为什么大多数机械减震器都被设计为接近临界阻尼状态。

## 3. 运动的几何表示

欧拉讨论了如何用几何方法来表示运动：

### 速度-时间图
在 $v$-$t$ 图上，速度曲线下的面积等于位移：
$$x = \int v\,dt$$

### 力-位移图
在 $F$-$x$ 图上，力曲线下的面积等于做的功：
$$W = \int F\,dx$$

欧拉指出，功与动能的关系为：
$$W = \Delta E_k = \frac{1}{2}mv_2^2 - \frac{1}{2}mv_1^2$$

> **批注**：这就是动能定理。欧拉通过积分牛顿第二定律直接得到了这个结果：
$$\int F\,dx = \int m\frac{dv}{dt}\,dx = \int mv\,dv = \frac{1}{2}mv^2 + C$$
这个定理将在后面的刚体动力学中发挥关键作用——刚体的动能定理是质点动能定理的直接推广。

## 4. 逆问题：从运动规律求力

欧拉讨论了力学的逆问题：已知质点的运动轨迹 $x = x(t)$，求作用在质点上的力。

由牛顿第二定律，力可以直接计算：
$$F = m\frac{d^2x}{dt^2}$$

**例子**：已知质点做简谐运动 $x = A\cos(\omega t)$，求力。
$$\frac{d^2x}{dt^2} = -A\omega^2\cos(\omega t) = -\omega^2 x$$
$$F = -m\omega^2 x$$

力与位移成正比，方向指向平衡点——这就是弹性恢复力。

> **批注**：逆问题的思想在力学中非常重要。开普勒从天文观测中总结出行星运动的规律（开普勒三定律），牛顿则从这些运动规律中"逆推"出了万有引力定律。欧拉在这里展示的方法，正是这种"从运动推力"的一般化。

第4章是对第3章的扩展，涵盖了：
