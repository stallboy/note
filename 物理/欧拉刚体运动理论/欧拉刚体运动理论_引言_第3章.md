# 引言第3章：自由质点的直线运动

*De motu rectilineo puncti liberi*

---

## 本章概述

这一章中，欧拉研究了最简单的力学问题：一个不受任何约束的自由质点在直线上的运动。虽然问题简单，但欧拉通过这个例子展示了如何用微分方程来求解力学问题——这种方法论上的示范为后续更复杂的问题铺平了道路。

> **批注**：欧拉并不是在教读者如何解决简单的运动学问题。他的真正目的是建立一套"方法论模板"：将物理问题转化为微分方程，然后求解。这个模板在后面的刚体问题中将被反复使用。

---

## 1. 无力作用的自由运动

首先考虑最简单的情形：一个质量为 $m$ 的自由质点，不受任何外力作用。

由牛顿第二定律，$F = 0$，所以：
$$m\frac{d^2x}{dt^2} = 0$$

解这个微分方程：
$$\frac{dx}{dt} = v_0 \quad \text{（常数）}$$
$$x = x_0 + v_0 t$$

**结论**：自由质点以恒定速度做匀速直线运动。这正是惯性原理的直接体现。

> **批注**：从微分方程的角度看，这是一个二阶常微分方程，需要两个初始条件（初始位置 $x_0$ 和初始速度 $v_0$）来确定唯一解。这个"初始条件决定运动"的思想是决定论力学的核心——拉普拉斯后来将其推广到整个宇宙。

## 2. 恒力作用下的直线运动

现在考虑一个受恒力 $F$ 作用的质点。运动方程为：
$$m\frac{d^2x}{dt^2} = F$$

求解：
$$\frac{dx}{dt} = v_0 + \frac{F}{m}t$$
$$x = x_0 + v_0 t + \frac{1}{2}\frac{F}{m}t^2$$

**结论**：在恒力作用下，质点做匀变速运动。位移是时间的二次函数。

欧拉特别讨论了重力作用下的自由落体：
$$\frac{d^2x}{dt^2} = g$$
其中 $g$ 是重力加速度。解为：
$$x = x_0 + v_0 t + \frac{1}{2}gt^2$$

> **批注**：自由落体公式 $s = \frac{1}{2}gt^2$ 最初是伽利略通过实验发现的，但欧拉通过微分方程将它变成了一个可以严格推导的结论。从实验规律到数学定理，这是物理学进步的基本模式。

## 3. 与位置相关的力

欧拉讨论了力是位置函数的情况：$F = F(x)$。运动方程为：
$$m\frac{d^2x}{dt^2} = F(x)$$

欧拉展示了如何用"首次积分"的方法求解这类方程。利用变换 $\frac{d^2x}{dt^2} = v\frac{dv}{dx}$，方程变为：
$$mv\frac{dv}{dx} = F(x)$$
$$\frac{1}{2}mv^2 = \int F(x)\,dx + C$$

这就是**能量积分**：
$$\frac{1}{2}mv^2 - \int F(x)\,dx = E = \text{常数}$$

> **批注**：这个积分就是今天所说的"机械能守恒"——动能加势能等于常数。虽然欧拉在这里没有使用"能量"这个词（这个术语是19世纪才普及的），但他清楚地知道这个积分的物理意义。首次积分方法是分析力学的基本技巧之一——它将二阶微分方程降为一阶微分方程，大大简化了求解过程。

### 谐振子的例子

欧拉讨论了一个重要的特例：恢复力与位移成正比，$F = -kx$。运动方程为：
$$m\frac{d^2x}{dt^2} = -kx$$

解为简谐运动：
$$x = A\cos(\omega t + \phi)$$
其中 $\omega = \sqrt{k/m}$ 是角频率。

> **批注**：谐振子是物理学中最重要的模型之一。从弹簧振子到分子振动，从电路振荡到量子谐振子，简谐运动无处不在。欧拉对它的严格分析为后来的波动理论、声学和电磁学奠定了基础。

## 4. 与速度相关的力

欧拉讨论了力是速度函数的情况：$F = F(v)$。这在处理阻力问题时非常重要。

运动方程为：
$$m\frac{dv}{dt} = F(v)$$

分离变量：
$$dt = \frac{m\,dv}{F(v)}$$
$$t = m\int\frac{dv}{F(v)} + C$$

### 线性阻力

对于线性阻力 $F = -bv$（$b > 0$）：
$$m\frac{dv}{dt} = -bv$$

解为指数衰减：
$$v = v_0 e^{-bt/m}$$

位置：
$$x = x_0 + \frac{mv_0}{b}(1 - e^{-bt/m})$$

**结论**：在阻力与速度成正比的介质中，质点的速度指数衰减，最终趋于静止。总位移为有限值 $x_0 + mv_0/b$。

> **批注**：指数衰减 $e^{-bt/m}$ 是一个极其重要的数学函数。它描述了从放射性衰变到RC电路放电等各种物理过程。欧拉对它的处理展示了微分方程与指数函数之间的深刻联系。

### 二次阻力

对于二次阻力 $F = -cv^2$（$c > 0$，运动方向与力方向相反）：
$$m\frac{dv}{dt} = -cv^2$$

解为：
$$\frac{1}{v} = \frac{1}{v_0} + \frac{c}{m}t$$
$$v = \frac{v_0}{1 + \frac{cv_0}{m}t}$$

> **批注**：二次阻力模型更接近实际情况——空气阻力通常与速度的平方成正比。欧拉对这两种阻力模型的分析，是流体阻力理论的基础。

## 5. 变力作用下的运动

欧拉讨论了更一般的情况：力同时依赖于位置和速度，$F = F(x, v)$。

运动方程为：
$$m\frac{d^2x}{dt^2} = F(x, v)$$

欧拉指出，这类方程通常没有解析解，需要用数值方法（即逐步逼近法）来求解。他描述了一种早期的数值积分方法——这本质上是后来"欧拉方法"（Euler method）的雏形。

> **批注**：欧拉方法是最基本的数值常微分方程求解方法。给定 $y' = f(t, y)$ 和初始条件 $y(t_0) = y_0$，欧拉方法通过逐步推进来近似求解：$y_{n+1} = y_n + f(t_n, y_n)\Delta t$。虽然精度不高，但它简单直观，是理解数值ODE求解的起点。在计算机时代，欧拉方法的各种改进（龙格-库塔法等）已经成为科学计算的基础工具。

---

## 本章小结

在第3章中，欧拉通过自由质点的直线运动展示了分析力学的基本方法：

1. **将物理问题转化为微分方程**
2. **求解微分方程**（直接积分、首次积分、变量分离）
3. **解释解的物理意义**

处理的具体问题包括：
- 无力作用：匀速直线运动
- 恒力作用：匀变速运动（自由落体）
- 位置相关力：谐振子，能量积分
- 速度相关力：线性/二次阻力，指数衰减
- 一般力：数值方法

> **批注**：这一章建立的方法论模板——"物理问题 → 微分方程 → 求解 → 物理解释"——在后续的刚体力学中被反复使用。刚体的旋转运动最终也归结为求解微分方程（欧拉旋转方程），只不过方程更加复杂（非线性、耦合的三个方程）。欧拉在这里展示的方法，是他后来解决刚体问题的"预演"。
