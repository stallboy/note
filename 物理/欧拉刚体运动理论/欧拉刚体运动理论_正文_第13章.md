# 正文第13章：自由刚体运动

*De motu corporis rigidi, in quo nullum punctum fixum manet*

---

## 本章概述

这一章中，欧拉处理了最一般的无约束刚体运动——没有固定点、没有固定轴，刚体完全自由地在空间中运动。自由刚体有6个自由度（3个平动 + 3个转动），是经典力学中最一般的刚体问题。

> **批注**：自由刚体运动是欧拉刚体动力学理论的"最终形态"。质心运动定理处理3个平动自由度，欧拉旋转方程处理3个转动自由度，两者组合就构成了自由刚体的完整运动方程组。这个6自由度系统在航天动力学、弹道学、机器人学中有广泛的应用。

---

## 1. 质心平动与绕质心转动的分离

**核心定理**：自由刚体的运动可以完全分解为：
- **质心的平动**：由外力的矢量和决定
- **绕质心的转动**：由外力矩决定

这个分解在数学上是严格的，不需要任何近似。

### 平动方程（来自第1章的质心运动定理）

$$M\frac{d^2\mathbf{R}_{\text{cm}}}{dt^2} = \mathbf{F}_{\text{外}} = \sum_i \mathbf{F}_i^{\text{外}}$$

分量形式：
$$M\ddot{X} = F_X, \quad M\ddot{Y} = F_Y, \quad M\ddot{Z} = F_Z$$

这里 $(X, Y, Z)$ 是质心在空间固定坐标系中的坐标。

### 转动方程（来自第10章的欧拉旋转方程）

以质心为参考点，在主轴坐标系（固连于刚体，原点在质心）中：

$$I_1\dot{\omega}_1 - (I_2 - I_3)\omega_2\omega_3 = M_1$$
$$I_2\dot{\omega}_2 - (I_3 - I_1)\omega_3\omega_1 = M_2$$
$$I_3\dot{\omega}_3 - (I_1 - I_2)\omega_1\omega_2 = M_3$$

加上运动学方程（欧拉角与角速度的关系），共6个方程。

### 总动能

$$T = \underbrace{\frac{1}{2}MV_{\text{cm}}^2}_{\text{质心平动}} + \underbrace{\frac{1}{2}(I_1\omega_1^2 + I_2\omega_2^2 + I_3\omega_3^2)}_{\text{绕质心转动}}$$

> **批注**：质心-转动分离使得6个自由度的问题被分解为两个3自由度的问题。但注意，两个子系统并非完全独立——力矩 $M_i$ 通常依赖于刚体的取向（刚体如何朝向），而取向由转动方程决定。因此，两个子系统通过力矩的取向依赖性耦合在一起。只有在特殊情况下（如均匀重力场中自由飞行的刚体），耦合才会消失。

## 2. 重力场中的自由飞行

考虑一个在均匀重力场中自由飞行的刚体（如一个抛在空中的旋转物体）。

### 质心运动

$$M\ddot{X} = 0, \quad M\ddot{Y} = 0, \quad M\ddot{Z} = -Mg$$

质心做抛物线运动：
$$X(t) = X_0 + V_{X0}t$$
$$Y(t) = Y_0 + V_{Y0}t$$
$$Z(t) = Z_0 + V_{Z0}t - \frac{1}{2}gt^2$$

### 旋转运动

重力作用于每个质点，但总的重力等价于一个作用于质心的力 $Mg$。由于力通过质心，力臂为零，**重力对质心的力矩为零**：

$$\mathbf{M}_{\text{重力}} = \mathbf{r}_{\text{cm}} \times (M\mathbf{g}) = \mathbf{0} \times M\mathbf{g} = 0$$

因此转动方程退化为自由旋转：
$$I_1\dot{\omega}_1 = (I_2 - I_3)\omega_2\omega_3$$
$$I_2\dot{\omega}_2 = (I_3 - I_1)\omega_3\omega_1$$
$$I_3\dot{\omega}_3 = (I_1 - I_2)\omega_1\omega_2$$

**结论**：在均匀重力场中自由飞行的刚体，其旋转运动与质心的平动完全独立。旋转是"自由"的——角动量和动能守恒。质心做抛物线运动，与刚体是否旋转、如何旋转无关。

> **批注**：这个结论解释了为什么一个旋转的球在飞行中旋转状态保持不变。无论球的质心轨迹如何（直线上抛、斜抛、自由下落），旋转运动都按照自由旋转的规律演化。这在体育运动中很常见——网球的上旋、足球的弧线球、乒乓球的侧旋，在飞行过程中旋转状态基本不变（忽略空气阻力的话）。

### 完整解

给定初始条件：
- 质心位置 $\mathbf{R}_0$ 和速度 $\mathbf{V}_0$
- 角速度 $\boldsymbol{\omega}_0$ 和初始取向（欧拉角 $\phi_0, \theta_0, \psi_0$）

质心运动由抛物线公式给出；旋转运动由自由旋转的欧拉方程给出（如第12章讨论的，可以用椭圆函数表示）。两者互不干扰。

## 3. 非均匀力场中的运动

当外力产生的力矩不为零时，平动和转动耦合在一起。

### 空气阻力

空气对运动物体的作用力通常不通过质心，因此产生力矩：

- **阻力**：$\mathbf{F}_{\text{阻}} = -\frac{1}{2}\rho v^2 C_D A\hat{\mathbf{v}}$，作用于压力中心（通常不在质心上）
- **升力**：如果物体不对称，还可能产生垂直于运动方向的升力
- **力矩**：$\mathbf{M} = \mathbf{r}_{\text{cp}} \times \mathbf{F}_{\text{空气}}$

力矩依赖于刚体相对于气流的方向，因此旋转和平动耦合。

> **批注**：空气动力学力矩是炮弹和火箭飞行稳定性的关键因素。炮弹在飞行中受到的空气动力矩取决于炮弹的攻角（飞行方向与对称轴之间的夹角）。如果力矩是恢复性的（使攻角减小），炮弹飞行稳定；如果力矩是发散性的（使攻角增大），炮弹将翻滚。这就是为什么炮弹和火箭需要有尾翼或旋转稳定。

### 重力梯度力矩

在非均匀重力场中（如卫星在地球轨道上），由于刚体的不同部分到地心的距离不同，受到的重力略有差异，产生一个微小的力矩：

$$\mathbf{M}_{\text{重力梯度}} \approx \frac{3GM}{R^3}\hat{\mathbf{R}} \times (\mathbf{I} \cdot \hat{\mathbf{R}})$$

其中 $R$ 是刚体质心到地心的距离，$\hat{\mathbf{R}}$ 是方向单位矢量，$\mathbf{I}$ 是惯量张量。

分量形式（在主轴坐标系中）：
$$M_1 = \frac{3\mu}{R^3}(I_3 - I_2)\gamma_2\gamma_3$$
$$M_2 = \frac{3\mu}{R^3}(I_1 - I_3)\gamma_3\gamma_1$$
$$M_3 = \frac{3\mu}{R^3}(I_2 - I_1)\gamma_1\gamma_2$$

其中 $\mu = GM_{\text{地球}}$，$(\gamma_1, \gamma_2, \gamma_3)$ 是径向方向在固连坐标系中的方向余弦。

> **批注**：重力梯度力矩虽然极其微小（对于低轨道卫星约 $10^{-5}$ Nm 量级），但在长时间作用下会产生显著的效果。它可以被利用（重力梯度稳定——使卫星的长轴指向地球），也可能需要被克服（精密指向的望远镜需要抵消重力梯度力矩）。这个力矩在欧拉的时代还不为人知（卫星尚不存在），但欧拉建立的运动方程完全适用于分析它。

### 磁力矩

如果刚体有磁矩 $\boldsymbol{\mu}$，在外磁场 $\mathbf{B}$ 中受力矩：
$$\mathbf{M}_{\text{磁}} = \boldsymbol{\mu} \times \mathbf{B}$$

力矩依赖于磁矩方向相对于磁场的方向——即依赖于刚体的取向。

## 4. 自由刚体的一般运动方程组

综合以上讨论，自由刚体的一般运动方程组为：

**平动方程**（空间固定坐标系中）：
$$M\ddot{X} = F_X(\mathbf{R}, \dot{\mathbf{R}}, \phi, \theta, \psi)$$
$$M\ddot{Y} = F_Y(\mathbf{R}, \dot{\mathbf{R}}, \phi, \theta, \psi)$$
$$M\ddot{Z} = F_Z(\mathbf{R}, \dot{\mathbf{R}}, \phi, \theta, \psi)$$

**转动方程**（固连主轴坐标系中）：
$$I_1\dot{\omega}_1 = (I_2 - I_3)\omega_2\omega_3 + M_1(\phi, \theta, \psi, \omega_i, \mathbf{R}, \dot{\mathbf{R}})$$
$$I_2\dot{\omega}_2 = (I_3 - I_1)\omega_3\omega_1 + M_2(\phi, \theta, \psi, \omega_i, \mathbf{R}, \dot{\mathbf{R}})$$
$$I_3\dot{\omega}_3 = (I_1 - I_2)\omega_1\omega_2 + M_3(\phi, \theta, \psi, \omega_i, \mathbf{R}, \dot{\mathbf{R}})$$

**运动学方程**：
$$\dot{\phi} = \frac{\omega_1\sin\psi + \omega_2\cos\psi}{\sin\theta}$$
$$\dot{\theta} = \omega_1\cos\psi - \omega_2\sin\psi$$
$$\dot{\psi} = \omega_3 - \frac{(\omega_1\sin\psi + \omega_2\cos\psi)\cos\theta}{\sin\theta}$$

总共9个一阶微分方程（或等价地，6个二阶微分方程），给定初始条件后唯一确定了刚体的未来运动。

> **批注**：这组方程是经典力学的"终极成果"之一。从牛顿的三个运动定律出发，通过欧拉的系统化工作，我们得到了描述任意刚体在任意力作用下运动的完整方程组。每一个航天器、每一颗卫星、每一个抛射体的运动，原则上都由这组方程描述。当然，在实际计算中，由于方程的非线性和力矩的复杂性，通常需要数值方法来求解。

## 5. 守恒量与可积性

### 无外力时的守恒量

当刚体不受任何外力时，有7个守恒量：

1. **总动量** $\mathbf{p} = M\mathbf{V}_{\text{cm}}$（3个分量）
2. **总角动量** $\mathbf{L}$（3个分量）
3. **总动能** $T = \frac{1}{2}MV_{\text{cm}}^2 + \frac{1}{2}\boldsymbol{\omega}\cdot\mathbf{I}\cdot\boldsymbol{\omega}$

3个动量守恒量直接给出质心做匀速直线运动。3个角动量守恒量和1个动能守恒量用于分析旋转运动。

### 重力场中的守恒量

在均匀重力场中：
1. **水平动量** $p_X$ 和 $p_Y$（2个）
2. **总能量** $E = T + V_{\text{重力}}$（1个）
3. **竖直角动量** $L_Z$（1个）

> **批注**：自由刚体在重力场中有4个守恒量（加上质心的可积性，共6个"积分"），而系统有6个自由度。在刘维尔意义上，系统是可积的——至少在原则上，解可以用积分来表示。但对于一般的三轴不等刚体，实际的积分计算非常复杂。

---

## 本章小结

正文第13章建立了自由刚体的完整运动理论：

1. **平动-转动分离定理**：质心平动（3个方程）+ 绕质心转动（3个欧拉方程）
2. **重力场中的独立性**：均匀重力场中旋转与平动完全独立
3. **一般耦合**：力矩依赖于取向时平动与转动耦合
4. **空气阻力、重力梯度、磁力矩**：具体的耦合机制
5. **9方程体系**：完整的自由刚体运动方程组
6. **守恒量与可积性**：自由情况下有7个守恒量

> **批注**：自由刚体力学是航天动力学的基础。每一颗人造卫星、每一枚火箭、每一个星际探测器，都是自由刚体（近似），它们的运动都由欧拉建立的这些方程描述。从牛顿到欧拉，从苹果到卫星——经典力学的伟大旅程，在这一章达到了它的终点。
