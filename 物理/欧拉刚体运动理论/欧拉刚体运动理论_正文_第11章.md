# 正文第11章：受任意力作用的刚体运动

*De motu corporis rigidi, a viribus quibuscunque sollicitati*

---

## 本章概述

欧拉在这一章中将欧拉旋转方程应用于具体问题。他系统地讨论了不同主转动惯量关系的刚体在不同力矩作用下的运动，从最简单的球对称体逐步过渡到一般的三轴不等刚体。

> **批注**：从本章开始，欧拉展示了欧拉旋转方程的强大威力。通过分析不同类型的刚体和不同的受力情况，他揭示了旋转运动丰富多彩的现象。本章的方法论是"分类讨论"——按照主转动惯量的关系将刚体分为几类，分别分析每类的运动特征。

---

## 1. 球对称体（$I_1 = I_2 = I_3 = I$）

当三个主转动惯量相等时，欧拉旋转方程大为简化：

$$I\dot{\omega}_1 = M_1, \quad I\dot{\omega}_2 = M_2, \quad I\dot{\omega}_3 = M_3$$

交叉项 $(I_2 - I_3)\omega_2\omega_3$ 等全部消失，因为 $I_2 - I_3 = 0$ 等。方程退化为三个**独立的线性**方程。

### 物理解释

对于球对称体，角动量 $\mathbf{L} = I\boldsymbol{\omega}$ 总是与角速度平行。因此：
$$\boldsymbol{\omega} \times \mathbf{L} = I\boldsymbol{\omega} \times \boldsymbol{\omega} = 0$$

旋转坐标系的效应完全消失。这意味着固连坐标系与空间坐标系中的动力学方程完全相同——坐标系的选择不影响方程的形式。

### 一般力矩下的解

$$\boldsymbol{\omega}(t) = \boldsymbol{\omega}_0 + \frac{1}{I}\int_0^t \mathbf{M}(t')\,dt'$$

**结论**：球对称体的旋转运动与质点的直线运动完全类似——角加速度正比于力矩，没有"耦合"效应。只要知道力矩的时间依赖关系，就可以直接积分求解。

### 具体例子：恒定力矩

设 $\mathbf{M} = (M_0, 0, 0)$（沿第一个主轴的恒定力矩）：

$$\omega_1(t) = \omega_{1,0} + \frac{M_0}{I}t, \quad \omega_2 = \omega_{2,0}, \quad \omega_3 = \omega_{3,0}$$

角速度沿力矩方向线性增长，其他分量不变——完全类似于质点在恒力下的匀加速运动。

> **批注**：球体的旋转动力学之所以如此简单，根本原因是惯量椭球退化为一个球面——所有方向的惯性特性完全相同。这是"对称性简化问题"的一个典型例子。在后面的第18章中，欧拉将专门讨论球体运动。

## 2. 两个主转动惯量相等的情况（$I_1 = I_2 \neq I_3$）

这对应于**旋转对称体**（如圆柱体、圆锥体、陀螺等）。主轴坐标系中，$z$ 轴取为对称轴。

欧拉旋转方程变为：
$$I_1\dot{\omega}_1 = (I_1 - I_3)\omega_2\omega_3 + M_1$$
$$I_1\dot{\omega}_2 = (I_3 - I_1)\omega_1\omega_3 + M_2$$
$$I_3\dot{\omega}_3 = M_3$$

### 第三个方程的分析

第三个方程特别简单：绕对称轴的角加速度只取决于绕该轴的力矩分量 $M_3$：
$$\dot{\omega}_3 = \frac{M_3}{I_3}$$

这是因为绕对称轴旋转时，惯量椭球是圆形的（$I_1 = I_2$），交叉项恰好为零。

### 前两个方程的耦合

前两个方程通过 $\omega_3$ 耦合。引入复变量 $\omega_+ = \omega_1 + i\omega_2$ 和 $M_+ = M_1 + iM_2$，可以将两个方程合并为一个：

$$I_1\dot{\omega}_+ = (I_3 - I_1)\omega_3\omega_+ \cdot (-i) + M_+$$

当 $\omega_3$ 已知时（例如从第三个方程解出），这是一个关于 $\omega_+$ 的一阶线性微分方程，可以用积分因子法求解。

### 无外力矩时的解（$M_1 = M_2 = M_3 = 0$）

由第三个方程：$\omega_3 = \Omega = \text{常数}$。

代入前两个方程，用复变量表示：
$$I_1\dot{\omega}_+ = i(I_3 - I_1)\Omega\omega_+$$

解为：
$$\omega_+(t) = \omega_+(0) e^{i\lambda t}$$

其中 $\lambda = \frac{(I_3 - I_1)}{I_1}\Omega$。

分量形式：
$$\omega_1(t) = A\cos(\lambda t + \alpha), \quad \omega_2(t) = A\sin(\lambda t + \alpha)$$

**物理意义**：在固连坐标系中看，角速度的横向分量 $(\omega_1, \omega_2)$ 以角频率 $\lambda$ 绕对称轴旋转。角速度矢量的端点在刚体内部画出一个圆锥面——这就是**本体极迹**（polhode），在这种情况下是一个圆。

> **批注**：旋转对称体的动力学比球体丰富得多，但仍然可以解析求解。$\lambda = (I_3 - I_1)\Omega/I_1$ 这个量称为**本体进动频率**——它描述了角速度在固连坐标系中绕对称轴旋转的快慢。对于扁体（$I_3 > I_1$，如地球），$\lambda > 0$；对于长体（$I_3 < I_1$，如长杆），$\lambda < 0$——旋转方向相反。

## 3. 一般三轴不等刚体（$I_1 \neq I_2 \neq I_3$）

这是最一般的情况，欧拉旋转方程的三个分量全部耦合：

$$I_1\dot{\omega}_1 = (I_2 - I_3)\omega_2\omega_3 + M_1$$
$$I_2\dot{\omega}_2 = (I_3 - I_1)\omega_3\omega_1 + M_2$$
$$I_3\dot{\omega}_3 = (I_1 - I_2)\omega_1\omega_2 + M_3$$

### 自由旋转（$M = 0$）的守恒量

即使没有外力矩，方程仍然是耦合的非线性微分方程。但有两个守恒量帮助分析：

**角动量守恒**（空间坐标系中）：
$$L^2 = I_1^2\omega_1^2 + I_2^2\omega_2^2 + I_3^2\omega_3^2 = \text{常数}$$

**动能守恒**：
$$2T = I_1\omega_1^2 + I_2\omega_2^2 + I_3\omega_3^2 = \text{常数}$$

这两个约束将 $(\omega_1, \omega_2, \omega_3)$ 的运动限制在一个空间曲线上——能量椭球和角动量球的交线。

### 数值求解方法

欧拉指出，对于一般的力矩，通常无法得到解析解。他描述了一种逐步逼近的数值方法：

1. 在时刻 $t$，已知 $(\omega_1, \omega_2, \omega_3)$
2. 计算力矩分量 $M_1, M_2, M_3$（可能依赖于取向）
3. 用欧拉方程计算 $\dot{\omega}_1, \dot{\omega}_2, \dot{\omega}_3$
4. 在时间步长 $\Delta t$ 内更新：$\omega_i(t + \Delta t) = \omega_i(t) + \dot{\omega}_i \Delta t$
5. 同时更新欧拉角（通过运动学方程）
6. 重复步骤2-5

> **批注**：欧拉在这里描述的正是"欧拉方法"——最简单的数值常微分方程求解方法。虽然精度为 $O(\Delta t)$，但它简单直观，是理解数值积分的基础。在现代计算中，通常使用更高阶的方法（如龙格-库塔法），但基本思想是相同的。

## 4. 一般力矩的处理

欧拉详细讨论了如何将一般的外力系简化为力矩分量。

### 力矩的计算

给定一组外力 $\mathbf{F}_1, \mathbf{F}_2, \ldots$，作用点分别为 $\mathbf{r}_1, \mathbf{r}_2, \ldots$（相对于固定点或质心），总力矩为：
$$\mathbf{M} = \sum_i \mathbf{r}_i \times \mathbf{F}_i$$

在主轴坐标系中的分量：
$$M_1 = \sum_i (y_i F_{i,z} - z_i F_{i,y})$$
$$M_2 = \sum_i (z_i F_{i,x} - x_i F_{i,z})$$
$$M_3 = \sum_i (x_i F_{i,y} - y_i F_{i,x})$$

### 力矩依赖于取向的情况

关键复杂化因素是：力矩通常依赖于刚体的取向。例如：

- **重力力矩**：取决于质心位置相对于竖直方向的位置，而竖直方向在固连坐标系中的表示随取向变化
- **空气动力矩**：取决于刚体相对于气流的方向
- **磁力矩**：取决于磁体的磁矩方向相对于外磁场的方向

这意味着力矩 $M_i$ 是欧拉角 $(\phi, \theta, \psi)$ 的函数，而欧拉角又通过运动学方程与 $(\omega_1, \omega_2, \omega_3)$ 耦合。最终，我们需要同时求解6个耦合的微分方程。

> **批注**：力矩的计算依赖于力的作用点在固连坐标系中的位置。由于固连坐标系随刚体旋转，作用点位置在固连坐标系中是常数——这是选择固连坐标系的又一个理由。如果在空间坐标系中计算，惯量张量随时间变化（难以处理），但力矩可能更简单。两种选择各有利弊，欧拉选择了在固连坐标系中工作。

## 5. 重力力矩的详细计算

欧拉详细讨论了重力产生的力矩——这是最常见的力矩类型。

### 设定

固定点为 $O$，质心在固连坐标系中的位置为 $(x_c, y_c, z_c)$，重力 $\mathbf{W} = -Mg\hat{\mathbf{Z}}$（沿空间竖直方向向下）。

### 方向余弦的引入

空间竖直方向 $\hat{\mathbf{Z}}$ 在固连坐标系中的分量（方向余弦）为 $(\gamma_1, \gamma_2, \gamma_3)$。这些方向余弦可以用欧拉角表示：

$$\gamma_1 = \sin\theta\sin\psi$$
$$\gamma_2 = \sin\theta\cos\psi$$
$$\gamma_3 = \cos\theta$$

### 力矩公式

重力力矩为：
$$\mathbf{M} = \mathbf{r}_c \times (-Mg\hat{\mathbf{Z}}) = -Mg\mathbf{r}_c \times \hat{\mathbf{Z}}$$

展开为分量：
$$M_1 = Mg(y_c\gamma_3 - z_c\gamma_2)$$
$$M_2 = Mg(z_c\gamma_1 - x_c\gamma_3)$$
$$M_3 = Mg(x_c\gamma_2 - y_c\gamma_1)$$

### 对称体的情况

对于旋转对称体（$I_1 = I_2$），质心在对称轴上：$x_c = y_c = 0$，$z_c = d$。力矩简化为：

$$M_1 = -Mgd\gamma_2 = -Mgd\sin\theta\cos\psi$$
$$M_2 = Mgd\gamma_1 = Mgd\sin\theta\sin\psi$$
$$M_3 = 0$$

将此代入欧拉旋转方程并利用运动学方程，就得到了关于欧拉角 $\phi, \theta, \psi$ 的二阶微分方程组。这就是**重陀螺方程**，将在第14章中详细讨论。

> **批注**：重力力矩的欧拉角表达式揭示了力学系统的深层结构——力矩不依赖于进动角 $\phi$（系统绕竖直轴的旋转对称性），这意味着相应的广义动量（竖直方向角动量）守恒。这种"对称性导致守恒"的关系，后来被诺特（Noether, 1918）定理所概括，成为理论物理最重要的原理之一。

### 完整的运动方程组

将力矩代入欧拉旋转方程，再结合运动学方程，得到重陀螺的完整方程组：

**动力学方程**（欧拉旋转方程 + 重力力矩）：
$$I_1\dot{\omega}_1 = (I_1 - I_3)\omega_2\omega_3 - Mgd\sin\theta\cos\psi$$
$$I_1\dot{\omega}_2 = (I_3 - I_1)\omega_1\omega_3 + Mgd\sin\theta\sin\psi$$
$$I_3\dot{\omega}_3 = 0$$

**运动学方程**：
$$\dot{\phi} = \frac{\omega_1\sin\psi + \omega_2\cos\psi}{\sin\theta}$$
$$\dot{\theta} = \omega_1\cos\psi - \omega_2\sin\psi$$
$$\dot{\psi} = \omega_3 - \frac{(\omega_1\sin\psi + \omega_2\cos\psi)\cos\theta}{\sin\theta}$$

这6个方程完整描述了受重力作用的对称刚体绕固定点的旋转运动。

## 本章小结

第11章系统地展示了欧拉旋转方程在具体受力情况下的应用：

1. **球对称体**：交叉项消失，方程完全线性化，可直接积分
2. **旋转对称体**：引入复变量 $\omega_+$ 简化前两个方程，$\omega_3$ 独立演化
3. **一般三轴不等刚体**：三个方程全耦合，需数值方法或守恒量分析
4. **一般力矩的计算方法**：在主轴坐标系中展开
5. **重力力矩的详细推导**：用方向余弦和欧拉角表示
6. **完整运动方程组**：动力学方程 + 运动学方程的6方程体系

> **批注**：本章的核心教训是：主转动惯量之间的关系决定了旋转运动的复杂性。球对称最简单（完全解耦），旋转对称次之（部分解耦），一般三轴不等最复杂（完全耦合）。此外，力矩对取向的依赖性使得运动方程成为非线性系统，这通常是不可积的——但对称性（如重力问题的竖直旋转对称性）提供的守恒量可以大大简化分析。
