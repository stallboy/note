# 附录第3章：摩擦减速下绕固定轴的旋转

*De motu gyratorio corporum gravium circa axem fixum a frictione retardato*

---

## 本章概述

欧拉讨论了摩擦力对旋转运动的减速效果。当旋转的刚体受到轴承摩擦或其他摩擦力矩时，角速度逐渐减小直至停止。

---

## 1. 轴承摩擦力矩

绕固定轴旋转的刚体，在轴承处受到摩擦力矩。如果轴承处的法向力为 $N$，轴的半径为 $r$，摩擦系数为 $\mu$：

$$M_{\text{摩擦}} = -\mu N r \cdot \text{sgn}(\omega)$$

负号表示摩擦力矩总是阻碍旋转。

## 2. 匀减速旋转

运动方程：
$$I\dot{\omega} = -\mu N r$$

解：
$$\omega(t) = \omega_0 - \frac{\mu N r}{I}t$$

停止时间：$t_{\text{停止}} = \frac{I\omega_0}{\mu N r}$

总转角：$\theta = \frac{I\omega_0^2}{2\mu N r}$

> **批注**：转动惯量 $I$ 越大，旋转停止所需的时间越长——这就是飞轮的原理。飞轮通过大的转动惯量来储存旋转动能，即使有摩擦也能长时间保持旋转。

## 3. 物理摆的摩擦阻尼

有摩擦时，物理摆的运动方程：
$$I\ddot{\theta} + \mu Nr\,\text{sgn}(\dot{\theta}) + Mgd\sin\theta = 0$$

库仑摩擦（与速度无关，只与方向有关）导致**振幅线性衰减**（不同于粘性阻尼的指数衰减）。

每半个周期，振幅减少一个固定量：
$$\Delta\theta = \frac{2\mu Nr}{Mgd}$$

最终，摆会在某个非零角度处停止（静摩擦力平衡了重力力矩）。

> **批注**：库仑摩擦与粘性阻尼的区别在于衰减的方式：粘性阻尼给出指数衰减（每周期减小相同的百分比），库仑摩擦给出线性衰减（每周期减小相同的角度值）。这个区别可以通过实验验证——记录摆的振幅随时间的变化，画在半对数坐标上。如果是直线，则是粘性阻尼；如果是线性递减，则是库仑摩擦。

---

## 本章小结

附录第3章讨论了摩擦对旋转运动的影响：

1. **轴承摩擦力矩**
2. **匀减速旋转**：转动惯量越大，减速越慢
3. **库仑摩擦阻尼**：振幅线性衰减
