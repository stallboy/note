# 正文第16章：绕柔性轴的刚体运动

*De motu corporis rigidi ad axem flexilem revocati*

---

## 本章概述

欧拉在本章中讨论了一个介于刚体和弹性体之间的系统——刚体连接在柔性（弹性）轴上的运动。这是刚体动力学与弹性动力学的耦合问题，是早期弹性-刚体耦合动力学的先驱性工作。

> **批注**：这一章的内容非常超前。弹性体与刚体的耦合动力学要到19世纪末才被系统地研究，但欧拉已经在这里处理了它的基本框架。在现代工程中，这种耦合问题无处不在——涡轮机叶片的振动、直升机旋翼的动力学、柔性机器人的控制等。欧拉的远见由此可见一斑。

---

## 1. 柔性轴的力学模型

欧拉将柔性轴建模为一个可以弯曲但提供恢复力的弹性元件。

### 基本假设

- 轴的一端固定（或连接到刚体），另一端连接到刚体
- 轴的质量可以忽略（或等效地附加到刚体上）
- 轴的弯曲变形是弹性的：弯曲角度 $\theta$ 产生正比于 $\theta$ 的恢复力矩
- 轴可以绕其长度方向自由旋转（无扭转阻力）或有扭转弹性

### 弹性恢复力矩

当轴弯曲角度为 $\theta$ 时，恢复力矩为：
$$M_{\text{弹性}} = -k\theta$$

其中 $k$ 是轴的**弯曲刚度**，取决于轴的材料和几何形状：
$$k = \frac{EI}{L}$$

这里 $E$ 是杨氏模量，$I$ 是轴截面的二次矩，$L$ 是轴的有效长度。

对于两个正交方向的弯曲（$\theta$ 和 $\phi$），恢复力矩为：
$$M_\theta = -k\theta, \quad M_\phi = -k\phi$$

> **批注**：欧拉的弹性轴模型是后来梁理论（Euler-Bernoulli beam theory）的基础之一。他在1744年的《曲线的变分法》(E65)中就已经研究了弹性梁的弯曲问题，并发现了梁的弯曲方程。在E289中，他将这些弹性力学知识与刚体动力学结合起来，开创了耦合动力学的新领域。

## 2. 刚体在弹性轴上的自由运动

将刚体固定在柔性轴的一端，轴的另一端固定。先考虑无外力的情况。

### 无自转的情况

如果刚体不自转（$\omega_3 = 0$），运动方程简化为两个独立的谐振子：

$$I_t\ddot{\theta} + k\theta = 0$$
$$I_t\ddot{\phi} + k\phi = 0$$

其中 $I_t$ 是刚体绕横向轴（垂直于轴方向）的转动惯量。

解为简谐运动：
$$\theta(t) = \theta_0\cos(\Omega_0 t + \alpha)$$
$$\phi(t) = \phi_0\cos(\Omega_0 t + \beta)$$

自然频率：$\Omega_0 = \sqrt{k/I_t}$

> **批注**：无自转时，两个弯曲方向的运动是独立的——各自以相同的频率 $\Omega_0$ 做简谐振荡。这对应于刚体在弹性约束下的最简单的振动模式。

### 有自转的情况——陀螺效应

当刚体同时自转（角速度 $\Omega$）且轴可以弯曲时，出现了陀螺效应。此时运动方程变为：

$$I_t\ddot{\theta} + I_3\Omega\dot{\phi} + k\theta = 0$$
$$I_t\ddot{\phi} - I_3\Omega\dot{\theta} + k\phi = 0$$

其中：
- $I_t$：绕横向轴的转动惯量
- $I_3$：绕自转轴（轴方向）的转动惯量
- $\Omega$：自转角速度
- $k$：轴的弯曲刚度
- $\theta, \phi$：轴端的两个弯曲角度

### 陀螺项的来源

陀螺项 $I_3\Omega\dot{\phi}$ 和 $-I_3\Omega\dot{\theta}$ 来自欧拉旋转方程中的交叉项。

在弯曲运动中，$\dot{\theta}$ 是绕一个横向轴的角速度，$\dot{\phi}$ 是绕另一个横向轴的角速度。当刚体同时以 $\Omega$ 自转时，自转角速度与横向角速度的叉积产生了一个新的力矩——这就是陀螺力矩。

> **批注**：陀螺效应的物理意义是：自转的刚体"抵抗"其旋转轴方向的改变。当轴弯曲时，陀螺效应产生一个垂直于弯曲方向的力矩，使弯曲运动在两个正交方向之间耦合。结果，弯曲振动不再是简单的简谐运动，而是变成了椭圆或圆形的运动模式。

## 3. 陀螺耦合振动的求解

引入复变量 $z = \theta + i\phi$，两个方程合并为：

$$I_t\ddot{z} - iI_3\Omega\dot{z} + kz = 0$$

尝试解 $z = Ze^{i\omega t}$，代入得特征方程：

$$-I_t\omega^2 + I_3\Omega\omega + k = 0$$

$$\omega = \frac{I_3\Omega \pm \sqrt{I_3^2\Omega^2 + 4I_tk}}{2I_t}$$

### 模态频率

两个模态频率为：
$$\omega_1 = \frac{I_3\Omega - \sqrt{I_3^2\Omega^2 + 4I_tk}}{2I_t} < 0$$
$$\omega_2 = \frac{I_3\Omega + \sqrt{I_3^2\Omega^2 + 4I_tk}}{2I_t} > 0$$

注意 $\omega_1 < 0$（反向旋转模态）和 $\omega_2 > 0$（正向旋转模态）。

### 无自转时的退化

当 $\Omega = 0$ 时：
$$\omega_1 = -\sqrt{k/I_t} = -\Omega_0, \quad \omega_2 = +\sqrt{k/I_t} = +\Omega_0$$

退化为两个以相同频率 $\Omega_0$ 反向旋转的模态——即简谐运动。

### 自转增大时的行为

当 $\Omega$ 增大时：
- $\omega_2$ 增大（正向模态频率升高）
- $|\omega_1|$ 减小（反向模态频率降低）
- 两个模态的频率差异增大

**极限情况**（$\Omega \to \infty$）：
$$\omega_2 \approx \frac{I_3\Omega}{I_t}$$
$$\omega_1 \approx -\frac{k}{I_3\Omega} \to 0$$

快旋转时，正向模态频率趋向无穷大（几乎不可能激发），反向模态频率趋向零——系统变得越来越"刚性"。

> **批注**：自转使系统更加"刚性"——这是陀螺效应的核心。在旋转机械中，高速旋转的转子在临界转速以下表现得像刚性转子（弯曲很小），但在临界转速附近会共振，超过临界转速后又恢复稳定。这个行为的分析需要欧拉在这里建立的陀螺耦合方程。

## 4. 临界转速

当外激励频率等于模态频率时，系统发生共振。对于弹性轴上的旋转刚体，临界转速对应于：

$$\Omega_{\text{临界}} \approx \Omega_0 = \sqrt{\frac{k}{I_t}}$$

更精确地说，临界转速是使某个模态频率等于旋转频率（或其倍数）的转速。

在临界转速附近，即使很小的扰动也会产生大幅振动——这对于旋转机械是极其危险的。

> **批注**：临界转速的概念是转子动力学的核心。在设计涡轮机、发动机、电机等旋转设备时，必须确保工作转速远离临界转速。欧拉在这里的分析是现代转子动力学分析的起点。

## 5. 有阻尼的情况

在实际系统中，总存在阻尼（材料内摩擦、空气阻力等）。加入阻尼后：

$$I_t\ddot{\theta} + c\dot{\theta} + I_3\Omega\dot{\phi} + k\theta = 0$$
$$I_t\ddot{\phi} + c\dot{\phi} - I_3\Omega\dot{\theta} + k\phi = 0$$

阻尼 $c > 0$ 使系统在临界转速附近的共振幅度减小，但不能完全消除。在工程设计中，通常需要在临界转速附近加入足够的阻尼来控制振动。

> **批注**：有阻尼的陀螺耦合系统展示了丰富的动力学行为，包括模态转换、参数共振等非线性现象。这些问题的完整分析需要现代动力学理论，但欧拉建立的基本框架——弹性恢复力 + 陀螺耦合 + 阻尼——至今仍是出发点。

---

## 本章小结

正文第16章引入了刚体与弹性系统的耦合动力学：

1. **弹性轴的力学模型**：弯曲恢复力矩 $M = -k\theta$
2. **无自转的振动**：简谐运动，频率 $\Omega_0 = \sqrt{k/I_t}$
3. **陀螺效应**：自转产生 $I_3\Omega\dot{\phi}$ 和 $-I_3\Omega\dot{\theta}$ 耦合项
4. **模态分析**：正向和反向两个旋转模态，频率随自转速度变化
5. **临界转速**：共振条件及其工程意义
6. **阻尼效果**：控制共振幅度

> **批注**：刚体-弹性耦合动力学是欧拉最具前瞻性的贡献之一。他不仅建立了基本方程，还识别了关键物理现象（陀螺效应、临界转速）。这些问题在后来的两个世纪中被不断深化，发展出了转子动力学、结构动力学、振动控制等完整的学科。欧拉在这里播下的种子，结出了丰硕的果实。
