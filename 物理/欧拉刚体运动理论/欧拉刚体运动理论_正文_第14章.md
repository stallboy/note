# 正文第14章：仅受重力作用的刚体运动

*De motu corporis rigidi, solius gravitatis vi moti*

---

## 本章概述

这一章中，欧拉讨论了仅受重力作用的刚体运动。这包括物理摆（复摆）的完整理论和重刚体的各种运动情形。对于绕固定点旋转的重刚体，欧拉分析了著名的"重陀螺"问题。

> **批注**：重陀螺问题是经典力学中最著名的难题之一。它看似简单——一个受重力作用的旋转刚体——但完整的解析解极其困难。欧拉建立了方程，但完整解要等到拉格朗日（1788）和柯瓦列夫斯卡娅（1888）的工作。值得注意的是，重陀螺是经典力学中少数几个完全可积的问题之一。

---

## 1. 重力力矩的计算

对于绕固定点 $O$ 旋转的刚体，质心在 $O$ 下方距离 $d$ 处。重力力矩为：

$$\mathbf{M} = dMg\hat{\mathbf{n}} \times \hat{\mathbf{Z}}$$

其中 $\hat{\mathbf{n}}$ 是质心方向的单位矢量（在固连坐标系中固定），$\hat{\mathbf{Z}}$ 是竖直向上的方向。

在主轴坐标系中，设质心坐标为 $(x_c, y_c, z_c)$，竖直方向的分量（用欧拉角表示）：

$$M_1 = Mg(y_c\gamma_3 - z_c\gamma_2)$$
$$M_2 = Mg(z_c\gamma_1 - x_c\gamma_3)$$
$$M_3 = Mg(x_c\gamma_2 - y_c\gamma_1)$$

其中 $(\gamma_1, \gamma_2, \gamma_3)$ 是竖直方向在固连坐标系中的方向余弦。

## 2. 运动积分

重陀螺有三个守恒量：

1. **能量**：$T + V = \frac{1}{2}(I_1\omega_1^2 + I_2\omega_2^2 + I_3\omega_3^2) + Mgz_c = E$
2. **竖直方向角动量**：$L_Z = L_1\gamma_1 + L_2\gamma_2 + L_3\gamma_3 = \text{常数}$
3. **绕竖直轴的不变性**：拉格朗日函数不显含进动角 $\phi$，因此对应的广义动量守恒

> **批注**：三个守恒量对于三自由度系统来说，刚好足以使系统可积（在刘维尔意义上）。这就是为什么重陀螺是可积系统——它有足够多的守恒量来"约束"运动，使得解可以用积分表示。

## 3. 对称重陀螺

对于旋转对称体（$I_1 = I_2$），质心在对称轴上（$x_c = y_c = 0$，$z_c = d$），运动方程有优美的解。

**守恒量**：
- $\omega_3 = \Omega = \text{常数}$（自转角速度）
- 竖直角动量 $L_Z = I_3\Omega\cos\theta + I_1\dot{\phi}\sin^2\theta = p_\phi$
- 能量 $E = \frac{1}{2}I_1(\dot{\theta}^2 + \dot{\phi}^2\sin^2\theta) + \frac{1}{2}I_3\Omega^2 + Mgd\cos\theta$

章动角 $\theta$ 满足等效的一维运动方程：
$$\frac{1}{2}I_1\dot{\theta}^2 = E - \frac{1}{2}I_3\Omega^2 - Mgd\cos\theta - \frac{(p_\phi - I_3\Omega\cos\theta)^2}{2I_1\sin^2\theta}$$

令 $u = \cos\theta$，方程变为：
$$\dot{u}^2 = f(u) = \text{三次多项式}$$

解为椭圆函数。

> **批注**：对称重陀螺的运动可以用椭圆函数完整描述——章动角在两个极限值之间振荡，进动角单调增加（或振荡），自转角速度恒定。这就是我们日常观察到的陀螺运动：陀螺轴绕竖直方向做圆锥运动（进动），同时上下摆动（章动），自转保持恒定。

## 4. 进动与章动

### 规则进动
当自转角速度 $\Omega$ 很大时（快陀螺），章动很小，运动近似为规则进动：

$$\dot{\phi} \approx \frac{Mgd}{I_3\Omega}$$

进动角速度与自转角速度成反比——转得越快，进动越慢。

> **批注**：这个公式解释了为什么陀螺转得越快越稳定——快陀螺的进动很慢，章动几乎为零。当陀螺减速时，进动加快，章动增大，最终陀螺倒下。这也是地球自转产生的岁差现象的基础——地球的进动周期约为26000年。

### 休止（sleeping）陀螺
当自转角速度足够大时，陀螺轴几乎竖直，章动和进动都很小，陀螺看起来"直立不动"——称为休止陀螺。稳定的条件是：
$$\Omega > \Omega_{\text{临界}} = \frac{2\sqrt{MgdI_1}}{I_3}$$

## 本章小结

正文第14章建立了重力作用下刚体运动的理论：

1. **重力力矩的计算**
2. **三个运动积分**：能量、竖直角动量
3. **对称重陀螺的解析解**：用椭圆函数描述
4. **进动与章动**：快陀螺的规则进动
5. **休止陀螺的稳定性条件**

> **批注**：重陀螺问题是经典力学的明珠之一。它展示了非线性微分方程可以产生极其丰富的动力学行为——规则进动、章动、倒立陀螺等。欧拉建立了方程并分析了主要的运动模式，为后来的拉格朗日和泊松的更深入分析铺平了道路。
