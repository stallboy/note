# 正文第18章：球体的运动

*De motu corporum sphaericorum*

---

## 本章概述

欧拉讨论了球形刚体的运动——这是最简单的三维刚体，因为三个主转动惯量相等。球体动力学的简单性使得欧拉能够给出完整的解析解。

> **批注**：球体是最"简单"的刚体——不仅因为它的几何对称性，更因为它的惯量张量在所有方向上都相同。这意味着球体的旋转运动没有任何"偏好方向"——绕任何轴的旋转都是等价的。

---

## 1. 球体的惯量性质

均匀球体（半径 $R$，质量 $M$）绕任何过球心的轴的转动惯量为：
$$I = \frac{2}{5}MR^2$$

惯量张量：$\mathbf{I} = I\mathbf{1}$（标量矩阵）

**推论**：
- 角动量总是平行于角速度：$\mathbf{L} = I\boldsymbol{\omega}$
- 所有过球心的轴都是主轴
- 惯量椭球是一个球面

## 2. 球体的自由旋转

欧拉旋转方程简化为：
$$I\dot{\omega}_1 = 0, \quad I\dot{\omega}_2 = 0, \quad I\dot{\omega}_3 = 0$$

解：$\boldsymbol{\omega} = \text{常数矢量}$

**球体的自由旋转就是匀角速度绕固定轴旋转**——与质点的匀速直线运动完全类似。

> **批注**：这是唯一一种自由旋转与绕固定轴旋转完全相同的刚体。所有其他形状的刚体，自由旋转时角速度的分量都会变化（由于 $\boldsymbol{\omega} \times \mathbf{L}$ 交叉项），但球体的交叉项为零。

## 3. 球体在力矩作用下的运动

当外力矩不为零时：
$$I\dot{\boldsymbol{\omega}} = \mathbf{M}$$

解可以直接积分：
$$\boldsymbol{\omega}(t) = \boldsymbol{\omega}_0 + \frac{1}{I}\int_0^t \mathbf{M}(t')\,dt'$$

这与质点在力作用下的运动完全平行。

## 4. 球体的滚动

欧拉讨论了球体在平面上的滚动运动。

### 纯滚动条件
$$\mathbf{v}_{\text{接触点}} = 0$$

即：$\mathbf{V}_{\text{cm}} + \boldsymbol{\omega} \times (-R\hat{\mathbf{n}}) = 0$

$$\mathbf{V}_{\text{cm}} = R\hat{\mathbf{n}} \times \boldsymbol{\omega}$$

其中 $\hat{\mathbf{n}}$ 是从球心指向接触点的单位矢量。

### 球体的动能
$$T = \frac{1}{2}MV_{\text{cm}}^2 + \frac{1}{2}I\omega^2 = \frac{1}{2}MV_{\text{cm}}^2 + \frac{1}{5}MR^2\omega^2$$

利用纯滚动条件 $V_{\text{cm}} = R\omega$：
$$T = \frac{7}{10}MV_{\text{cm}}^2$$

> **批注**：球体的滚动动能比纯平移动能大 $7/5$ 倍——额外的 $2/5$ 来自旋转动能。这就是为什么一个滚动的球比一个在无摩擦面上滑动的同等质量的块更难停下来——它有更多的总动能。

---

## 本章小结

正文第18章讨论了球形刚体的运动：

1. **惯量各向同性**：所有方向等价
2. **自由旋转**：匀角速度绕固定轴
3. **一般运动**：与质点运动完全平行
4. **滚动运动**：纯滚动条件和动能

> **批注**：球体是理解刚体旋转的"零号模型"——它展示了当惯量各向同性时，旋转运动退化为最简单的形式。
