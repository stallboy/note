# 正文第15章：绕固定点旋转的刚体运动

*De motu corporis rigidi, quod circa punctum fixum gyrari debet, a viribus quibuscunque sollicitati*

---

## 本章概述

欧拉在本章中研究了绕固定点旋转的刚体在任意外力作用下的运动。这是刚体动力学中最经典、应用最广泛的问题之一。陀螺仪、天文望远镜的指向机构、万向节、机器人关节等，都是绕固定点旋转的实例。

> **批注**：绕固定点旋转的刚体只有3个旋转自由度，比自由刚体的6个自由度简单。但它已经包含了旋转运动最丰富的现象——进动、章动、稳定性变化等。欧拉在这里的分析是整个应用刚体动力学的核心。

---

## 1. 一般受力下的运动方程

对于绕固定点 $O$ 旋转的刚体，受任意力系 $\{\mathbf{F}_i\}$ 作用，运动方程为欧拉旋转方程（在主轴坐标系中）：

$$I_1\dot{\omega}_1 - (I_2 - I_3)\omega_2\omega_3 = M_1$$
$$I_2\dot{\omega}_2 - (I_3 - I_1)\omega_3\omega_1 = M_2$$
$$I_3\dot{\omega}_3 - (I_1 - I_2)\omega_1\omega_2 = M_3$$

力矩分量：
$$\mathbf{M} = \sum_i \mathbf{r}_i \times \mathbf{F}_i$$

其中 $\mathbf{r}_i$ 是第 $i$ 个力的作用点相对于固定点 $O$ 的位置矢量（在固连坐标系中是常数）。

加上运动学方程（欧拉角与角速度的关系），共6个方程。

## 2. 旋转对称体（$I_1 = I_2$）的一般受力分析

旋转对称体的方程为：
$$I_1\dot{\omega}_1 = (I_1 - I_3)\omega_2\omega_3 + M_1$$
$$I_1\dot{\omega}_2 = (I_3 - I_1)\omega_1\omega_3 + M_2$$
$$I_3\dot{\omega}_3 = M_3$$

### 引入复变量

令 $\omega_+ = \omega_1 + i\omega_2$，$M_+ = M_1 + iM_2$。前两个方程合并为：

$$I_1\dot{\omega}_+ = i(I_3 - I_1)\omega_3\omega_+ + M_+$$

当 $\omega_3$ 已知时，这是一个关于 $\omega_+$ 的一阶线性微分方程。解为：

$$\omega_+(t) = e^{i\lambda(t)}\left[\omega_+(0) + \frac{1}{I_1}\int_0^t M_+(t')e^{-i\lambda(t')}\,dt'\right]$$

其中 $\lambda(t) = \frac{I_3 - I_1}{I_1}\int_0^t \omega_3(t')\,dt'$。

### 恒定对称轴力矩的情况

如果 $M_3 = \text{常数}$，则 $\omega_3 = \omega_{3,0} + M_3 t/I_3$。

当 $M_1 = M_2 = 0$ 时（只有对称轴方向的力矩）：

$\omega_+$ 的方程变为：
$$I_1\dot{\omega}_+ = i(I_3 - I_1)\left(\omega_{3,0} + \frac{M_3}{I_3}t\right)\omega_+$$

这是一个系数随时间线性变化的方程——角速度的横向分量以变化的频率旋转。

> **批注**：这个例子展示了旋转动力学的一个特征——即使力矩很简单（只有一个恒定分量），运动也可能相当复杂。这是因为角速度的不同分量通过交叉项耦合在一起。

## 3. 恒定横向力矩的情况

设 $M_1 = M_0$（常数），$M_2 = M_3 = 0$。

运动方程：
$$I_1\dot{\omega}_1 = (I_1 - I_3)\omega_2\omega_3 + M_0$$
$$I_1\dot{\omega}_2 = (I_3 - I_1)\omega_1\omega_3$$
$$I_3\dot{\omega}_3 = 0$$

由第三个方程：$\omega_3 = \Omega = \text{常数}$。

前两个方程：
$$\dot{\omega}_1 = \frac{I_1 - I_3}{I_1}\Omega\omega_2 + \frac{M_0}{I_1}$$
$$\dot{\omega}_2 = \frac{I_3 - I_1}{I_1}\Omega\omega_1$$

这是一个非齐次的线性耦合方程组。齐次解为本体进动（频率 $\lambda = (I_3 - I_1)\Omega/I_1$），特解为恒定偏移。

通解：
$$\omega_1(t) = \frac{M_0}{(I_3 - I_1)\Omega} + A\cos(\lambda t + \alpha)$$
$$\omega_2(t) = A\sin(\lambda t + \alpha)$$

**物理意义**：恒定横向力矩使角速度的平均值向力矩方向偏移了一个量 $M_0/[(I_3 - I_1)\Omega]$，同时叠加了一个频率为 $\lambda$ 的振荡。

> **批注**：当 $\Omega$ 很大（快旋转）时，偏移量 $M_0/[(I_3 - I_1)\Omega]$ 很小——角速度几乎不受恒定横向力矩的影响。这就是陀螺的"刚性"——快速旋转使得角速度方向难以被外力矩改变。这个性质在导航和稳定中有重要应用。

## 4. 冲量力矩的作用

欧拉详细讨论了冲量力矩（短时间内的大力矩）的效果。这在碰撞问题中至关重要。

### 冲量力矩的定义

冲量力矩 $\hat{\mathbf{M}}$ 定义为力矩在短时间内的积分：
$$\hat{\mathbf{M}} = \int_{\Delta t} \mathbf{M}(t)\,dt$$

当 $\Delta t \to 0$ 时，力矩趋于无穷大，但积分保持有限。

### 角速度的突变

冲量力矩直接改变角动量，从而改变角速度。碰撞前的角速度 $\boldsymbol{\omega}^-$ 和碰撞后的角速度 $\boldsymbol{\omega}^+$ 之间的关系为：

$$I_1(\omega_1^+ - \omega_1^-) = \hat{M}_1$$
$$I_2(\omega_2^+ - \omega_2^-) = \hat{M}_2$$
$$I_3(\omega_3^+ - \omega_3^-) = \hat{M}_3$$

即：
$$\Delta\omega_i = \frac{\hat{M}_i}{I_i}$$

> **批注**：注意，不同方向的角速度变化量反比于相应的主转动惯量——惯量越大，角速度变化越小。这与平动碰撞中 $\Delta v = \hat{F}/m$（质量越大，速度变化越小）完全平行。

### 能量损失

冲量力矩做的功为：
$$W = \int \mathbf{M} \cdot \boldsymbol{\omega}\,dt$$

对于完全弹性碰撞（能量守恒），需要满足特定的条件。一般的非弹性碰撞会损失能量：

$$\Delta T = T^+ - T^- = \frac{1}{2}\sum_i I_i[(\omega_i^+)^2 - (\omega_i^-)^2]$$

利用 $\omega_i^+ = \omega_i^- + \hat{M}_i/I_i$：

$$\Delta T = \sum_i \omega_i^-\hat{M}_i + \frac{1}{2}\sum_i \frac{\hat{M}_i^2}{I_i}$$

第一项是冲量力矩在初始角速度上做的功，第二项是新的旋转运动带来的动能。

### 具体例子：偏心碰撞

一个刚体绕固定点 $O$ 静止，受到一个冲量 $\hat{\mathbf{F}}$ 作用在距 $O$ 为 $\mathbf{r}$ 的点。

冲量力矩：$\hat{\mathbf{M}} = \mathbf{r} \times \hat{\mathbf{F}}$

碰撞后的角速度：
$$\omega_i = \frac{(\mathbf{r} \times \hat{\mathbf{F}})_i}{I_i}$$

如果 $\hat{\mathbf{F}} = \hat{F}\hat{\mathbf{z}}$（竖直方向的冲量），$\mathbf{r} = (d, 0, 0)$（沿第一个主轴），则：

$\hat{\mathbf{M}} = (d, 0, 0) \times (0, 0, \hat{F}) = (0 \cdot \hat{F} - 0, 0 - d\hat{F}, 0) = (0, -d\hat{F}, 0)$

碰撞后：
$$\omega_1 = 0, \quad \omega_2 = -\frac{d\hat{F}}{I_2}, \quad \omega_3 = 0$$

刚体绕第二个主轴以角速度 $d\hat{F}/I_2$ 开始旋转。

> **批注**：偏心碰撞在体育运动中极为常见。击打台球的偏心位置会使球旋转；踢足球时击球的偏心位置决定了球的旋转类型（上旋、下旋、侧旋）。欧拉的冲量力矩分析为这些现象提供了精确的数学描述。

## 5. 周期性力矩下的运动

欧拉讨论了周期性力矩（如振动引起的力矩）对旋转运动的影响。

设 $M_1 = M_0\cos(\Omega t)$，$M_2 = M_3 = 0$。

对于旋转对称体（$\omega_3 = \Omega_0$ 为常数），前两个方程为受迫振荡：

$$I_1\dot{\omega}_1 = (I_1 - I_3)\Omega_0\omega_2 + M_0\cos(\Omega t)$$
$$I_1\dot{\omega}_2 = (I_3 - I_1)\Omega_0\omega_1$$

这是一个受迫的耦合振荡系统。自由振荡频率为 $\lambda = (I_3 - I_1)\Omega_0/I_1$，驱动力频率为 $\Omega$。

当 $\Omega \approx \lambda$ 时发生**共振**——角速度的横向分量振幅急剧增大。

> **批注**：旋转系统中的共振现象在工程中非常重要。涡轮机、直升机旋翼等旋转机械在特定转速下会出现危险的振动——这就是旋转共振。欧拉的分析框架可以预测共振发生的条件（外力频率等于本体进动频率），为工程设计提供了指导。

## 本章小结

正文第15章全面分析了绕固定点旋转的刚体在一般受力下的运动：

1. **一般运动方程**：欧拉旋转方程 + 运动学方程
2. **旋转对称体的复变量方法**：$\omega_+$ 的线性微分方程
3. **恒定横向力矩**：角速度偏移 + 本体进动
4. **冲量力矩**：$\Delta\omega_i = \hat{M}_i/I_i$，偏心碰撞的详细分析
5. **能量损失**：非弹性碰撞中的旋转能量变化
6. **周期性力矩**：受迫振荡与共振

> **批注**：本章的内容在工程中有广泛的应用。从台球碰撞到涡轮机振动，从陀螺仪到天文望远镜的指向控制，绕固定点旋转的动力学无处不在。欧拉建立的分析框架——欧拉旋转方程 + 冲量分析 + 共振分析——至今仍是这些应用的理论基础。
