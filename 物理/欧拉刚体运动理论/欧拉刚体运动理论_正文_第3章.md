# 正文第3章：旋转运动的产生

*De motus gyratorii generatione*

---

## 本章概述

这一章中，欧拉研究了力如何产生和改变旋转运动。他引入了力矩的概念，推导了绕固定轴的旋转运动方程——这是牛顿第二定律在旋转中的对应物。更重要的是，他正式定义了**转动惯量**——刚体抵抗角加速度的度量。

> **批注**：这一章的核心可以用一句话概括：力矩产生角加速度，正如力产生线加速度。但转动惯量的引入使得这个类比变得丰富而深刻——转动惯量不仅取决于物体的总质量，还取决于质量的分布方式。这是旋转运动与直线运动的本质区别。

---

## 1. 力矩的定义

**力矩（momentum virium）**是力产生旋转效果的度量。

对于一个绕 $z$ 轴旋转的刚体，力 $\mathbf{F}$ 对 $z$ 轴的力矩为：
$$M_z = xF_y - yF_x$$

或者用极坐标（距轴距离 $r$，力在切向的分量 $F_t$）：
$$M_z = rF_t$$

更一般地，力矩是一个矢量：
$$\boldsymbol{\tau} = \mathbf{r} \times \mathbf{F}$$

> **批注**：力矩的定义揭示了力的一个基本事实——对于旋转效果而言，力的作用点和力的大小同样重要。一个力如果通过旋转轴（$r = 0$），无论多大都不会产生旋转效果。反过来，即使一个很小的力，如果作用于距离轴很远的地方（$r$ 很大），也可以产生显著的旋转效果。这就是为什么扳手越长越省力、门把手安装在门的边缘而不是中间。

## 2. 转动惯量的正式定义

**定义**：刚体绕某轴的转动惯量定义为：
$$I = \sum_i m_i r_i^2$$

其中 $m_i$ 是第 $i$ 个质点的质量，$r_i$ 是该质点到旋转轴的垂直距离。

对于连续分布的刚体：
$$I = \int r^2\,dm = \int r^2\rho(\mathbf{r})\,dV$$

> **批注**：转动惯量是欧拉对力学最重要的贡献之一。它揭示了旋转运动的一个基本事实：质量分布比总质量更重要。两个质量相同的刚体，如果一个把质量集中在旋转轴附近，另一个把质量集中在远离轴的地方，后者的转动惯量远大于前者，因此更难被加速（或减速）旋转。这就是为什么飞轮（质量集中在边缘的圆盘）能够有效地储存旋转动能，而花样滑冰运动员收回手臂时转得更快（转动惯量减小，角速度增大以保持角动量守恒）。

## 3. 旋转运动方程

**命题**：绕固定轴旋转的刚体，其角加速度等于外力矩除以转动惯量：
$$\boxed{I\frac{d\omega}{dt} = M_{\text{外}}}$$

### 推导

欧拉的推导基于对每个质点应用牛顿第二定律。

第 $i$ 个质点在切向的运动方程为：
$$m_i a_{i,t} = F_{i,t}^{\text{外}} + \sum_{j \neq i} F_{ij,t}$$

其中 $a_{i,t} = r_i\dot{\omega}$ 是切向加速度，$F_{ij,t}$ 是内力的切向分量。

乘以 $r_i$ 并对所有质点求和：
$$\sum_i m_i r_i^2 \dot{\omega} = \sum_i r_i F_{i,t}^{\text{外}} + \sum_i\sum_{j \neq i} r_i F_{ij,t}$$

内力的贡献：由于 $\mathbf{F}_{ij} = -\mathbf{F}_{ji}$（牛顿第三定律），且 $r_i = r_j$（沿力的作用线），内力矩成对抵消。

因此：
$$\dot{\omega}\sum_i m_i r_i^2 = \sum_i r_i F_{i,t}^{\text{外}}$$

即：
$$I\dot{\omega} = M_{\text{外}}$$

$\blacksquare$

> **批注**：这个推导与第1章中质心运动定理的推导有着相同的结构——关键步骤都是内力的成对抵消。但由于这里乘了一个因子 $r_i$（到轴的距离），内力不仅大小相等方向相反，而且力矩也成对抵消。这要求内力沿两质点的连线方向——欧拉隐含地假设了这一点。

## 4. 角动量形式的运动方程

将 $I\dot{\omega} = M$ 写成角动量的形式：

$$\frac{dL}{dt} = M_{\text{外}}$$

其中 $L = I\omega$ 是角动量。

**角动量定理**：角动量的时间变化率等于外力矩。

**推论**：当外力矩为零时，$L = I\omega = \text{常数}$（角动量守恒）。

> **批注**：角动量定理是刚体动力学最基本的定理。欧拉早在1752年（E177）就发现了它的一般形式，在这里它被严格地应用于绕固定轴旋转的特殊情形。注意，对于绕固定轴的情况，$I$ 是常数（刚体形状不变），所以 $L = I\omega$ 守恒直接意味着 $\omega$ 守恒。但在一般情形下（转动惯量可能随时间变化），情况要复杂得多。

## 5. 力矩做的功

力矩 $M$ 在角位移 $d\theta$ 上做的功为：
$$dW = M\,d\theta$$

功率为：
$$P = M\omega$$

力矩做的功转化为旋转动能：
$$W = \int M\,d\theta = \Delta\left(\frac{1}{2}I\omega^2\right)$$

> **批注**：这与平移情况下的功-能关系完全平行：$W = \int F\,dx = \Delta(\frac{1}{2}mv^2)$。力矩对应力，角位移对应线位移，旋转动能对应平移动能。

## 6. 常见几何形状的转动惯量

欧拉计算了几个简单形状的转动惯量，作为后面更一般理论的具体示例：

### 均匀细杆（绕中心，垂直于杆）
$$I = \frac{1}{12}mL^2$$

### 均匀细杆（绕一端，垂直于杆）
$$I = \frac{1}{3}mL^2$$

### 均匀圆盘（绕中心轴）
$$I = \frac{1}{2}mR^2$$

### 均匀球体（绕直径）
$$I = \frac{2}{5}mR^2$$

### 均匀圆柱体（绕对称轴）
$$I = \frac{1}{2}mR^2$$

> **批注**：这些公式至今仍是每本力学教材中的标准内容。它们显示了转动惯量如何依赖于质量分布：质量越远离旋转轴，转动惯量越大。例如，将质量集中在边缘的圆环的转动惯量为 $I = mR^2$，而均匀圆盘的转动惯量只有 $I = \frac{1}{2}mR^2$——同样的质量和半径，但分布更集中。

## 7. 平行轴定理

欧拉证明了一个极其有用的定理：

**刚体绕任意轴的转动惯量 $I$ 等于绕过质心的平行轴的转动惯量 $I_{\text{cm}}$ 加上总质量乘以两轴间距离的平方：**
$$I = I_{\text{cm}} + Md^2$$

其中 $d$ 是两平行轴之间的距离。

### 证明

设旋转轴为 $L$（过点 $O$），质心 $C$ 到轴 $L$ 的垂直距离为 $d$。过质心 $C$ 作轴 $L$ 的平行线 $L'$。

建立直角坐标系：取轴 $L$ 为 $z$ 轴方向，原点 $O$ 在轴 $L$ 上。第 $i$ 个质点的位置矢量 $\mathbf{r}_i$ 可分解为平行于轴的分量 $z_i\hat{\mathbf{k}}$ 和垂直于轴的分量 $\boldsymbol{\rho}_i$（即从轴到质点的垂直矢量）：

$$\mathbf{r}_i = \boldsymbol{\rho}_i + z_i\hat{\mathbf{k}}$$

到轴 $L$ 的垂直距离的平方为：
$$r_{i,\perp}^2 = |\boldsymbol{\rho}_i|^2 = x_i^2 + y_i^2$$

绕轴 $L$ 的转动惯量：
$$I = \sum_i m_i r_{i,\perp}^2 = \sum_i m_i(x_i^2 + y_i^2)$$

现在将质点位置分解为质心到轴的垂直矢量 $\mathbf{d}$ 加上相对于质心的垂直矢量 $\boldsymbol{\rho}_i'$：

$$\boldsymbol{\rho}_i = \mathbf{d} + \boldsymbol{\rho}_i'$$

其中 $\mathbf{d}$ 是从轴 $L$ 到质心 $C$ 的垂直矢量（$|\mathbf{d}| = d$），$\boldsymbol{\rho}_i'$ 是从轴 $L'$ 到第 $i$ 个质点的垂直矢量。

代入：
$$r_{i,\perp}^2 = |\boldsymbol{\rho}_i|^2 = |\mathbf{d} + \boldsymbol{\rho}_i'|^2 = d^2 + 2\mathbf{d}\cdot\boldsymbol{\rho}_i' + |\boldsymbol{\rho}_i'|^2$$

乘以 $m_i$ 并求和：
$$I = \sum m_i r_{i,\perp}^2 = d^2\sum m_i + 2\mathbf{d}\cdot\sum m_i\boldsymbol{\rho}_i' + \sum m_i|\boldsymbol{\rho}_i'|^2$$

中间项：$\sum m_i\boldsymbol{\rho}_i' = M\boldsymbol{\rho}_{\text{cm}}' = 0$（因为 $\boldsymbol{\rho}_i'$ 是相对于质心量度的，质心相对于自身的位置为零）。

最后一项：$\sum m_i|\boldsymbol{\rho}_i'|^2 = I_{\text{cm}}$（绕过质心的平行轴 $L'$ 的转动惯量）。

因此：
$$\boxed{I = I_{\text{cm}} + Md^2}$$

$\blacksquare$

$\blacksquare$

> **批注**：平行轴定理表明，在所有平行的轴中，过质心的那个轴对应的转动惯量最小。这意味着旋转轴越远离质心，转动惯量越大——这与直觉一致：让物体绕远离其"中心"的轴旋转更"费力"。

## 本章小结

正文第3章建立了绕固定轴旋转的动力学：

1. **力矩**：$\tau = \mathbf{r} \times \mathbf{F}$，力产生旋转效果的度量
2. **转动惯量**：$I = \sum m_i r_i^2$，抵抗角加速度的度量
3. **旋转运动方程**：$I\dot{\omega} = M$（力矩 = 转动惯量 × 角加速度）
4. **角动量定理**：$\dot{L} = M$
5. **旋转功-能关系**：$\int M\,d\theta = \Delta(\frac{1}{2}I\omega^2)$
6. **平行轴定理**：$I = I_{\text{cm}} + Md^2$

> **批注**：到这一章为止，欧拉已经完整地建立了绕固定轴旋转的理论。但这个理论有一个根本性的限制——旋转轴必须是固定的。在实际中，旋转轴通常不是固定的（想想一个抛在空中的旋转物体）。从下一章开始，欧拉将逐步放松这个限制，最终达到最一般的情形。
