# 第四章 有阻力介质中的直线运动

## 指数衰减、终端速度与阻尼振动的发现

---

## 阻力是什么

前两章处理的都是"绝对力"——重力、弹性力这些与物体运动状态无关的力。但现实中，物体在空气或水中运动会受到**阻力**。

阻力的两个特点：
1. 方向始终与运动方向相反（只减速，不拐弯）
2. 大小依赖于速度（通常速度越快阻力越大）

欧拉指出：虽然阻力在物理上是一种不同的力，但在数学上可以把它当作一种特殊的绝对力来处理——只要把介质看作静止的，阻力就变成了一个与速度有关的、方向与运动相反的力。

> **评注**：欧拉将阻力介质问题转化为"静止介质中运动物体受绝对力作用"的框架，本质上是选择了一个方便的参考系。这一处理完全正确，但在相对论力学中需要注意：参考系变换不再是简单的伽利略变换。在牛顿力学范围内，将阻力视为速度的函数 $R(v)$ 并纳入统一的微分方程框架，在方法论上是有效的——它将第二章建立的绝对力理论直接推广到了包含速度依赖力的情形。

## 只有阻力、没有其他力的情况

先看最简单的：一个物体在阻力介质中运动，没有重力或其他力。

### 阻力正比于速度

设阻力 $R = kv$。运动方程：$\frac{dv}{dt} = -kv$。

这是一个简单的微分方程，解出来就是：

$$\boxed{v = v_0 e^{-kt}}$$

**指数衰减！** 物体的速度以指数方式衰减，越接近零越慢，理论上永远到不了零。

> **评注**：这是指数函数 $e^{-kt}$ 在力学中的**首次系统应用**。指数衰减的正确性已被此后两个多世纪的物理学反复验证——它不仅出现在阻力问题中，还是放射性衰变（$N = N_0 e^{-\lambda t}$）、RC电路放电（$V = V_0 e^{-t/RC}$）、光的比尔-朗伯吸收（$I = I_0 e^{-\alpha x}$）等一切一阶线性衰减过程的数学形式。欧拉是第一个认识到这一数学形式之普遍性的人。需要指出的是，$R \propto v$ 的阻力定律（斯托克斯阻力）仅适用于低雷诺数（$Re \ll 1$）的情况，如微小颗粒在粘性流体中的运动。

### 阻力正比于速度平方

自然界中观察到的主要阻力形式。设 $R = kv^2$。运动方程：$\frac{dv}{dt} = -kv^2$。

解：

$$v = \frac{v_0}{1 + kv_0 t}$$

速度按**双曲衰减**——比指数衰减更慢。$1/(1+at)$ 形式的衰减在流体力学中至今是基本结果。

### 一般幂次

欧拉用类似方法处理了 $R \propto v^{1/2}, v, v^{3/2}, v^2, v^3$ 等各种情况。纯粹是为了数学上的丰富性——自然界中观察到的基本只有平方阻力，但其他情况提供了很好的分析练习。

## 重力+阻力：有阻力的下落

现在加入重力。物体在重力 $g$ 和阻力 $R$ 同时作用下下落。

### 阻力正比于速度的情况

运动方程：$\frac{dv}{dt} = g - kv$

解：

$$v = \frac{g}{k}(1 - e^{-kt})$$

> 当 $t \to \infty$ 时，$v \to g/k$。

**这就是终端速度！** 物体从静止开始加速，但由于阻力随速度增大而增大，最终重力与阻力达到平衡，物体以恒定速度下落。

整个过程是这样的：
- 初始时重力主导 → 加速
- 速度增大 → 阻力增大
- 最终 $mg = R$ → 匀速下落

> **评注**：终端速度的概念完全正确，且具有直接的物理意义：当阻力等于重力时加速度为零，物体以恒定速度下落。对于 $R \propto v$ 的情况，$v_T = mg/k$；对于 $R \propto v^2$ 的情况，$v_T = \sqrt{mg/k}$。现代修正：实际终端速度的数值取决于物体形状和流体性质，需要用阻力系数 $C_d$ 来计算（$v_T = \sqrt{2mg/(\rho A C_d)}$），其中 $\rho$ 是流体密度，$A$ 是截面积。欧拉的解析处理忽略了这些工程细节，但物理框架是完全正确的。

### 阻力正比于速度平方的情况

解涉及双曲正切函数：$v = v_T \tanh^2(\alpha x)$

终端速度仍然是 $v_T = g/k$，但达到终端速度的方式不同——双曲正切而非指数。

## 一般框架方程

在处理各种具体情形之前，欧拉先建立了一个统一的框架。当绝对力 $p$ 和阻力 $R$ 同时作用时，功-能关系的微分形式变为：

$$dv = p\,dx - \frac{V}{Q}\,dx$$

其中 $V$ 是阻力定律函数（如 $V = v^m$），$Q$ 是阻力指数。这就是说：**速度对应高度的变化 = 绝对力做的功 − 阻力消耗的功**。

欧拉系统分析了这个方程何时可以分离变量求解。对于 $R \propto v^2$（最物理的情况），他引入了**积分因子** $e^{x/k}$ 来求解一阶线性微分方程——这是该方法的重要早期实例。

> **评注**：积分因子法是求解一阶线性常微分方程的标准方法（现代教材中写成 $e^{\int P(x)dx}$ 乘子）。欧拉在此的应用表明他已经完全掌握了这一技术，尽管系统的一般理论要到后来才由莱布尼茨和克莱罗等人发展。这一方法论的贡献经常被其物理结果的辉煌所掩盖。

## 有心力 + 阻力

欧拉没有止步于均匀力——他还将阻力加入了第三章讨论的变力（中心力）框架。

当有心力正比于距离的 $n$ 次幂，同时存在阻力时，运动方程通过积分因子 $e^{x/k}$ 可以约化为一阶微分方程。欧拉给出了上升和下降的完整解。

一个有趣的结果：在有心力+阻力的情况下，欧拉处理了**等时下落问题**——求使从任意高度下落都等时的力律。答案是弹性恢复力（力正比于距离），总下落时间正比于 $\pi$（常数），与无阻力情况一致——但这个等时性只对半周期成立，因为阻力破坏了上升和下降的对称性。

> **评注**：有心力+阻力的组合在物理上是重要的情况——例如卫星在大气层中螺旋下降、带电粒子在介质中辐射减速等。欧拉的框架虽然在力律选择上受到时代限制，但处理方法是完全正确的。现代天体力学中处理大气阻力对卫星轨道的影响时，仍使用类似的微分方程框架。

### 上升与下降的不对称性

在有阻力时，上抛运动和自由下落不再对称。上升时重力和阻力**都减速**，下降时重力加速但阻力减速。这意味着：
- 上升时间 < 下降时间
- 落回出发点的速度 < 上抛初速度
- 能量被阻力耗散，不可逆

这与第三章中欧拉证明的时间可逆性形成鲜明对比——阻力打破了时间反演对称性。

## 阻尼简谐运动的等时性

这是一个划时代的分析。欧拉处理了弹性恢复力（力正比于距离，指向平衡位置）加上阻力的组合。

恢复力：$F = -kx$（负号表示指向平衡位置）
阻力：$R = cv^n$

当阻力正比于速度（$n=1$）时，运动方程是：

$$m\frac{d^2x}{dt^2} + c\frac{dx}{dt} + kx = 0$$

用标准写法：

$$\ddot{x} + 2\gamma\dot{x} + \omega^2 x = 0$$

解：

$$\boxed{x = A e^{-\gamma t}\cos(\omega' t + \phi)}$$

其中 $\omega' = \sqrt{\omega^2 - \gamma^2}$。

**这就是阻尼简谐运动的完整解。** 欧拉首次用解析方法得到了三种情况：

1. **欠阻尼**（$\gamma < \omega$）：振幅指数衰减的振荡。你推一下门，门来回摆几次后停住。
2. **过阻尼**（$\gamma > \omega$）：无振荡，缓慢回到平衡位置。你推一下很粘的液体里的物体，它慢慢蠕回去。
3. **临界阻尼**（$\gamma = \omega$）：最快的无振荡回归。这是工程师最喜欢的设计点——天平的指针、汽车的减震器都调在临界阻尼附近。

> **评注**：阻尼简谐运动的解 $x = Ae^{-\gamma t}\cos(\omega' t + \phi)$ 是正确的，其中三种阻尼状态的分类（欠阻尼、过阻尼、临界阻尼）构成了现代振动理论的基础。牛顿的几何方法确实无法处理此类二阶线性常系数微分方程——这恰恰体现了分析方法的优越性。欧拉的解在数学上与现代结果完全一致，唯一的现代修正是在欠阻尼情形中 $\omega' = \sqrt{\omega_0^2 - \gamma^2}$ 的精确表达式（欧拉得到了等价的结果，只是记号不同）。在工程应用中，临界阻尼（$\gamma = \omega_0$）具有特殊重要性——它是系统在不振荡的情况下最快回到平衡位置的阻尼强度，广泛应用于减震器、天平指针、仪表表头的设计中。

## 第四章总结

| 主题 | 核心结果 | 现代对应 |
|------|---------|---------|
| 阻力 $\propto$ 速度 | $v = v_0 e^{-kt}$ | 指数衰减 |
| 阻力 $\propto$ 速度² | $v = v_0/(1+kv_0 t)$ | 双曲衰减 |
| 重力+阻力下落 | 终端速度 $v_T = g/k$ | 斯托克斯/牛顿阻力 |
| 阻尼简谐运动 | $x = Ae^{-\gamma t}\cos(\omega't+\phi)$ | 振动理论的基石 |

欧拉在这一章中首次系统地将指数函数用于力学问题的求解，建立了阻尼和衰减现象的完整数学理论。从这章开始，$e$ 成了力学中无处不在的常客。
