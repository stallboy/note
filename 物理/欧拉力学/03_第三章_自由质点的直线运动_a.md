# 第三章 自由质点的直线运动（上）

## 匀力、变力与中心力的基本理论

---

## 为什么从直线运动开始

第二章建立了力学的基本原理：$F = ma$ 和 $d(\frac{1}{2}v^2) = \frac{F}{m}ds$。现在欧拉开始应用这些原理了。

他先处理**直线运动**——不是因为直线运动最重要，而是因为它最简单。直线运动发生在一种特殊情况下：**力的方向与运动方向在同一条直线上**。

如果力和运动方向共线，力只改变速度大小，不改变方向。所以运动始终沿直线。

这可以两种方式出现：
- 力与运动同向 → 加速
- 力与运动反向 → 减速

## 匀力下的落体

最简单的情况：一个恒定的力（比如地球表面的重力）把一个质点从静止往下拉。

设质量为 $m$，力为 $F$（常数），通过的距离为 $s$。从功-能定理出发：

$$d\!\left(\frac{1}{2}v^2\right) = \frac{F}{m}\,ds$$

积分，利用起点速度为零：

$$\frac{1}{2}v^2 = \frac{F}{m}s$$

所以：

$$\boxed{v = \sqrt{\frac{2Fs}{m}}}$$

速度与下落距离的平方根成正比。这就是伽利略的自由落体定律——他先从实验发现，现在欧拉从力学原理直接推出了。

用加速度 $a = F/m$ 写就是：

$$v^2 = 2as, \quad s = \frac{1}{2}at^2, \quad v = at$$

### 地球上的具体数值

欧拉用了当时的实验数据：物体在地球表面1秒内下落约15625莱茵尺分（约4.9米）。

从 $s = \frac{1}{2}at^2$，$s = 15625$，$t = 1$，得 $a = 31250$ 莱茵尺分/秒² ≈ 9.81 m/s²。这就是重力加速度 $g$。

> **评注**：欧拉选择莱茵尺分（scruple）作为距离单位，因为 $15625 = 125^2$ 是完全平方数，避免了频繁的无理数运算。这一选择反映了前计算时代的实用考量。现代物理学则通过统一单位制（SI）来避免此类问题，但在概念层面并无本质区别——选择合适的单位制以简化计算始终是重要的方法论原则。

### "速度对应高度"

欧拉引入了一个很聪明的概念：**对应于某个速度的高度**——就是物体从该高度在地表自由下落时恰好获得这个速度的高度。

这个高度正比于速度的平方：$h \propto v^2$。

用现代的话说，$h = v^2/(2g)$，也就是 $mgh = \frac{1}{2}mv^2$——重力势能等于动能。这个概念其实就是**用能量来表示速度**，让很多计算变得方便。

欧拉用符号 $v$ 表示这个高度（注意这跟现代用 $v$ 表示速度不同），然后基本的功-能关系就变成了 $dv = (F/m)\,ds$——非常简洁。

### 末速度 = 2 × 平均速度

一个关于匀加速运动的经典性质：物体从静止下落通过距离 $s$ 后的速度，等于以匀速在同样时间内走过 $2s$ 的速度。

这就是 $v_{\text{末}} = 2\bar{v}$——匀加速运动的末速度是平均速度的两倍。

### 从不同天体表面下落

牛顿指出，同一物体在太阳、木星、土星、地球表面所受重力之比为 10000 : 835 : 525 : 400。

从功-能定理 $v^2 = 2(F/m)s$ 可知，获得相同速度所需的高度与重力加速度成反比。所以同一物体从太阳表面下落获得一定速度只需要地球表面 $\frac{400}{10000}$ 即 $4\%$ 的高度。

而且所有物体在同一个天体表面等速下落——这意味着重力与质量成正比，加速度对所有物体相同。

## 有初速的情况

如果物体不是从静止出发，而是已经有了一个初速度呢？

从功-能定理：$\frac{1}{2}v^2 = \frac{1}{2}v_0^2 + \frac{F}{m}s$

即：

$$v^2 = v_0^2 + 2as$$

这就是大家熟悉的匀加速运动公式的一般形式。

### 上抛运动：完美的对称性

如果初速度向上，力（重力）向下，物体做减速运动。到最高点速度为零，然后加速下落。

关键结论：**上抛运动和自由下落完全对称**——同一位置的速度大小相同，上升时间等于下降时间。物体以多大的速度上抛，就回到出发时多大的速度。

> **评注**：欧拉在此实质上表述了经典力学的**时间可逆性**：在保守力系统中，若将所有速度反向，系统将沿原路径返回。这一定理完全正确，且在分析力学中有精确表述：牛顿方程在时间反演 $t \to -t$ 下不变（当力为保守力时）。当存在耗散力（阻力、摩擦）时，时间反演对称性破缺，运动不可逆。时间可逆性与热力学第二定律（熵增）之间的张力，是19世纪统计力学和现代非平衡态物理学的核心议题。

## 变力的世界：力心与中心力

匀力的情况处理完了。欧拉说，现在来看更现实的力——**随位置变化的力**。

在自然界中，几乎所有的力都是变化的：
- 行星离太阳越近，被吸引越强
- 物体离地表越远，重量越小
- 磁铁对铁屑的吸引随距离减弱

最常见的情况是力指向一个固定点（**力心**），大小依赖于到力心的距离。这种力叫**向心力**（如果推离力心则叫离心力）。

### 能量积分：$v$ 作为距离的函数

设力心在 $C$，物体从距离 $a$ 处从静止开始被吸引。力的大小正比于距离的 $n$ 次幂。力等于重力时的距离为 $f$。

从功-能定理，到达距力心 $y$ 处的速度对应高度：

$$v = \frac{a^{n+1} - y^{n+1}}{(n+1)f^n}$$

用现代符号：$\frac{1}{2}v^2 = \int_{y}^{a} \frac{r^n}{f^n}\,dr = \frac{a^{n+1} - y^{n+1}}{(n+1)f^n}$

这就是**能量守恒**：动能 = 势能的减少。

### 各种力律下到达力心的速度

- $n + 1 > 0$（包括 $n = 1, 0, -2$ 等）：到达力心时速度**有限**
- $n + 1 < 0$：到达力心时速度**无穷大**
- $n = -1$（力与距离成反比）：速度 $v \propto \ln(a/y)$，到达力心时也是无穷大

> **评注**：力心处速度无穷大的结论在数学上正确，但在物理上需要修正。实际物体有有限大小，不会到达真正的力心——碰撞会在有限距离处发生。在更基本层面，$F \propto r^n$（$n < -1$）在 $r \to 0$ 时的发散表明牛顿力学在小尺度上失效，需要量子力学或广义相对论的修正。但对天体力学中的轨道问题，物体从不接近力心到这种程度，所以经典分析完全适用。

### 过了力心怎么办？

如果 $n$ 是奇数（$n = 1, 3, 5, \ldots$），物体到达力心后继续前进，力在另一侧方向反转，物体做**往复运动**——在两个对称点之间来回振荡。而且两个对称位置的速度相等。

$n = 1$（力正比于距离）的情况特别重要：物体在力心两侧做**简谐运动**。

## 匀力下的时间公式

回到匀力情况。从静止下落距离 $s$ 的时间：

$$t = \sqrt{\frac{2s}{a}}$$

有初速的情况：

$$t = \frac{\sqrt{v_0^2 + 2as} - v_0}{a}$$

上抛到最高点的时间等于从最高点自由下落的时间。

## 等时下落：力正比于距离的奇迹

当力与距力心的距离成正比（$n = 1$）时，欧拉发现了一个惊人的性质：

**从任意距离出发的下落，到达力心的时间都相同！**

总下落时间 = $\pi\sqrt{f/2}$，与起始距离无关。

> **评注**：力正比于距离（$F \propto r$）时的等时性是简谐运动的核心性质。这一结论完全正确，在现代力学中可从微分方程 $\ddot{x} + \omega^2 x = 0$ 直接得出：周期 $T = 2\pi/\omega$ 与振幅无关。物理上，$n=1$ 对应胡克定律（弹性恢复力），是弹簧、摆的小角度近似、LC电路等一切线性振荡系统的模型。伽利略发现等时性的实验观察，至此获得了完整的理论解释。需要注意的是，严格的等时性只对理想的线性恢复力成立，真实摆在大角度下偏离等时性。

## 逆平方力下的时间与开普勒第三定律

当力与距离平方成反比（$n = -2$，即牛顿引力）时：

$$v = \frac{f^2(a-y)}{ay}$$

下落时间用圆弧来表示（计算略）。总下落时间正比于 $a^{3/2}$——距离的二分之三次方。

**这就是开普勒第三定律**（$T^2 \propto a^3$）的一个特殊情况——沿直线落向太阳的时间也遵循同样的规律。

## 单位的确定

欧拉在这一章中确定了之前一直悬而未决的比例常数 $n$ 的值。

通过令地球表面重力加速度为 1（即 $F/m = 1$），得出 $n = 1/2$。这使得功-能关系变成极其简洁的形式：

$$dv = \frac{F}{m}\,ds$$

其中 $v$ 是速度对应高度，$F/m$ 是力与重量之比（即相对加速度）。从此以后，所有的计算都不再需要拖着一个比例常数了。

> **评注**：欧拉令地表重力加速度 $g = 1$（即力/质量=1）的做法，等价于选择了一个特殊的单位制，使得功-能关系 $d(\frac{1}{2}v^2) = (F/m)\,ds$ 中的系数全部为1。在现代物理学中，这对应于"自然单位制"的思想——选择单位使得某些物理常数等于1，以简化公式。粒子物理中令 $\hbar = c = 1$ 的做法与此一脉相承。不同单位制的选取不影响物理结论的正确性，仅影响计算的便利性。
