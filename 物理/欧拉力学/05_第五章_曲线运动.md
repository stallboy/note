# 第五章 曲线运动

## 全书精华——牛顿《原理》的解析重建

---

## 为什么曲线运动是高潮

前四章处理的都是直线运动。但自然界中大多数运动都是曲线——行星绕太阳转、炮弹飞过天空、摆来回摆动。直线运动只是特例。

现在欧拉要处理最一般的情况：力与运动方向不一致，物体被迫沿曲线运动。这是全书的精华——欧拉将在这里用解析方法重建牛顿《原理》的核心内容。

## 曲线运动的两个基本方程

这是整个第五章的基石。欧拉说：不管曲线运动看起来多复杂，本质上只受两个方程控制。

想象你在一条弯曲的公路上开车。你的速度变化（踩油门或刹车）和方向变化（转动方向盘）是两件独立的事。欧拉把力也按同样的方式拆开了——**切向分量**和**法向分量**。

### 第一个方程——管方向的（法向方程）

$$\boxed{a_n = \frac{v^2}{\rho}}$$

要让物体沿着曲率半径为 $\rho$ 的曲线运动，需要一个指向曲率中心的法向力 $F_n = mv^2/\rho$。速度越快，弯越急，需要的力就越大。

这就是你开车转弯时被"甩"向车窗的感觉——不是真的有什么力在甩你，而是你的身体想走直线（惯性），车子在拐弯。

### 第二个方程——管速度的（切向方程）

$$\boxed{m\frac{dv}{dt} = F_t}$$

牛顿第二定律的切向版本：沿运动方向的力负责加速或减速。

> **评注**：这两个方程的正确性毋庸置疑——它们是牛顿第二定律在自然坐标系下的投影。法向方程 $a_n = v^2/\rho$ 是向心加速度的正确表达式（牛顿在《原理》中用几何方式得到过等价结果），切向方程 $ma_t = F_t$ 是牛顿第二定律的切向分量。欧拉的贡献不在于发现这些方程本身，而在于将它们系统化、代数化为标准形式，使之成为可一般性应用的工具。这一定式化方式在现代力学教材中仍然通用。

## 抛体运动——微分方程推导抛物线

欧拉先把新框架应用到最简单的曲线运动——重力竖直向下的抛体运动。这是对整个方法的第一次检验。

### 推导过程

**设定**：均匀重力 $g$ 竖直向下，物体从 $A$ 点以速度 $v_0$ 沿方向 $AH$ 抛出，$AH$ 与水平线 $AB$ 的夹角的正弦为 $\alpha$。取水平方向为 $x$，竖直方向为 $y$（向上为正）。

**第一步：列出运动方程。** 对均匀力 $g$，两个基本方程变为：

$$dv = -g\,dy$$

$$gr\,dx = 2v\,ds$$

**第二步：积分能量方程。** 第一个方程直接积分：

$$v = c - gy$$

其中 $c = v_0^2/2g$ 是初始速度对应的高度（初始条件 $y=0$ 时 $v=c$）。这就是能量守恒：动能 + 势能 = 常数。

**第三步：代入曲率关系。** 由 $r\,dx = 2v\,ds/g$ 和曲率公式 $r = ds^3/(-dx\,ddy)$（取 $dx$ 为常数），联立得到：

$$-g\,ds^2 = 2(c-gy)\,ddy$$

**第四步：变量分离并积分。** 利用 $ds^2 = dx^2 + dy^2$，经过代数运算和积分，最终得到轨迹方程：

$$\boxed{y = \alpha x - \frac{g\,x^2}{4\alpha^2 c}}$$

**这正是一条抛物线。**

### 重要推论

从这个方程可以直接读出伽利略的经典结果：

| 性质 | 结果 |
|------|------|
| 落回水平线 $AB$ 的距离 | $AB = 4\alpha c/g$ |
| 最大射程（对 $g$ 为常数） | $2c/g$，在 45° 抛射时取得 |
| 最大高度 | $DE = \alpha^2 c/g$，在 $x = AB/2$ 处 |
| 水平运动 | 匀速，时间 $t = x/\sqrt{c}$ |
| 抛物线参数 | $4\alpha^2 c/g$ |
| 轨道上 $M$ 点的速度 | $v = g \cdot MF$（$MF$ 是 $M$ 到焦点的距离） |

最后一个结果特别深刻：**轨道上任意一点的速度，等于从焦点距离的高度自由落体获得的速度**。这把抛物线的几何性质与能量守恒直接联系起来了。

> **评注**：欧拉的推导将伽利略的抛物线结果从一个"实验发现"提升为"微分方程的必然推论"。更重要的方法论意义在于：推导过程是机械化、可重复的——你可以用完全相同的步骤处理任何力函数。事实上，欧拉紧接着就做了这件事。

### 任意力函数下的抛体运动

欧拉指出，如果竖直向下的力不是常数 $g$，而是高度 $y$ 的任意函数 $P(y)$，同样的方法仍然适用。

能量方程变为 $v = c - \int P\,dy$，曲率关系不变。经过类似的推导：

- **水平运动仍然是匀速的**——这是出人意料的结论！不管竖直方向的力有多复杂，水平速度始终恒定。
- 轨迹方程由 $dx = dy\sqrt{c}/\sqrt{\alpha^2 c - \int P\,dy}$ 确定，只要力函数已知就能积分。
- 曲线关于过最高点的竖直线对称。

逆问题也可以解：给定轨道曲线，反推出所需的力函数。

## 平面内的有心力运动

现在进入重头戏。物体被一个指向固定点（力心）的力吸引，而且力的大小只依赖于到力心的距离。这就是**有心力运动**——行星运动的物理模型。

### 角动量守恒的推导

欧拉不是简单陈述角动量守恒——他从运动方程推出了它。推导的核心步骤：

设力心在 $C$，物体在 $M$，$CM = y$。从 $C$ 到切线的垂距为 $p$。

运动方程：$dv = -P\,dy$（径向），$Pr\,dx = 2v\,ds$（法向）。

消去力 $P$，得 $\frac{dv}{v} = \frac{-2\,ds\,dy}{r\,dx}$。

关键替换：利用几何关系 $r = y\,dy/dp$ 和 $ds/dx = y/p$，代入得：

$$\frac{dv}{v} = -2\,\frac{dp}{p}$$

积分即得：

$$\boxed{v \cdot p^2 = \text{const}}$$

由于 $p = y\sin\phi$（$\phi$ 是位矢与速度的夹角）且 $v \propto (ds/dt)^2$，这等价于：

$$r^2\dot{\theta} = \text{const}$$

用几何语言说：物体与力心的连线在相等时间内扫过相等的面积——这就是**开普勒第二定律**（面积定律）。

> **评注**：$dv/v = -2\,dp/p$ 这一步是整个推导的关键。它依赖于两个几何关系：$r = y\,dy/dp$（曲率半径与垂距的关系）和 $ds/dx = y/p$（弧元与径向距离的关系）。这两个关系来自相似三角形的分析。欧拉的推导完全正确，且比牛顿的几何论证更加透明。

> **评注**：角动量守恒 $L = mr^2\dot{\theta} = \text{const}$ 是正确的。开普勒发现面积定律（1609年），牛顿在《原理》中将其与向心力联系起来，但都是几何表述。欧拉首次将其表达为解析的微分关系 $r^2 d\theta = C\,ds$ 并用作求解工具——从"已知规律"转变为"计算工具"的方法论升级。在现代理论力学中，角动量守恒从转动对称性（诺特定理）自然得出，比欧拉的时代有了更深刻的理解。

### 轨道方程的分离

对于有心力运动，欧拉把问题分离为**径向**和**角向**两个独立部分：

- **径向方程**：$d(\frac{1}{2}v^2) = \frac{F(r)}{m}\,dr$（力做功 = 动能增加）
- **角向方程**：$r^2\dot{\theta} = L/m$（角动量守恒）

从这两个方程，可以完全确定轨道。

> **为什么这是力学史上最重要的成就之一？** 欧拉证明了：对于**任何**有心力，轨道问题都可以分解为径向和角向两个一维问题。这个"分离"方法是后来所有天体力学计算的基础——从拉普拉斯到现代的卫星轨道计算，都遵循这个框架。

## 三种可以用代数曲线表示轨道的有心力

欧拉发现，只有三种有心力情况下的轨道可以用代数曲线（圆锥曲线等）表示。

### 1. 力正比于距离：$F \propto r$

轨道是以力心为**中心**的**椭圆**。

用直角坐标表示，轨道方程为：

$$a^2 z^2 + 2cf \cdot x^2 = 2a^2 cf$$

其中 $a$ 是初始距离，$c$ 是初始速度对应高度，$f$ 是力常数。椭圆的半轴分别为 $a$ 和 $\sqrt{2cf}$。

**周期**：$T = 2\pi\sqrt{2f}$——**与椭圆大小无关！** 这就是简谐运动等时性在二维的推广。

> **评注**：力正比于距离时轨道为以力心为中心的椭圆，这一结论完全正确。在现代力学中，这就是各向同性谐振子——其势能为 $U = \frac{1}{2}kr^2$（二维）。需要注意谐振子椭圆与开普勒椭圆的根本区别：前者的力心在椭圆**中心**（$F \propto r$），后者的力心在椭圆**焦点**（$F \propto 1/r^2$）。在物理学中，谐振子轨道在分子振动（Lennard-Jones势的底部近似）、固体中的晶格振动（声子）中有广泛应用。

> **评注**：力正比于距离时轨道为以力心为中心的椭圆，这一结论完全正确。在现代力学中，这就是各向同性谐振子——其势能为 $U = \frac{1}{2}kr^2$（二维）。需要注意谐振子椭圆与开普勒椭圆的根本区别：前者的力心在椭圆**中心**（$F \propto r$），后者的力心在椭圆**焦点**（$F \propto 1/r^2$）。在物理学中，谐振子轨道在分子振动（Lennard-Jones势的底部近似）、固体中的晶格振动（声子）中有广泛应用。

### 2. 力反比于距离平方：$F \propto 1/r^2$

轨道是以力心为**焦点**的**圆锥曲线**——椭圆、抛物线或双曲线，取决于初速度。

$$r = \frac{p}{1 + e\cos\theta}$$

**圆锥曲线的类型完全由初始速度决定**（设力心距为 $a$，力常数为 $f$）：

| 初始速度条件 | 轨道类型 | 物理意义 |
|-------------|---------|---------|
| $v < v_{\text{逃逸}}$（即 $c < f^2/a$） | 椭圆（$e < 1$） | 束缚态，如行星 |
| $v = v_{\text{逃逸}}$（即 $c = f^2/a$） | 抛物线（$e = 1$） | 临界逃逸 |
| $v > v_{\text{逃逸}}$（即 $c > f^2/a$） | 双曲线（$e > 1$） | 非束缚态，如流星 |
| $v = v_{\text{圆}}$（即 $c = f^2/(2a)$） | 圆（$e = 0$） | 特殊椭圆 |

其中逃逸速度对应的高度为 $f^2/a$，圆轨道速度对应的高度为 $f^2/(2a)$。

**周期**：$T \propto a^{3/2}$——这正是开普勒第三定律。

**这就是开普勒第一定律的解析证明**：行星轨道是以太阳为焦点的椭圆。欧拉不仅证明了正向问题（给定力求轨道），还完成了逆向问题（命题81：给定任意初始条件，证明轨道必为圆锥曲线）。

> **评注**：平方反比力下轨道为圆锥曲线 $r = p/(1+e\cos\theta)$，力心在焦点——这就是开普勒第一定律。牛顿在《原理》第一卷命题11-13中用几何方法证明了这一结果，是全书最精妙的几何论证之一。欧拉的解析证明在物理结论上完全等价，但在方法论上更优：从微分方程出发，通过标准积分步骤直接得到结果，每一步都是机械化的代数运算。这种方法的可推广性远优于几何证明——改变力律只需改变微分方程，不需要重新构思几何构图。

### 3. 力反比于距离立方：$F \propto 1/r^3$

轨道是**对数螺线**或**双曲螺线**。

这种力在自然界中不常见，但数学上很有趣——螺线轨道意味着物体一边旋转一边靠近（或远离）力心。

## 逆问题：已知轨道求力

已知物体沿某条曲线运动，需要多大的力？欧拉给出了系统的解法。

给定轨道的极坐标方程 $r(\theta)$，从法向方程 $F_n = mv^2/\rho$ 和角动量守恒出发，可以求出所需的力作为距离的函数。

### 进动轨道

欧拉发现：如果椭圆轨道的长轴缓慢旋转（进动），需要在平方反比力上附加一个与距离立方成反比的微小力：

$$F = \frac{k}{r^2} + \frac{\epsilon}{r^3}$$

更一般地，对于力 $\propto r^n$ 的近圆轨道，从近心点到远心点的角度为：

$$\text{拱线角度} = \frac{180°}{n+3}$$

这个公式给出了一个极其有用的判据：

| 力律 | $n$ | 近心点→远心点角度 | 轨道行为 |
|------|-----|-----------------|---------|
| $F \propto r$ | $+1$ | 90° | 闭合椭圆（力心在中心） |
| 常数力 | $0$ | 103.9° | 不闭合 |
| $F \propto 1/r$ | $-1$ | 127.3° | 不闭合 |
| $F \propto 1/r^2$ | $-2$ | 180° | 闭合椭圆（力心在焦点） |
| $F \propto 1/r^3$ | $-3$ | ∞ | 永不闭合（螺线） |

只有在 $n = 1$ 和 $n = -2$ 时轨道才是闭合的（拱线角度能整除360°）。这是一个极其深刻的结论：**在所有幂次力律中，只有力正比于距离和反比于距离平方给出闭合轨道**。伯特兰定理（1873年）后来严格证明了这是所有可能的中心力律中的仅有的两种情况。

> **评注**：进动轨道的分析是为月球运动做准备。月球近地点确实存在约每8.85年一圈的进动，主要由太阳对地月系统的引力扰动引起。欧拉用额外的 $1/r^3$ 力来模拟这一效应，数学上是可行的近似方法。值得对比的是水星近日点进动：牛顿引力预测的进动约为每世纪5319角秒，其中大部分可由其他行星的扰动解释，但剩余的43角秒/世纪无法用牛顿力学解释，直到1915年爱因斯坦用广义相对论的时空弯曲效应给出了解释。欧拉的 $1/r^3$ 修正虽然在数学形式上与广义相对论的后牛顿展开中的相应项相似，但物理机制完全不同。

## 月球运动——三体问题的雏形

欧拉将前面建立的框架应用于月球运动的初步分析。

### 问题的本质

月球围绕地球运动，但地球本身围绕太阳运动。所以月球同时受地球引力和太阳引力。这是**三体问题**——两个大质量物体（太阳、地球）加一个小质量物体（月球）。

欧拉的策略很聪明：把问题转换到**地球参考系**。对太阳和月球同时施加一个与地球的太阳加速度等大反向的加速度（利用力的叠加原理），不改变它们的相对运动。

### 月球受到的有效力

转换到地球参考系后，月球受到：

- **沿地月连线方向**：$h/y^2 + fy/z^3$（地球引力 + 太阳引力的径向分量）
- **垂直于地月连线方向**：$af/z^3$（太阳引力的横向分量）

其中 $y$ 是地月距离，$z$ 是日月距离，$h$ 和 $f$ 是引力常数，$a$ 是日地距离。

这不再是简单的有心力问题——太阳引力引入了与方向有关的扰动项，使得运动从二维变成了三维。

### 三维处理的必要性

欧拉认识到平面运动不足以描述月球运动，需要三维分析。月球轨道面（白道面）相对地球轨道面（黄道面）有约5°的倾角，而且**交点线**（两个平面的交线）在不断退行——每18.6年旋转一周。这种退行是太阳引力横向分量 $af/z^3$ 的直接后果。

> **评注**：欧拉对月球运动的分析是"限制性三体问题"的最早系统处理之一。他正确地将问题分解为：地球引力提供主要的椭圆轨道，太阳引力提供周期性扰动。这个框架后来被拉普拉斯和德洛内大幅发展，最终形成了19世纪的月球运动理论。完整的月球运动解析理论至今仍是天体力学中最复杂的成就之一。

## 三维曲线运动

最后，欧拉将整个理论推广到三维。

在三维情况下，力分解为三个分量：
1. **切向力** $F_t$：沿切线方向
2. **主法向力** $F_{n1}$：沿主法线方向（曲线弯曲方向的法线）
3. **副法向力** $F_{n2}$：沿副法线方向（垂直于切线和主法线的方向）

法向方程扩展为两个：
- $F_{n1} = mv^2/\rho_1$
- $F_{n2} = mv^2/\rho_2$

> **评注**：三维曲线运动需要两个法向方程（对应主曲率和挠率），这一结论正确且具有重要历史意义。Frenet-Serret公式（1852年、1847年独立提出）将这一框架系统化为微分几何的标准工具。欧拉的表述虽然不如Frenet-Serret的最终形式完整，但核心思想——三维曲线需要两个独立的曲率参数来描述——是正确的。在现代力学中，三维运动方程为 $m\vec{a} = \vec{F}$ 在Frenet标架下的三个投影，与欧拉的三个方程（切向+主法向+副法向）完全对应。

### 三维运动的交点退行

在三维情况下，轨道平面与参考平面的交线（**交点线**）的位置随时间变化。欧拉推导出了关键结果：

**相邻交点之间的角度间隔** $= 180°/m$，其中 $m^2 = (f+g)/g$，$f$ 是径向力常数，$g$ 是法向力常数。

当 $g > 0$（法向力指向参考平面）且 $m > 1$ 时，交点间隔小于180°——即交点**退行**（逆行旋转）。

对月球的应用：观察到的交点退行约每圈43'，可以反推法向力的强度。

> **评注**：交点退行公式是三维轨道理论的重要结果。在现代天体力学中，月球交点退行（18.6年周期）是潮汐摩擦和太阳扰动的综合效应，但欧拉的一阶近似已经抓住了主要行为——交点退行的存在和方向。

## 第五章总结

第五章是全书的精华，建立了曲线运动的完整解析理论：

| 主题 | 核心结果 | 历史意义 |
|------|---------|---------|
| 切向/法向分解 | $a_n = v^2/\rho$, $a_t = dv/dt$ | 自然坐标系 |
| 角动量守恒 | $r^2\dot{\theta} = \text{const}$ | 开普勒第二定律的解析形式 |
| 轨道分离法 | 径向+角向独立求解 | 天体力学的基础方法 |

欧拉在这一章中完成了牛顿《原理》核心内容的解析重建，并大大扩展了其范围。牛顿的几何证明虽然优美，但只能处理特殊情况；欧拉的解析方法是通用的——任何力律、任何初始条件，都能系统地求解。
